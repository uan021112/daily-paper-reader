## 摘要
以物体为中心的场景理解是计算机视觉中的一个基本挑战。现有方法通常依赖于多阶段流水线，首先应用预训练的分割器提取单个物体，随后进行逐物体的三维重建。此类方法计算开销大，对分割错误较为敏感，且在场景复杂度增加时扩展性较差。我们提出了 OCH3R，这是一个用于从单张 RGB 图像进行以物体为中心的全景三维重建的统一框架。OCH3R 通过一次前向传递，同时预测所有物体实例及其 6D 位姿和详细的三维重建。其核心思想是一个 Transformer 架构，用于预测像素级属性，包括基于 CLIP 的类别嵌入、度量深度、归一化物体坐标 (NOCS) 以及代表每个物体的固定数量的 3D 高斯。为了监督这些高斯重建，我们利用预测的 6D 位姿将它们转换到规范空间，并将其与预渲染的规范真值对齐，从而避免了昂贵的逐图像高斯标签生成。在标准室内基准测试中，OCH3R 在单目深度估计、开放词汇语义分割以及仅 RGB 的类别级 6D 位姿估计方面均达到了最先进的性能，同时生成了高保真、可编辑的逐物体重建结果。至关重要的是，推理过程完全是前向传播的，且其扩展性与物体数量无关，在拥挤场景中相比传统的多阶段流水线实现了数量级的加速。

## Abstract
Object-centric scene understanding is a fundamental challenge in computer vision. Existing approaches often rely on multi-stage pipelines that first apply pre-trained segmentors to extract individual objects, followed by per-object 3D reconstruction. Such methods are computationally expensive, fragile to segmentation errors, and scale poorly with scene complexity. We introduce OCH3R, a unified framework for Object-Centric Holistic 3D Reconstruction from a single RGB image. OCH3R performs one forward pass to simultaneously predict all object instances with their 6D poses and detailed 3D reconstructions. The key idea is a transformer architecture that predicts per-pixel attributes, including CLIP-based category embeddings, metric depth, normalized object coordinates (NOCS), and a fixed number of 3D Gaussians representing each object. To supervise these Gaussian reconstructions, we transform them into canonical space using the predicted 6D poses and align them with pre-rendered canonical ground truth, avoiding costly per-image Gaussian label generation. On standard indoor benchmarks, OCH3R achieves state-of-the-art performance across monocular depth estimation, open-vocabulary semantic segmentation, and RGB-only category-level 6D pose estimation, while producing high-fidelity, editable per-object reconstructions. Crucially, inference is fully feed-forward and scales independently of the number of objects, offering orders-of-magnitude speedups over conventional multi-stage pipelines in cluttered scenes.

## 1. 核心问题与整体含义
论文针对的是计算机视觉中的一个长期挑战：如何从单张 RGB 图像中理解并重建由离散、具有位姿的物体组成的 3D 场景。
*   **研究动机**：现有的方法主要分为两类：一是“场景级”方法，虽然速度快但生成的是无结构的几何“汤”，无法区分个体；二是“物体级”方法，依赖于复杂的多阶段流水线（分割 -> 逐物体重建 -> 对齐），这种方式计算昂贵、易受上游分割错误影响，且难以扩展到复杂拥挤的场景。
*   **核心目标**：提出一种统一的、以物体为中心的框架，通过单次前向传播直接输出场景中所有实例的语义、6D 位姿和高保真 3D 重建结果。

## 2. 方法论
OCH3R 采用了一个基于 Transformer 的统一预测架构，核心思想是将 3D 重建任务转化为像素级的属性预测。

*   **核心架构**：使用 DINOv2 作为骨干网络，连接一个 48 层的 ViT 编码器和多个 DPT 风格的解码器头。
*   **像素级预测**：对于每个像素 $(u, v)$，网络同时预测：
    1.  **语义嵌入 ($\hat{e}$)**：基于 CLIP 的类别嵌入，用于实例发现和语义识别。
    2.  **度量深度 ($\hat{d}$)**：用于将像素反投影到 3D 空间。
    3.  **归一化物体坐标 (NOCS, $\hat{c}$)**：建立像素与物体规范空间之间的对应关系。
    4.  **3D 高斯原语 ($\hat{G}$)**：每个像素预测固定数量（$k=2$）的各向异性 3D 高斯参数。
*   **关键技术细节**：
    *   **实例与位姿恢复**：通过聚类语义嵌入发现实例，并利用预测的 NOCS 坐标通过 RANSAC-Umeyama 算法求解每个物体的 $SIM(3)$ 位姿。
    *   **规范空间监督 (Canonical-Space Supervision, CSS)**：这是训练 3D 高斯的精髓。预测的高斯通过估计的位姿转换到“规范空间”，并与预渲染的 42 个视角的真值图像进行对比训练。这种方式避免了为每张训练图生成高斯标签，并强制模型学习物体的完整（非可见部分）形状补全。
    *   **相机空间偏移**：高斯均值在相机射线上预测偏移量，允许高斯移动到遮挡区域之后。

## 6. 主要结论与发现
*   **性能领先**：OCH3R 在 3D 重建精度和语义一致性上显著优于现有的多阶段和生成式方法。
*   **速度优势**：推理速度比 Gen3DSR 等方法快约 **2000 倍**，比强力多阶段流水线快 **30 倍**以上。
*   **协同效应**：多任务联合训练能够提升单项任务的性能，例如语义信息有助于稳定位姿估计和几何重建。
*   **泛化能力**：模型能够很好地处理未见过的物体类别和复杂的遮挡场景。

## 7. 优点
*   **统一性**：将复杂的 3D 场景分解任务简化为单次前向预测，极大地提高了系统的鲁棒性。
*   **非可见补全**：通过 CSS 机制，模型能够补全物体被遮挡的部分，生成完整的 3D 模型。
*   **可编辑性**：输出结果是以物体为中心的 3D 高斯集合，天然支持物体的选择、移动和 AR 编辑。
*   **高效扩展**：推理时间基本不随场景中物体数量的增加而增长。

## 8. 不足与局限
*   **特殊材质处理**：对于高反射（镜面）或透明（玻璃）物体，深度和 NOCS 预测仍存在困难，导致几何畸变。
*   **深度耦合风险**：3D 高斯的初始位置高度依赖深度预测，深度的系统性偏差会直接传播到重建结果中。

*   **计算资源需求**：1.4B 的参数量和对 H100 的依赖表明该模型对训练算力的要求极高。

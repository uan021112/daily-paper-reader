## 摘要
我们研究了从 2D 图像和初始 3D 形状中再生 3D 对象的问题。大多数 3D 生成器以单次方式运行，将文本或图像转换为 3D 对象，但可控性有限。与此不同，我们引入了 3D-ReGen，这是一种以初始 3D 形状为条件的 3D 再生器。这种概念上简单的表述使我们能够支持许多有用的任务，包括 3D 增强、重建和编辑。3D-ReGen 使用了一种基于 VecSet 的新型条件机制，允许再生器以一致的细粒度细节更新或改进输入几何。3D-ReGen 通过自监督代理任务和增强，从现成的 3D 数据集中学习广泛适用的再生先验，无需额外标注。我们评估了 3D-ReGen 的几何一致性和细粒度质量，在多项任务的可控 3D 生成中实现了最先进的性能。

## Abstract
We consider the problem of regenerating 3D objects from 2D images and initial 3D shapes. Most 3D generators operate in a one-shot fashion, converting text or images to a 3D object with limited controllability. We introduce instead 3D-ReGen, a 3D regenerator that is conditioned on an initial 3D shape. This conceptually simple formulation allows us to support numerous useful tasks, including 3D enhancement, reconstruction, and editing. 3D-ReGen uses a new conditioning mechanism based on VecSet, which allows the regenerator to update or improve the input geometry with consistent fine-grained details. 3D-ReGen learns a widely applicable regeneration prior from off-the-shelf 3D datasets via self-supervised pretext tasks and augmentations, without additional annotations. We evaluate both the geometric consistency and fine-grained quality of 3D-ReGen, achieving state-of-the-art performance in controllable 3D generation across several tasks.

#### 2. 论文提出的方法论
*   **核心思想**：利用潜空间扩散模型（Latent Diffusion Model），将 3D 再生任务建模为条件分布采样 $p(x_H | y_L, I)$，其中 $x_H$ 是高质量目标，$y_L$ 是低信息量的初始形状，$I$ 是引导图像。
*   **关键技术细节**：
    *   **VecSet 潜空间**：使用 VecSet 作为 3D 表示，将形状编码为一组潜向量（Tokens）。其优点是 Token 数量可根据形状复杂度动态调整。
    *   **Token 拼接条件机制**：不同于常见的交叉注意力（Cross-Attention），3D-ReGen 将初始形状 $y_L$ 编码后的 Token 与带噪声的目标 Token 直接在序列维度进行**拼接**。这种设计允许模型通过自注意力机制直接处理输入几何与生成几何之间的空间关系。
    *   **预处理模块**：对条件 Token 应用零初始化（Zero-initialized）的 MLP 和位置编码，以避免在引入新控制信号时对预训练模型产生“冲击”。
    *   **自监督数据构建**：通过对高质量 3D 数据进行自动降级（如加噪、降采样、局部遮掩、多对象组合降级）来生成配对的训练数据，无需人工标注。

*   **通用性强**：不依赖于特定任务的架构设计，一个模型解决多种 3D 逆向问题。
*   **几何保真度高**：相比纯图像生成模型，由于引入了 3D 几何约束，生成的模型在姿态和比例上更加准确。
*   **数据驱动**：巧妙地利用自监督代理任务解决了 3D 配对数据稀缺的问题。

*   **潜空间限制**：模型性能高度依赖于 VecSet VAE 的重建质量，如果 VAE 无法捕捉极细微的拓扑变化，再生过程也会受限。

*   **推理开销**：作为 2.8B 参数的扩散模型，且涉及大量的 Token 处理，其实时性可能不如一些轻量级的 feed-forward 模型。

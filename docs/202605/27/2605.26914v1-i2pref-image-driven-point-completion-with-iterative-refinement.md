## 摘要
我们提出一种以图像为条件进行点云补全的方法，该方法将图像作为主要几何来源而非辅助引导。为此，我们引入一个图像到点云（I2P）模块，可以直接从单张RGB图像重构完整的点云，无需3D输入。此外，我们还引入一个基于Transformer的点到点（P2P）优化模块，该模块利用点令牌与图像特征之间的自注意力和交叉注意力，对粗糙的I2P输出进行迭代优化。I2P模块使图像编码器能够学习丰富的几何表示，而P2P模块则逐步恢复精细细节。与依赖于辅助损失或融合模块的现有多模态方法不同，我们的显式I2P任务仅基于图像提供了强几何先验知识。在ShapeNet-ViPC上的大量实验表明，我们的方法实现了最先进的补全性能，相比先前方法，Chamfer距离相对提升了12.3%。代码开源于：https://github.com/AzharSindhi/I2PRef.git

## Abstract
We present an image-conditioned point cloud completion approach that treats images as the primary geometric source rather than a secondary guide. To this end, we introduce an Image-to-Point (I2P) module that can reconstruct complete point clouds directly from a single RGB image, with no need for 3D inputs. Additionally, we introduce a transformer-based Point-to-Point (P2P) refinement module that uses self- and cross-attention between point tokens and image features to iteratively refine the coarse I2P output. The I2P module enables the image encoder to learn rich geometric representations, while the P2P module progressively recovers fine-grained details. Unlike existing multimodal methods that rely on auxiliary losses or fusion modules, our explicit I2P task provides a strong, geometry-aware prior based on images alone. Extensive experiments on ShapeNet-ViPC demonstrate state-of-the-art completion performance with a 12.3% relative Chamfer Distance improvement over prior methods. Code is available at: https://github.com/AzharSindhi/I2PRef.git

## 1. 核心问题与整体含义（研究动机和背景）

- **问题背景**：点云补全旨在从部分观测的3D点云恢复完整几何结构。真实场景中因遮挡、传感器噪声等，3D扫描常不完整。现有方法多依赖3D输入或辅助多模态融合，将图像信息作为次要引导（如辅助对齐损失或融合模块），未能充分挖掘图像中蕴含的丰富几何信息。
- **核心动机**：最近单视图3D重建的成功表明，单张RGB图像本身足以提供足够的几何线索。因此，本文提出将图像作为点云补全**主要**几何源，而非辅助引导，以解决从稀疏/不完整观测推断完整3D几何的欠定问题。

## 2. 方法论：核心思想、关键技术细节

- **整体架构**：由两个核心模块组成——**图像到点（I2P）模块**和**点对点（P2P）迭代优化模块**，实现“粗→细”两步补全。
- **I2P模块**：
    - 输入：单张RGB图像 `I ∈ R^{H×W×3}`。
    - 使用U-Net风格层次编码器（含7×7卷积、ResNet块、注意力层、下采样），提取多尺度特征。
    - 将扁平化特征输入**点生成器**（类似XMFNet的可差分生成器），输出**粗点云** `P0`（1024个点），再与部分输入点云（下采样至1024点）拼接为2048点粗补全。
    - 特点：仅6M参数，轻量化，无需任何3D输入。
- **P2P优化模块**：
    - 输入：粗点云 `P0` 和图像特征 `FI`。
    - 将点坐标线性投影为嵌入 `X0`，经L个Transformer块迭代（实验中L=4）。
    - 每个Transformer块：先**自注意力**（点与点之间）捕捉几何上下文；再**交叉注意力**（点作为查询、图像特征作为键/值）注入图像指导；最后通过MLP预测坐标偏移 `Δxyz`，逐点累加更新位置。
    - 最终输出精细点云 `Pc`。
- **损失函数**：
    - 联合监督粗点云和最终点云：`L_cd = CD(Pc, Pgt) + α·CD(P0, Pgt)`，其中Chamfer距离为对称形式。
    - α从0.7线性退火至0.1，训练初期侧重粗结构，后期侧重细节优化。

## 摘要
视觉几何 Transformer (VGGT) 是一种用于多种 3D 任务的强大前馈模型，但其交替注意力 (AA) 堆栈随总 Token 数量呈二次方缩放，导致处理长视频片段的开销巨大。现有的 Token 减少加速器在 AA 内部运行，使得进入 AA 的 patch 网格未被压缩。我们提出了 PaceVGGT，这是一个交替注意力前 (pre-AA) Token 剪枝框架，它在冻结的 VGGT 的第一个 AA 块之前对 DINO patch Token 进行剪枝。PaceVGGT 训练了一个轻量级的 Token 评分器，用于根据 DINO 特征估计每个 Token 的重要性。该评分器首先针对未剪枝骨干网络的 AA 内部注意力目标进行蒸馏，然后在下游相机、深度和点图损失下进行微调。每帧保留预算固定了骨干网络可见的序列长度，而重要性自适应的合并/剪枝分配则在固定的总合并预算下，保留了高显著性帧的残余内容。特征引导恢复模块重建了预测头所需的密集空间网格。在 ScanNet-50 和 7-Scenes 数据集上，PaceVGGT 在降低推理延迟的同时，保持在重建质量-延迟的前沿。在 ScanNet-50 上，当 N=300 时，它比未修改的 VGGT 减少了 5.1 倍的延迟；当 N=1000 时，比 LiteVGGT 减少了 1.47 倍。这些结果表明，交替注意力前剪枝是冻结的 VGGT 式几何 Transformer 的一种可行加速路径。

## Abstract
Visual Geometry Transformer (VGGT) is a strong feed-forward model for multiple 3D tasks, but its Alternating-Attention (AA) stack scales quadratically in the total token count, making long clips expensive.   Existing token-reduction accelerators operate inside AA, leaving the patch grid that enters AA uncompressed.   We introduce PaceVGGT, a pre-AA token pruning framework that prunes DINO patch tokens before the first AA block of a frozen VGGT. PaceVGGT trains a lightweight Token Scorer that estimates per-token importance from DINO features. The scorer is first distilled against an AA-internal attention target from the unpruned backbone, then refined under downstream camera, depth, and point-map losses.   A per-frame keep budget fixes the backbone-visible sequence length, while an importance-adaptive merge/prune assignment preserves residual content from high-saliency frames under a fixed total merge budget.   A Feature-guided Restoration module reconstructs the dense spatial grid required by the prediction heads.   On ScanNet-50 and 7-Scenes, PaceVGGT remains on the reconstruction quality--latency frontier while reducing inference latency.   On ScanNet-50, it reduces latency by \(5.1\times\) over unmodified VGGT at \(N=300\) and \(1.47\times\) over LiteVGGT at \(N=1000\).   These results identify pre-AA pruning as a viable acceleration route for frozen VGGT-style geometry transformers.

*   **研究背景**：视觉几何 Transformer (VGGT) 是 3D 重建领域强大的前馈模型，能同时预测相机参数、深度图和点云。然而，其核心的“交替注意力”（Alternating-Attention, AA）机制在处理长视频序列时，计算复杂度随 Token 数量呈二次方增长，导致显存占用极高（如 360 帧序列在 H100 上需 95GB VRAM）。
*   **核心问题**：现有的加速方案（如 FastVGGT, LiteVGGT）多在 AA 模块内部进行 Token 缩减，这意味着进入 AA 模块前的初始 Patch 网格未被压缩，导致模型前段依然存在巨大的计算冗余。
*   **研究动机**：能否在 Token 进入 AA 堆栈之前（即 DINO 骨干网络与 AA 模块的接口处）就进行剪枝，从而实现全流程的加速？

### 2. 论文提出的方法论
PaceVGGT 提出了一种“预交替注意力”（Pre-AA）剪枝框架，主要包含三个核心模块：
*   **Token 评分器 (Token Scorer)**：
    *   这是一个轻量级的深度卷积网络，直接作用于 DINO 提取的特征图。
    *   **核心思想**：通过蒸馏学习来预测 Token 的“几何重要性”。其监督信号（Target）结合了两个维度：**相机锚定得分**（相机 Token 对 Patch 的注意力）和**跨视图匹配得分**（Patch 之间的全局匹配强度）。
*   **重要性自适应 Token 剪枝 (Token Pruning)**：
    *   将 Token 分为三类：**保留 (Keep)**、**合并 (Merge)** 和 **剪枝 (Prune)**。
    *   **自适应分配**：根据每帧的显著性动态分配合并预算。高显著性帧保留更多细节，低显著性帧则更多地进行剪枝。
    *   **相似度聚合**：被选为“合并”的 Token 会根据 DINO 特征相似度加权聚合到邻近的“保留”Token 中，以减少信息丢失。
*   **特征引导恢复模块 (Feature-guided Restoration)**：
    *   由于下游预测头（如深度图预测）需要稠密的 Patch 网格，该模块利用交叉注意力机制，以原始 DINO 特征为查询（Query），以经过骨干网络处理的稀疏特征为键值（K, V），重建出高质量的稠密空间网格。

### 6. 论文的主要结论与发现
*   **加速效果显著**：在 ScanNet-50 上，当 N=300 时，比原始 VGGT 快 **5.1 倍**；当 N=1000 时，比当前最强的 LiteVGGT 还快 **1.47 倍**。

*   **关键发现**：在 AA 模块前进行剪枝是可行的，但前提是必须使用“几何感知”的评分准则，单纯依靠 2D 图像特征相似度（如 ToMe）会导致几何信息的严重丢失。

### 7. 优点（亮点）
*   **切入点精准**：识别出 AA 模块前的接口是加速的“源头”，通过在此处剪枝实现了全流程的计算节省。
*   **轻量且即插即用**：作为一种“包装”方案，它不需要重新训练庞大的 VGGT 骨干网络，仅需训练极小的附加模块。
*   **恢复机制巧妙**：通过特征引导的交叉注意力解决了剪枝后如何保持稠密预测精度的难题。

### 8. 不足与局限
*   **领域偏移风险**：Token 评分器是在 ScanNet 上训练的，对于极端不同的场景（如室外大场景或高速运动场景），评分器的有效性尚未得到充分验证。

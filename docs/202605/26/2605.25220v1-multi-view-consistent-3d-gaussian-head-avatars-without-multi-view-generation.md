## 摘要
高保真度三维高斯头部头像生成对于增强现实/虚拟现实、远程临场和数字人应用至关重要。现有方法依赖于多视图数据集、三维捕获或中间二维视图合成。相比之下，我们仅从随机采样的二维图像中学习条件和非条件三维头部模型，而不使用多视图数据、三维监督或中间视图生成。我们引入MVCHead，一种单次状态空间模型，直接在三维表示中强制执行多视图一致性，同时在这些约束下回归三维高斯体。其核心是，我们提出了一种层次化状态空间（HiSS）模块，从粗到细逐步优化高斯体，同时捕获长距离依赖关系。在每个HiSS模块中，我们通过提出的层次化双向状态扫描（HiBiSS）修改Mamba的标准单向扫描，使递归与多视图不一致性最强的轴对齐。最后，我们设计了一个SE(3)多视图评判器，判断一组自渲染图是否来自单个底层三维配置，在不观察真实多视图对的情况下奖励跨视图像素对齐。MVCHead在感知质量上达到最先进水平，在纹理和几何一致性上超越先前方法，并保持可比的重合一致性。为展示可扩展性，我们发布了FaceGS-10K，这是第一个大规模即用型三维高斯头部资产数据集，用于训练和评估三维头部模型。项目页面和代码：https://humansensinglab.github.io/MVCHead/

## Abstract
High-fidelity 3D Gaussian head avatar generation is critical for applications such as AR/VR, telepresence, and digital humans. Existing methods depend on multi-view datasets, 3D captures, or intermediate 2D view synthesis. In contrast, we learn both conditional and unconditional 3D head models from randomly sampled 2D images alone, without using multi-view data, 3D supervision, or intermediate view generation. We introduce MVCHead, a single-shot state space model that enforces multi-view consistency (MVC) directly in the 3D representation while regressing 3D Gaussians under these constraints. At its core, we propose a Hierarchical State Space (HiSS) block that progressively refines Gaussians from coarse to fine, while capturing long-range dependencies. Within each HiSS block, we modify Mamba's standard unidirectional scan with the proposed Hierarchical Bi-directional State Scan (HiBiSS) that aligns recurrence with the axes along which multi-view inconsistencies are strongest. Finally, we design an SE(3) Multi-view Critic that judges whether a set of self-renders arises from a single underlying 3D configuration, rewarding cross-view pixel alignment without observing real multi-view pairs. MVCHead achieves state-of-the-art perceptual quality, surpasses prior methods in both texture and geometric consistency, and maintains comparable shape consistency. To demonstrate scalability, we release FaceGS-10K, the first large-scale dataset of ready-to-use 3D Gaussian head assets for training and evaluation of 3D head models. Project Page and code: https://humansensinglab.github.io/MVCHead/

#### 1. 论文的核心问题与整体含义

-   **研究动机**：高保真度的3D高斯（3D Gaussian Splatting, 3DGS）头部化身生成在AR/VR、数字人等领域至关重要。现有主流方法在生成多视图一致的3D头部时，通常依赖昂贵的多视角拍摄设备、3D扫描数据，或者先通过扩散模型生成中间多视角图像再进行3D重建。
-   **核心问题**：上述方法资源消耗大、可扩展性差。中间视图生成不仅计算量大，且生成的视图间可能存在身份漂移和不一致性，无法端到端地优化跨视图像素对齐。
-   **整体目标**：本文旨在探索一个“最小资源设定”下的解决方案，即**仅从随机采样的2D人脸图像数据中，学习一个无条件和条件化的3D高斯头部生成模型**，过程中不依赖任何多视角数据、3D真实标注或生成中间视图。

#### 2. 论文提出的方法论：核心思想与关键技术

论文提出了 **MVCHead**，一个直接从潜码 (latent code) `z` 预测全部3D高斯参数的单次前馈状态空间模型。其核心思想是通过架构设计和损失函数，将多视图一致性（MVC）内在地构建到模型之中。

-   **核心架构**:
    1.  **层次化状态空间（HiSS）模块**：堆叠的多层模块，以从粗到细的方式逐步精化高斯体。精细层的高斯参数被表示为粗粒度锚点高斯体的偏移量，确保结构稳定性。
    2.  **特征提取与条件控制**：初始的随机可学习标记（Tokens）经过位置编码和AdaIN层的几何-外观解耦控制后，送入HiSS模块。
    3.  **Dual-Mixer架构**：每个HiSS模块内包含两个并行的混合器：
        -   **自注意力**：捕获全局语义和身份相关的长距离依赖。
        -   **状态空间模块**：执行提出的HiBiSS扫描，强制局部网格对齐的一致性。
    4.  **SE(3)多视图评判器**：这是强制MVC的关键组件。它是一个可训练的编码器 `Eψ`，输入一组自渲染的图像和对应的相机姿态 `{R(S_θ(z), T_k), T_k}`，输出一个标量一致性分数。它通过区分“同一头像的不同视角渲染（正样本）”和“不同头像的同一视角渲染（负样本）”来学习判断3D几何的合理性。

-   **关键技术细节**:
    -   **HiBiSS（层次化双向状态空间扫描）**：基于“多视图不一致性主要沿水平（偏航）和垂直（俯仰）图像轴分布”的观察（Insight 1），HiBiSS在2D token网格上执行四个方向的扫描（左→右、右→左、上→下、下→上），将状态空间传播与不一致性最强的轴对齐，实现各向异性的、姿态感知的平滑。
    -   **一致性构建（Insight 2）**：论文的核心洞察是，尽管真实的多视角图像可能不一致，但**从一个确定的3D模型渲染出的所有视角，在几何上是天然一致的**。基于此，可以训练一个评判器来识别这种“一致性”，从而指导生成器学习产生更一致的3D几何。

-   **算法流程**:
    1.  从潜码 `z` 开始，通过HiSS模块生成所有3D高斯参数 `S_θ(z)`。
    2.  使用可微分光栅化器 `R`，从随机相机姿态 `T_k` 渲染出K个视图 `{I_k}`。
    3.  将渲染图和相机姿态输入SE(3)多视图评判器 `E_ψ`，最大化其输出的MVC分数。
    4.  同时，使用对抗性纹理鉴别器 `D_φ` 确保渲染图符合真实图像分布；并添加空间正则化损失 `L_knn` 和 `L_ctr` 约束高斯体分布。
    5.  联合优化所有组件，总损失函数为：`L_total = λ_mvc * L_mvc + L_adv + λ_knn * L_knn + λ_ctr * L_ctr`

## 摘要
从稀疏视图图像重建铰接物体是一个不适定问题，需要同时推断几何和底层铰接结构。现有基于NeRF和3D高斯泼溅（3DGS）的铰接物体重建方法通常依赖密集视图或强先验（如深度图、关节类型、预定义关节数量），并且需要昂贵的逐物体优化。在本文中，我们提出了ArtSplat，这是首个用于铰接3D高斯泼溅的前馈框架。它通过单次前向传递，从跨越多个铰接状态的稀疏多视图图像中同时重建几何和关节参数。为了解决单次前向铰接重建的挑战，我们引入了逐像素关节图表示，使得关节参数估计能够集成到前馈流程中。我们进一步提出了一种带有状态令牌的跨状态注意力（CSA）机制，该机制有效地捕捉了输入状态间的离散运动。在来自PartNet-Mobility的68个铰接物体（包括单关节和多关节配置）上的实验表明，ArtSplat在几何和关节估计方面均达到了具有竞争力的性能，同时速度比基线方法快400倍以上。

## Abstract
Articulated object reconstruction from sparse-view images is an ill-posed problem that requires simultaneous inference of geometry and underlying articulation structure. Existing methods for articulated object reconstruction based on NeRF and 3D Gaussian Splatting (3DGS) typically rely on dense views or strong priors (e.g., depth maps, joint types, predefined number of joints) and require costly per-object optimization. In this paper, we propose ArtSplat, the first feed-forward framework for articulated 3D Gaussian Splatting. It reconstructs both geometry and joint parameters from sparse multi-view images across multiple articulation states in a single forward pass. To address the challenges of single-pass articulated reconstruction, we introduce a per-pixel joint map representation that enables the integration of joint parameter estimation into the feed-forward pipeline. We further propose a Cross-State Attention (CSA) mechanism with state tokens, which effectively captures discrete motion across input states. Experiments on 68 articulated objects from PartNet-Mobility, including both single- and multi-joint configurations, demonstrate that ArtSplat achieves competitive performance in both geometry and joint estimation, while being over 400 times faster than baselines.

## 1. 核心问题与整体含义（研究动机和背景）
- **问题**：从稀疏视图图像重建铰接物体是一个病态问题，需要同时恢复物体几何和底层铰接结构（关节类型、轴、运动量）。现有方法（基于NeRF或3DGS）要么依赖密集视图（每状态50+张）、深度图、已知关节类型或数量等强先验，要么需要逐物体优化（数十分钟到数小时），效率极低，难以扩展到大量新物体。
- **动机**：虽然针对静态场景的前馈式3D重建（如DUSt3R、VGGT、AnySplat）和动态场景的模型已取得进展，但铰接物体既不完全是静态（不同视图间有部分运动），也不是有序的小运动视频帧，而是少数离散状态间的大幅运动，现有方法难以处理这种“离散、大幅位移”的多状态输入。
- **整体意义**：提出首个能同时预测几何和关节参数的前馈式铰接3DGS框架，消除了对每物体优化的依赖，显著提升效率，为机器人操作、数字孪生等下游任务提供快速重建能力。

## 2. 方法论：核心思想、关键技术细节、公式或算法流程
- **核心思想**：将铰接重建分解为几何（3D高斯）和关节参数（逐像素关节图）的联合预测，通过跨状态注意力机制捕捉两个状态间的离散运动，再通过可微的关节变换将高斯集变换到任意目标状态进行渲染。
- **关键技术细节**：
  1. **逐像素关节图表示**：每个像素用一个11维向量表示，包括关节类型（3类one-hot）、轴方向（3D单位向量）、枢轴位置（3D点）、旋转角（1）或平移位移（1）。这种表示是端到端可微的，避免了非可微的部件分割或固定数量槽位分配。
  2. **关节预测模块**：
     - **状态令牌**（State Token）：每个状态添加一个可学习令牌，初始化为随机，通过VGGT backbone的注意力层聚合该状态所有视图的patch令牌信息。
     - **跨状态注意力（CSA）**：一个独立的注意力块，以状态令牌为查询、另一状态的patch令牌为键和值，让每个状态令牌能关注另一状态的图像特征，从而捕获状态间差异。
     - **不变/可变DPT解码器**：双分支DPT头。不变分支（9通道）从两个状态令牌的平均值（`c_inv=(z0+z1)/2`）获得条件，预测所有状态一致的属性（类型、轴、枢轴）；可变分支（2通道）从各自状态令牌（`c_var_s`）获得条件，预测θ或d。特征调制采用FiLM。
  3. **高斯变换**：预测的高斯集通过关节图进行变换。训练时使用真值部件标签监督每个像素的关节参数；推理时使用HDBSCAN算法根据Plücker坐标（旋转关节）或轴方向（平移关节）对高斯进行聚类得到部件伪标签，然后对每个部件应用Rodrigues公式（旋转）或平移公式。
  4. **两阶段训练**：
     - Stage 1：仅使用直接监督损失（位姿、深度、关节一致性、平滑性），无渲染损失，学习稳定的几何和关节预测。
     - Stage 2：在Stage 1基础上增加渲染损失（RGB MSE + LPIPS），通过可微关节变换反向传播细化关节图质量。
- **主要损失函数**：Stage 1包括位姿损失、深度损失、关节损失（交叉熵+轴L1+枢轴垂直于真值轴距离+θ/d的Huber损失）、状态间一致性损失、部件感知平滑损失；Stage 2加入渲染损失λ_rgb=1（MSE+0.1LPIPS）。

## 3. 实验设计
- **数据集**：PartNet-Mobility，包含大规模带URDF注释的铰接物体（旋轴、平移关节），涵盖多种类别。训练时渲染48个视角×16个铰接状态，共768帧/物体。测试时保留68个物体（18个单关节、50个多关节）。
- **评估协议**：每个方法输入8张图像（2个状态×4个视角），在每状态12个保留视角上评估。报告单关节和多关节子集的结果。
- **基线方法**：对比了4种逐物体优化方法：PARIS（隐式，RGB+关节数量）、DTA（RGBD+关节数量）、ArtGS（RGBD+关节数量）、REArtGS（RGB+关节数量）、ScrewSplat（RGB，无先验）。其中ScrewSplat是RGB-only的最新3DGS方法。
- **评估指标**：
  - 几何：Chamfer Distance（整体CD-w、静态部件CD-s、可动部件CD-m）。
  - 运动：旋转轴角度误差（Ang_m）、轴位置误差（Pos_m）。
  - 外观：PSNR、SSIM。
- **推理速度**：比较每物体平均重建时间（单块A6000 GPU上）。

## 4. 资源与算力

## 6. 主要结论与发现
1. ArtSplat在所有RGB-only方法中几何和运动精度最优，尤其在可动部件CD-m上远低于ScrewSplat（单关节0.028 vs 0.301，多关节0.030 vs 0.405），说明其很好地识别了运动部件并准确估计关节。
2. 运动估计：轴角度误差（Ang_m）大幅低于所有基线（单关节6.65° vs 第二名26.27°，多关节5.79° vs 46.78°），证实跨状态注意力能有效捕获离散运动。
3. 外观：尽管仅用8张输入图像，PSNR/SSIM与需要密集视图优化的方法相当或更好（例如单关节PSNR 28.70 vs ArtGS 28.01）。
4. 速度：前馈推断<2秒，比最慢的逐物体方法快约400倍（ScrewSplat 23分钟），比最快的PARIS（12分57秒）快约400倍。
5. 消融实验表明，CSA、状态令牌、不变/可变分支、两阶段训练都是必要的。

## 7. 优点
- **前馈架构创新**：首次将铰接关节参数估计融入前馈3DGS管道，无需每物体优化，极大提升效率。
- **逐像素关节图**：完全可微，避免了传统部件分割或固定槽位分配带来的非可微问题，实现端到端训练。
- **跨状态注意力**：显式建模两个状态间的对比，有效从稀疏视图推断关节运动，是运动精度突破的关键。
- **两阶段训练**：先直接监督几何和关节，再引入渲染损失，避免早期梯度不稳定，训练更稳定。
- **通用性**：能处理单关节和多关节物体，且无需关节数量先验。
- **推理速度**：<2秒/物体，具有实际应用潜力。

## 8. 不足与局限
- **仅处理单自由度关节**：当前每个关节只能进行旋转或平移，且必须是直接连接到静态基座的部分。无法处理运动链（如机器人臂中串联的多关节）。
- **合成数据偏差**：所有训练和测试数据来自PartNet-Mobility渲染图像，未在真实世界多视角拍摄数据上验证，可能限制真实场景泛化性。
- **小运动模糊**：当两个输入状态间关节运动量很小时，轴估计容易偏离（论文附录中的失败案例），可能无法处理近似静态的输入。
- **网格重建质量**：由于使用3DGS作为表示，显式表面连接性差，提取的网格可能不完美（附录图I所示）。

- **实验覆盖局限**：未与其他前馈铰接重建方法（如ART、LARM）定量对比（仅在相关工作中提及），也未在多关节物体上按关节数量细分性能。同时，未提供误差分析或可视化更多失败案例。

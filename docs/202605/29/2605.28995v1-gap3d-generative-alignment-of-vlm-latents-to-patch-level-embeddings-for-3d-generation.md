## 摘要
近期将视觉-语言模型（VLM）作为生成模型条件提示编码器的方法，通常依赖于昂贵的端到端训练，或将特征映射到压缩表示，从而丢弃了像3D资产生成这类几何感知任务所需的密集空间结构。为解决此问题，我们提出了GAP3D，一种基于扩散的模块化方法，它将VLM生成的潜变量直接对齐到预训练图像编码器的完整补丁级特征空间，使得冻结的下游生成模型能够利用VLM作为提示编码器，同时保持空间结构化的条件信号。在3D资产生成任务上的评估表明，我们的方法主要通过通用领域的图像-文本对进行训练，从而绕过了对大规模3D数据的需求。尽管仅使用文本输入进行训练，但它在多模态提示下展现出涌现的零样本能力。最后，虽然目前更注重高层语义而非细粒度细节，但GAP3D证明了通过基于扩散的对齐可以部分弥合VLM和图像编码器特征空间之间的表示差距，为通过生成对齐到密集嵌入空间实现基础模型的模块化集成迈出了第一步。

## Abstract
Recent approaches integrating vision-language models (VLMs) as prompt encoders for generative model conditioning typically rely on expensive end-to-end training or map features to compressed representations, discarding the dense spatial structure required for geometry-aware tasks like 3D asset generation. To address this, we propose GAP3D, a modular, diffusion-based approach that aligns VLM-generated latents directly to the complete, patch-level feature space of a pre-trained image encoder, enabling a frozen downstream generative model to utilize a VLM as prompt encoder while maintaining a spatially structured conditioning signal. Evaluated on 3D asset generation, our method bypasses the need for large-scale 3D data by training mainly on general-domain image-text pairs. It also exhibits emergent zero-shot capabilities for multimodal prompts, despite being trained exclusively on text input. Finally, while currently prioritizing high-level semantics over fine-grained detail, GAP3D demonstrates that the representation gap between VLM and image-encoder feature spaces can be partially bridged through diffusion-based alignment, taking the first steps towards a modular integration of foundation models through generative alignment to dense embedding spaces.

- **整体含义**：能否将 VLM 的语义表示直接对齐到预训练图像编码器（如 DINOv2）的**完整补丁级嵌入空间**，从而使冻结的下游生成模型获得既具语义推理能力、又保留空间细节的条件信号？GAP3D 探索了这一方向，并将该方法应用于文本到 3D 资产生成，验证了可行性。

## 2. 方法论

- **核心思想**：通过基于扩散的生成对齐（generative alignment），将冻结 VLM 生成的潜变量（通过可训练软标记提取）映射到预训练视觉编码器 DINOv2 的完整特征空间，包括分类令牌（CLS）、注册令牌（register tokens）以及所有空间补丁嵌入（patch embeddings）。选择生成式（而非确定性回归）是因为文本到视觉映射本质上具有非确定性——需要从语义推理出几何和外观细节。  
- **关键技术细节**：
  - **VLM 编码**：使用 Qwen2.5‑VL‑3B‑Instruct（冻结），在输入文本嵌入后附加 s 个可训练软标记，提取其隐状态作为条件信号 C ∈ ℝ^(s×D_VLM)。
  - **目标表示**：采用 DINOv2 ViT‑L/14（518×518 输入 → 37×37 补丁网格，1369 个补丁 token + 1 个 CLS + 4 个 register token，维度 1024）。
  - **生成架构**：基于 Lumina‑Next 的扩散 Transformer（DiT，36 层，隐藏维度 1792）。对补丁网格使用 2D RoPE，对 CLS/register 使用学习位置嵌入 + 恒等 RoPE。条件信号 C 通过交叉注意力注入。
  - **训练目标**：使用**整流流（Rectified Flow）**，对三个组件（补丁、CLS、register）独立采样噪声，共享时间步 t，联合预测速度场。损失为加权流匹配损失：  
    \[
    \mathcal{L}(\theta) = \mathbb{E}_{t,x_0,x_1}\left[\sum_{k} \lambda_k \left\| v^k_\theta(\mathbf{x}_t, t, C) - (x^k_1 - x^k_0) \right\|^2\right]
    \]
    权重为 λ_p=0.4（补丁）、λ_cls=0.3、λ_reg=0.3。
  - **域适应**：两阶段训练——先在通用图像‑文本对（BLIP3‑o，~31M）上预训练 3 个 epoch，再在 3D 渲染数据（Objaverse‑XL，60k 图像）上微调 1000 个 epoch，以弥合自然图像与 3D 渲染之间的分布偏移。

## 6. 论文的主要结论与发现

1. **可行性与部分弥合**：通过扩散生成对齐，可以将 VLM 的语义潜变量映射到高维、稠密的补丁级图像嵌入空间（DINOv2），在余弦相似度、范数比等指标上接近 BLIP3‑o 在压缩空间上的表现，证明了表示差距可被部分弥合。
2. **模块化 3D 生成有效**：利用预训练 TRELLIS 的图像到 3D 分支，GAP3D 实现了文本到 3D 生成，无需重新训练 3D 模型；微调后质量接近原生 TRELLIS text‑to‑3D 基线（FD、KD、CLIP 分数），且显著优于预训练版本。
3. **域适应的关键性**：通用域预训练模型在 3D 渲染上产生严重背景伪影；通过域内 3D 渲染微调可大幅提升生成质量，而单纯的提示工程（如指令“黑背景”）无法替代。
4. **涌现零样本多模态能力**：尽管仅用文本训练，GAP3D 能利用 VLM 的联合表示空间，零样本处理多模态输入（图像+文本指令），实现颜色/材质编辑，但几何保持较弱。

## 7. 优点

- **模块化与可迁移**：不修改下游生成模型（TRELLIS），可独立升级 VLM 或对齐模块；利用通用图像‑文本数据，避免大规模 3D 数据需求。
- **保留空间结构**：将条件扩展到完整的补丁级特征，而非压缩 token，更利于几何感知任务。
- **域适应策略**：两阶段训练有效弥合分布偏移，且简单高效。
- **零样本多模态涌现**：体现冻结 VLM 的泛化潜力，无需额外多模态训练数据。
- **开源**：作者承诺开源代码，促进可复现性。

## 8. 不足与局限

- **细粒度细节丢失**：生成的嵌入在高层语义（类别、颜色）上表现尚可，但低层细节（精确形状、纹理、比例）常被忽略，导致 3D 资产几何保真度低于图像条件基线。
- **灾难性遗忘**：域内微调显著提升 3D 渲染域性能，但导致通用域（MS‑COCO）对齐质量下降，未尝试混合数据以维持泛化。
- **计算资源需求较高**：DiT 使用 128 块 MI250 GPU，训练成本高，可能限制可访问性。
- **多模态编辑局限**：当前模型将输入图像视为“语义提示”，不能严格保持输入几何，距离实用编辑仍有差距。
- **对比的不公平性**：与 BLIP3‑o 对比时，后者使用更大的 7B VLM 和更多训练数据，GAP3D 的指标差距可能部分源于资源差异，而非方法缺陷。
- **评估范围有限**：3D 生成仅在 Toys4K（对象级、简化背景）上评估，未在复杂场景或真实世界 3D 数据上验证；未对比其他对齐范式（如 Q‑former 映射）。

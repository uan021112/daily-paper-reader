## 摘要
视觉几何 Transformer (VGGT) 是用于多种 3D 任务的强力前馈模型，但其交替注意力 (AA) 堆栈随总令牌数量呈二次方缩放，导致处理长视频片段的成本极高。现有的令牌减少加速器通常在 AA 内部运行，使得进入 AA 的补丁网格保持未压缩状态。我们引入了 PaceVGGT，这是一种预 AA 令牌剪枝框架，在冻结的 VGGT 第一个 AA 块之前对 DINO 补丁令牌进行剪枝。PaceVGGT 训练了一个轻量级的令牌评分器，用于从 DINO 特征中估计每个令牌的重要性。该评分器首先针对来自未剪枝主干网络的 AA 内部注意力目标进行蒸馏，随后在下游相机、深度和点图损失下进行精调。每帧保留预算固定了主干网络可见的序列长度，而重要性自适应的合并/剪枝分配则在固定的总合并预算下，保留了高显著性帧的残余内容。特征引导的恢复模块负责重建预测头所需的密集空间网格。在 ScanNet-50 和 7-Scenes 数据集上，PaceVGGT 在降低推理延迟的同时，保持在重建质量-延迟的前沿。在 ScanNet-50 上，当 N=300 时，其延迟比未修改的 VGGT 降低了 5.1 倍；当 N=1000 时，比 LiteVGGT 降低了 1.47 倍。这些结果表明，预 AA 剪枝是冻结的 VGGT 式几何 Transformer 的一种可行加速路径。

## Abstract
Visual Geometry Transformer (VGGT) is a strong feed-forward model for multiple 3D tasks, but its Alternating-Attention (AA) stack scales quadratically in the total token count, making long clips expensive.   Existing token-reduction accelerators operate inside AA, leaving the patch grid that enters AA uncompressed.   We introduce PaceVGGT, a pre-AA token pruning framework that prunes DINO patch tokens before the first AA block of a frozen VGGT. PaceVGGT trains a lightweight Token Scorer that estimates per-token importance from DINO features. The scorer is first distilled against an AA-internal attention target from the unpruned backbone, then refined under downstream camera, depth, and point-map losses.   A per-frame keep budget fixes the backbone-visible sequence length, while an importance-adaptive merge/prune assignment preserves residual content from high-saliency frames under a fixed total merge budget.   A Feature-guided Restoration module reconstructs the dense spatial grid required by the prediction heads.   On ScanNet-50 and 7-Scenes, PaceVGGT remains on the reconstruction quality--latency frontier while reducing inference latency.   On ScanNet-50, it reduces latency by \(5.1\times\) over unmodified VGGT at \(N=300\) and \(1.47\times\) over LiteVGGT at \(N=1000\).   These results identify pre-AA pruning as a viable acceleration route for frozen VGGT-style geometry transformers.

### 1. 核心问题与整体含义（研究动机和背景）
视觉几何 Transformer (VGGT) 是一种强大的 3D 视觉基础模型，能够单次前馈预测相机参数、深度图和点云。然而，其核心架构——**交替注意力 (AA) 堆栈**的计算复杂度随视频帧数（Token 总数）呈二次方增长。
*   **痛点**：现有的加速方案（如 FastVGGT, LiteVGGT）多在 AA 内部进行 Token 缩减，这意味着模型的前几层仍需处理全量 Token，导致长序列推理时显存溢出（OOM）和高延迟。
*   **核心目标**：在 Token 进入 AA 堆栈之前（即 DINO 特征提取后）就进行剪枝，从而让整个主干网络都运行在精简后的序列上，实现更彻底的加速。

### 2. 方法论：核心思想与关键技术
PaceVGGT 提出了一个“预 AA 剪枝”框架，主要包含三个模块：
*   **令牌评分器 (Token Scorer)**：一个轻量级的深度卷积网络，输入 DINO 特征，预测每个 Token 的“几何重要性”。
    *   **蒸馏机制**：评分器通过模仿未剪枝主干网络内部的注意力分布（相机锚定分数和跨视图匹配分数）进行训练。
*   **令牌剪枝模块 (Token Pruning)**：
    *   **三路分配策略**：将 Token 分为“保留 (Keep)”、“合并 (Merge)”和“剔除 (Prune)”。
    *   **重要性自适应分配**：根据帧的显著性动态分配合并预算，高显著性帧保留更多细节。
    *   **相似度加权合并**：将合并集的 Token 聚合到保留集中，减少信息丢失。
*   **特征引导恢复模块 (Feature-guided Restoration)**：
    *   由于下游预测头（如深度图预测）需要密集的空间网格，该模块利用原始 DINO 特征作为查询（Query），通过交叉注意力机制从主干网络输出的稀疏特征中重建出密集的特征图。

*   **数据集**：主要在 **ScanNet-50**（室内场景）上进行评估，并在 **7-Scenes** 和 **NRGBD** 上进行零样本（Zero-shot）泛化测试。
*   **Benchmark 任务**：3D 点云重建（Chamfer 距离、准确率、完整性等）和相机位姿估计（ATE, ARE 等）。
*   **对比方法**：原始 **VGGT** 以及当前最先进的加速器：**FastVGGT**、**Co-Me** 和 **LiteVGGT**。

*   **硬件**：所有实验均在单张 **NVIDIA H100 GPU** 上运行，使用了 FP16 混合精度和 Flash-Attention 2。

### 6. 主要结论与发现
*   **加速效果显著**：在 ScanNet-50 上，当 N=300 时，PaceVGGT 比原始 VGGT 快 **5.1 倍**；当 N=1000 时，比 LiteVGGT 快 **1.47 倍**。
*   **质量保持良好**：在大幅加速的同时，重建质量（Chamfer 距离）和相机位姿精度几乎没有下降，甚至在某些长序列任务中优于基准模型。
*   **预 AA 剪枝可行性**：证明了只要评分准则具备“几何感知力”（而非单纯的 2D 相似度），在模型入口处进行剪枝是可行的。

### 7. 优点与亮点
*   **源头加速**：通过将剪枝位置提前，解决了长序列处理的显存瓶颈，使模型能处理 1000 帧以上的超长视频。
*   **几何感知评分**：不依赖简单的启发式规则，而是通过蒸馏 AA 内部的注意力机制来学习哪些 Token 对 3D 重建最重要。
*   **即插即用**：作为冻结主干网络的包装器，不需要重新训练庞大的 VGGT 模型，训练成本极低。

### 8. 不足与局限
*   **领域偏移风险**：评分器在 ScanNet 上训练，面对极端不同的场景（如室外大尺度场景）时，评分准确性可能会下降。
*   **任务覆盖有限**：目前仅评估了 3D 重建和位姿估计，未涉及 VGGT 的 2D 跟踪头。
*   **恢复模块瓶颈**：单层交叉注意力在处理极高分辨率输出时可能表现不足，可能需要更复杂的层次化重建设计。
*   **运动假设**：实验主要集中在小到中等运动的室内场景，对于大基线（Large-baseline）运动的鲁棒性有待进一步验证。

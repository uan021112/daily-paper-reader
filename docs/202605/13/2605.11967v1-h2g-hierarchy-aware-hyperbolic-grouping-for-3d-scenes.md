## 摘要
层级化三维分组旨在恢复跨多个粒度的场景组，从精细的物体部件到完整的物体，而不依赖于语义标签或固定词汇表。主要挑战在于将二维基础模型线索转化为连贯的层级监督，并将该层级嵌入到三维表示中。我们提出了 H2G，一种用于层级化三维分组的双曲亲和场。我们的方法通过 Dasgupta 的基于相似性的层级聚类目标来解释基础模型亲和力，从而推导出语义组织的树状监督。这种监督被蒸馏到一个单一的洛伦兹双曲特征场中，其几何结构非常适合树状分支结构。一个层级感知目标将该场与细粒度分配、粗粒度物体结构、紧凑特征簇以及最近公共祖先（LCA）排序对齐。这种公式化方法在一个特征空间中表示了多个分组级别，实现了基于二维基础模型知识的语义层级分组。

## Abstract
Hierarchical 3D grouping aims to recover scene groups across multiple granularities, from fine object parts to complete objects, without relying on semantic labels or a fixed vocabulary. The main challenge is to transform 2D foundation-model cues into coherent hierarchy supervision and embed that hierarchy in a 3D representation. We propose H2G, a hyperbolic affinity field for hierarchical 3D grouping. Our method derives semantically organized tree supervision by interpreting foundation-model affinities through Dasgupta's objective for similarity-based hierarchical clustering. This supervision is distilled into a single Lorentz hyperbolic feature field, whose geometry is well suited for tree-like branching structures. A hierarchy-aware objective aligns the field with fine-level assignments, coarse object structure, compact feature clusters, and LCA (Lowest Common Ancestor) ordering. This formulation represents multiple grouping levels in one feature space, enabling semantic hierarchical grouping grounded in 2D foundation-model knowledge.

#### 1. 核心问题与整体含义
论文探讨了 **3D 场景的层级化分组** 问题。在 3D 场景中，同一个点可以根据粒度不同，被归类为物体部件、完整物体或更大的物体组合。
*   **研究动机**：现有的 3D 分组方法（如 GARField）通常依赖于物理尺度（深度信息）来区分层级，或者仅依赖 2D 掩码的重叠关系。这导致它们在处理大型物体、部分遮挡或缺乏明显尺度差异的语义层级时表现不佳。
*   **核心目标**：利用 2D 基础模型（如 SAM 和 DINO）提供的语义线索，构建一个能够自动组织语义层级并在 3D 空间中一致表达的特征场。

#### 2. 方法论
H2G 提出了一种基于**双曲几何（Hyperbolic Geometry）**的亲和场方法，主要分为两个阶段：

*   **2D 层级监督构建**：
    *   利用 SAM 生成候选掩码，并用 DINO 提取区域特征。
    *   引入 **Dasgupta 目标函数**（一种衡量层级聚类质量的代价函数），将 2D 区域的相似性图转化为树状结构。
    *   为了提高效率，采用**递归谱二分法（Recursive Spectral Bisection）**作为 Dasgupta 目标的近似求解手段，快速生成每张图像的层级树。
*   **3D 双曲亲和场学习**：
    *   **洛伦兹模型（Lorentz Model）**：在 NeRF 架构中预测 3D 样本点的特征，并将其投影到具有负曲率的双曲空间中。双曲空间的体积随半径指数增长，天然适合嵌入具有大量分支的树状结构。
    *   **层级感知损失函数**：
        1.  **叶级角度分离（Leaf-level Separation）**：确保细粒度部件在角度上可分。
        2.  **根级角度分组（Root-level Grouping）**：使属于同一大类的部件向共同的根质心靠拢。
        3.  **紧凑性损失（Compactness）**：控制簇内特征的扩散。
        4.  **LCA 排序损失（LCA Ordering）**：利用双曲空间中点到原点的距离来模拟树的深度，确保特征间的距离关系符合层级合并顺序。

#### 6. 主要结论与发现
*   **性能提升**：H2G 在中等粒度的语义分组上表现尤为出色，显著优于依赖物理尺度的 GARField。
*   **无需尺度查询**：H2G 可以在单一特征空间内通过一次渲染完成多层级推理，无需像 GARField 那样进行多次尺度采样渲染。
*   **双曲优势**：实验证明双曲几何能以更低的失真度捕捉复杂的树状分支关系，LCA 排序损失有效地保持了层级合并的逻辑顺序。

#### 7. 优点
*   **几何创新**：将双曲几何引入 3D 场景理解，利用其数学特性解决层级表示问题，具有很强的理论支撑。
*   **语义驱动**：摆脱了对深度/尺度的过度依赖，能够处理语义相关但物理尺寸不一的物体分组。
*   **效率高**：推理时无需尺度扫描，且通过谱二分法优化了预处理速度。

#### 8. 不足与局限
*   **依赖 2D 基础模型**：如果 SAM 的掩码质量差或 DINO 的语义特征在某些区域（如小物体、遮挡处）失效，H2G 的层级树构建也会受影响。
*   **跨视图一致性**：层级树是逐图像构建的，虽然通过 3D 蒸馏缓解了矛盾，但在极端视角变化下仍可能存在不一致。
*   **后处理依赖**：最终掩码的提取仍然需要手动选择阈值或使用 HDBSCAN 等聚类算法，尚未实现完全自动的端到端层级解析。

## 摘要
最近的基于前馈的几何基础模型通过在单次前向传播中恢复深度和姿态，展示了令人印象深刻的泛化能力。然而，这些模型通常受到全局坐标系假设的限制。这种依赖关系成为了长上下文和流式重建的一个显著瓶颈，因为它迫使网络维护一个任意的时间原点，并处理随时间无界增长的平移幅度。我们的解决方案，称为$R^3$，采用了相对回归。我们使用一个轻量级的MLP来预测置信度加权的相对约束。这些置信度作为一个统一的锚点：在训练中加权损失，在推理中指导姿态聚合。$R^3$既支持全上下文的离线重建，也支持因果的、有界内存的流式重建。我们在离线与流式设置下的评估验证了我们的相对机制的有效性。项目页面：https://kevinxu02.github.io/r3-site

## Abstract
Recent feed-forward geometry foundation models have demonstrated impressive generalization by recovering depth and poses in a single forward pass. However, these models are typically constrained by a global coordinate frame assumption. This dependency becomes a significant bottleneck for long-context and streaming reconstruction, as it forces the network to maintain an arbitrary temporal origin and handle translation magnitudes that grow unbounded over time. Our solution, which we call $R^3$, employs relative regression. We employ a lightweight MLP to predict confidence-weighted relative constraints. These confidences serve as a unified anchor: weighting losses during training and guiding pose aggregation during inference. $R^3$ supports both full-context offline reconstruction and causal, bounded-memory streaming. Our evaluation in both offline and streaming settings validates the effectiveness of our relative mechanism. Project page: https://kevinxu02.github.io/r3-site

### 1. 论文的核心问题与整体含义（研究动机和背景）

- **核心问题**：当前前馈式三维重建基础模型（如 VGGT、π³、DA3）依赖于**全局坐标系假设**——它们要么预测共享帧中的点图，要么将所有相机姿态相对于一个序列级的世界帧（如第一帧）进行回归。这种设计在短离线片段中工作良好，但在长视频和流式重建中成为瓶颈：网络必须维护一个任意的时间原点，并处理随时间无界增长的平移幅度，导致坐标漂移和内存膨胀。
- **整体含义**：本文提出应该从**学习局部相对关系**入手，将全局姿态回归转变为**成对相对姿态回归**，再通过下游聚合得到全局轨迹。这样相对位移的尺度仅取决于两帧之间的基线，而非总轨迹长度，能够自然支持长序列和流式场景。

### 2. 论文提出的方法论：核心思想、关键技术细节

- **核心思想**：采用**相对回归**框架，不直接回归全局姿态，而是预测每对帧之间的相对变换（旋转 + 平移）及对应的**可分离置信度**（旋转置信度和平移置信度）。这些置信度作为统一锚点：在训练中加权损失，在推理中指导姿态聚合和关键帧管理。
- **关键技术细节**：
  - **相对姿态预测头**：基于 DA3 骨干网络提取每帧的相机 token z_i，使用轻量级 MLP 处理 token 对 (z_i, z_j)，输出相对旋转（单位四元数）、相对平移，以及两个标量置信度。
  - **置信度设计**：旋转和平移分开预测，因为两者的失败模式不同（平移在弱几何线索时更不可靠）。置信度在训练中与残差乘积形成损失（c·ℓ - α log c），迫使模型对不准确对降低置信度。
  - **全局轨迹聚合**：流式时，使用关键帧库作为参考集；对每个新帧 j，从每个参考帧 i 通过组合绝对姿态与相对姿态得到候选姿态，再用**置信度加权 softmax 融合**得到最终姿态（旋转和平移分别使用各自的置信度权重）。支持 top-K 效率上限。
  - **因果流式推理**：帧依次到达，仅与活跃上下文（第1帧 + 动态关键帧库）配对。关键帧库通过**token 新颖性阈值**（预注意力 token 余弦相似度）准入，满容量时基于效用 u_j = d_j * c_j 淘汰最无用帧（d_j 是 token 独特性，c_j 是最高成对置信度）。
  - **全上下文推理**：测试时移除因果注意力掩码，使骨干同时看到所有帧，再运行一次轻量级置信度加权姿态图优化（仅姿态残差，无光束法平差或深度优化）。

## 摘要
我们研究了从二维图像和初始三维形状再生三维物体的问题。大多数三维生成器以单次（one-shot）方式运行，将文本或图像转换为三维物体，但可控性有限。相比之下，我们引入了 3D-ReGen，这是一种以初始三维形状为条件的 3D 再生器。这种概念上简单的表述使我们能够支持许多有用的任务，包括三维增强、重建和编辑。3D-ReGen 使用了一种基于 VecSet 的新型条件机制，允许再生器以一致的细粒度细节更新或改进输入几何。3D-ReGen 通过自监督代理任务和数据增强，从现成的三维数据集中学习广泛适用的再生先验，无需额外标注。我们评估了 3D-ReGen 的几何一致性和细粒度质量，在多项任务的可控三维生成中实现了最先进的性能。

## Abstract
We consider the problem of regenerating 3D objects from 2D images and initial 3D shapes. Most 3D generators operate in a one-shot fashion, converting text or images to a 3D object with limited controllability. We introduce instead 3D-ReGen, a 3D regenerator that is conditioned on an initial 3D shape. This conceptually simple formulation allows us to support numerous useful tasks, including 3D enhancement, reconstruction, and editing. 3D-ReGen uses a new conditioning mechanism based on VecSet, which allows the regenerator to update or improve the input geometry with consistent fine-grained details. 3D-ReGen learns a widely applicable regeneration prior from off-the-shelf 3D datasets via self-supervised pretext tasks and augmentations, without additional annotations. We evaluate both the geometric consistency and fine-grained quality of 3D-ReGen, achieving state-of-the-art performance in controllable 3D generation across several tasks.

传统的 3D 生成模型（如 Text-to-3D 或 Image-to-3D）大多采用“一次性”生成模式，用户对生成过程的干预手段有限，难以实现精确的几何控制。论文提出了**3D 再生（3D Regeneration）**的概念，旨在解决如何基于给定的初始（通常是低质量、不完整或粗糙的）3D 形状和 2D 图像参考，重新生成高质量、细节丰富的 3D 模型。这一框架统一了 3D 增强、3D 重建和 3D 编辑等多种下游任务，填补了生成式 AI 在 3D 内容创作工作流中“迭代优化”环节的空白。

#### 2. 方法论
*   **核心思想**：利用潜空间扩散模型（Latent Diffusion Model），将初始 3D 形状作为条件信号注入生成过程。
*   **关键技术细节**：
    *   **VecSet 潜空间**：采用 VecSet 作为 3D 表示，将 3D 形状编码为一组潜向量（Tokens）。这种表示法比体素（Voxel）更灵活，能根据形状复杂度调整 Token 数量。
    *   **条件注入机制**：不同于常见的交叉注意力（Cross-attention），3D-ReGen 将条件形状的潜向量 $z_L$ 与带噪声的目标潜向量 $z_H$ 在 Token 维度上直接**拼接（Concatenation）**。
    *   **零初始化 MLP 与位置编码**：在拼接前，对条件 Token 应用零初始化的 MLP（类似于 ControlNet 的思想）以避免训练初期的震荡，并加入位置编码以保持空间一致性。
    *   **自监督数据构建**：通过对高质量 3D 数据进行自动退化处理（如添加噪声、遮罩、低分辨率采样、多视图重建模拟等），在无需人工标注的情况下生成海量的“粗糙-精细”配对数据。

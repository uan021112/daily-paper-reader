## 摘要
我们研究了从 2D 图像和初始 3D 形状中再生 3D 对象的问题。大多数 3D 生成器以单次（one-shot）方式运行，将文本或图像转换为 3D 对象，但可控性有限。与此不同，我们引入了 3D-ReGen，这是一种以初始 3D 形状为条件的 3D 再生器。这种概念上简单的表述使我们能够支持许多有用的任务，包括 3D 增强、重建和编辑。3D-ReGen 使用了一种基于 VecSet 的新型条件机制，允许再生器以一致的细粒度细节更新或改进输入几何。3D-ReGen 通过自监督代理任务和增强，从现成的 3D 数据集中学习广泛适用的再生先验，无需额外标注。我们评估了 3D-ReGen 的几何一致性和细粒度质量，在多项任务的可控 3D 生成中实现了最先进的性能。

## Abstract
We consider the problem of regenerating 3D objects from 2D images and initial 3D shapes. Most 3D generators operate in a one-shot fashion, converting text or images to a 3D object with limited controllability. We introduce instead 3D-ReGen, a 3D regenerator that is conditioned on an initial 3D shape. This conceptually simple formulation allows us to support numerous useful tasks, including 3D enhancement, reconstruction, and editing. 3D-ReGen uses a new conditioning mechanism based on VecSet, which allows the regenerator to update or improve the input geometry with consistent fine-grained details. 3D-ReGen learns a widely applicable regeneration prior from off-the-shelf 3D datasets via self-supervised pretext tasks and augmentations, without additional annotations. We evaluate both the geometric consistency and fine-grained quality of 3D-ReGen, achieving state-of-the-art performance in controllable 3D generation across several tasks.

传统的 3D 生成模型（如文本转 3D 或图像转 3D）大多采用“单次生成”模式，用户对生成结果的几何控制力较弱。在实际的艺术创作流中，艺术家通常需要从粗糙的草图、低分辨率扫描件或部分遮挡的形状开始，进行迭代式的精细化和编辑。
**3D-ReGen** 的核心目标是解决 **3D 几何再生（Regeneration）** 问题，即：如何在一个统一的框架下，结合 2D 图像引导和初始的、低质量的 3D 形状（如粗糙网格、不完整点云、带掩码的形状），生成高质量、细节丰富且几何一致的 3D 模型。

#### 2. 论文提出的方法论
该论文提出了一种基于扩散模型（Diffusion Model）的统一架构，核心思想是将初始 3D 形状作为条件信号注入生成过程。

*   **核心架构：** 基于 **VecSet** 潜空间表示。VecSet 将 3D 形状编码为一组潜向量（Tokens），并使用 Transformer 进行处理。
*   **条件注入机制：** 
    *   使用相同的 VecSet 编码器将“低信息”初始形状 $y_L$ 编码为条件 Token $z_L$。
    *   **Token 拼接策略：** 不同于常见的交叉注意力（Cross-Attention），3D-ReGen 直接将条件 Token $z_L$ 与带噪声的潜变量 $z_H$ 在 Token 维度进行拼接。
    *   **零初始化 MLP：** 在拼接前，通过一个零初始化的 MLP 处理条件 Token，以避免在训练初期对预训练模型产生剧烈冲击（类似于 ControlNet 的思想）。
*   **自监督学习流程：** 
    *   为了解决配对数据（高质量 vs 低质量 3D 形状）稀缺的问题，论文设计了自动化的数据增强管线。
    *   通过对高质量 3D 模型进行噪声扰动、降采样、遮挡掩码等操作，自动生成训练所需的“低信息”初始形状。
*   **多模态引导：** 结合 DINOv2 提取的图像特征，通过交叉注意力机制引导生成过程，确保 3D 几何与 2D 参考图一致。

#### 6. 主要结论与发现
*   **统一框架的有效性：** 单一架构通过不同的自监督预设任务，可以同时胜任增强、重建和编辑任务。
*   **拼接优于注意力：** 在处理密集或高噪声的 3D 条件时，直接在潜空间拼接 Token 比使用交叉注意力更能保留几何细节。
*   **自监督数据的力量：** 通过简单的几何退化算法（如 SDEdit 噪声、网格简化）生成的合成数据，足以训练出能够处理真实世界复杂退化（如摄影测量噪声）的模型。
*   **保真度提升：** 相比于纯图像生成模型，引入初始 3D 形状能显著提升生成结果的位姿准确性和几何保真度。

#### 7. 优点
*   **高可控性：** 解决了 3D 生成中“位姿难对齐”和“细节难控制”的痛点。
*   **架构简洁：** 最小化了对预训练 DiT 结构的修改，易于集成到现有的 3D 生成管线中。
*   **泛化能力：** 能够处理从极简的“方块拼搭（Block-outs）”到复杂的“损坏扫描件”等多种输入。
*   **效率优化：** 利用 VecSet 特性，对粗糙输入使用较少的 Token，降低了计算开销。

#### 8. 不足与局限
*   **动态性缺失：** 目前仅限于静态物体，尚未扩展到 4D（动态）或带骨骼绑定的资产。
*   **潜在的伪影：** 在极端遮挡或图像与 3D 形状严重冲突时，模型可能会产生浮点（Floaters）或几何畸变。
*   **依赖初始质量：** 虽然能增强细节，但如果初始形状的拓扑结构存在严重错误，再生过程可能难以完全修复。
*   **算力门槛：** 2.8B 的参数量和庞大的训练步数意味着该模型难以在消费级硬件上进行全量训练或微调。

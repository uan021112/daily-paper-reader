## 摘要
近期的前馈模型在从传感器观测中推断密集三维结构的几何感知方面取得了显著进展。然而，其关键能力仍然分散在多个不兼容的范式中，包括在线感知、离线重建、多模态集成、长时程可扩展性以及度量尺度估计。我们提出了UniT，这是一个基于新型分组自回归变换器构建的统一模型，它将看似分散的这些能力重新整合到单一框架中。关键思想是将传感器观测分组视为基本的自回归单元，并以无锚点和尺度自适应的方式预测对应的点图。具体而言，在线和离线设置中的多样化视角配置被自然地统一到单一的分组自回归过程中。通过改变组大小，在线模式使用单帧分组在多个自回归步骤中运行，而离线模式则在单次前向传递中聚合多帧分组。同时，队列式的KV缓存机制确保了在长时程上的有界自回归内存。这是通过无锚点关系建模减少对早期帧的长距离依赖来实现的，从而允许在运行中丢弃过时的记忆。为了提高跨场景的度量尺度泛化能力，该框架进一步引入了尺度自适应几何损失。它将相对几何约束与部分绝对尺度项耦合，隐式地正则化全局尺度，并诱导从尺度不变几何到度量尺度解的渐进过渡。结合用于集成辅助模态的专用模态注意力模块，UniT在统一几何感知中实现了最先进的性能，这一点在涵盖七个代表性任务的十个基准上得到了验证。

## Abstract
Recent feed-forward models have significantly advanced geometry perception for inferring dense 3D structure from sensor observations. However, its essential capabilities remain fragmented across multiple incompatible paradigms, including online perception, offline reconstruction, multi-modal integration, long-horizon scalability, and metric-scale estimation. We present UniT, a unified model built upon a novel Group Autoregressive Transformer, which reformulates these seemingly disparate capabilities within a single framework. The key idea is to treat groups of sensor observations as the basic autoregressive units and predict the corresponding point maps in an anchor-free and scale-adaptive manner. More specifically, diverse view configurations in both online and offline settings are naturally unified within a single group autoregression process. By varying the group size, online mode operates over multiple autoregressive steps with single-frame groups, whereas offline mode aggregates a multi-frame group in a single forward pass. Meanwhile, a queue-style KV caching mechanism ensures bounded autoregressive memory over long horizons. This is enabled by reducing long-range dependencies on early frames through anchor-free relational modeling, thereby allowing outdated memory to be discarded on the fly. To improve metric-scale generalization across scenes, a scale-adaptive geometry loss is further introduced within this framework. It couples relative geometric constraints with a partial absolute scale term, implicitly regularizing global scale and inducing a progressive transition from scale-invariant geometry to metric-scale solutions. Together with a dedicated modal attention module for integrating auxiliary modalities, UniT achieves state-of-the-art performance in unified geometry perception, as validated on ten benchmarks spanning seven representative tasks.

### 1. 核心问题与整体含义（研究动机和背景）
- **问题**：现有前馈几何感知模型的能力高度碎片化：在线感知（流式连续处理）、离线重建（批量联合优化）、多模态融合（深度图、相机参数等辅助信号）、长时程可扩展性（长序列推理）、度量尺度估计（物理世界真实尺度）这五个关键能力分散在不同的、互不兼容的范式中。例如，CUT3R专做流式感知，VGGT专做离线重建，MapAnything专注于多模态度量重建。缺乏一个能同时支持所有能力的统一框架。
- **整体目标**：提出UniT，一个基于**分组自回归变换器**的统一前馈模型，将上述分散能力整合到单一框架中，实现“一个模型，多种几何感知任务”。

### 2. 方法论：核心思想、关键技术细节
- **核心思想**：将传感器观测的**组**作为基本自回归单元，以**无锚点（anchor-free）** 和**尺度自适应**的方式逐组预测对应的3D点图。
- **关键技术细节**：
    - **分组自回归公式化**：将联合条件概率分解为逐组自回归：预测当前组（大小为G）的点图时，仅依赖该组及之前所有组的观测。当G=1时退化为标准在线自回归（逐帧预测）；当G=N时退化为离线单步预测；G取中间值时（如4-8）适配多相机阵列流。
    - **分组因果连接**：全局注意力中，“组内”使用双向注意力（全可见），“组间”使用因果注意力（只能看过去组，不能看未来组）。通过调整组因果掩码即可灵活切换视角配置。
    - **队列式KV缓存（Queue-Style KV Caching）**：利用无锚点关系建模（π3的anchor-free设计）使模型不再依赖固定的第一帧坐标系，从而可以只维护固定容量Q的KV缓存队列。当序列长度超过Q时，丢弃过时的KV条目（例如采用步长丢弃策略），实现**有界内存复杂度O(Q)**，而非线性增长O(N)。
    - **模态注意力（Modal Attention）**：一种改进的交叉注意力模块，将图像token与多模态token（深度图、内参、外参等，经MLP编码）在**空间对齐位置拼接**，然后进行交叉注意力。只在四个特定层插入（约占总参数量3%），以控制复杂度。
    - **尺度自适应几何损失**：将度量尺度学习分解为两个部分：
        - **相对项**（尺度不变损失）：施加于局部点图和相机外参，强制预测与GT在尺度归一化下一致（L_point_rel, L_cam_rel）。
        - **绝对项**（部分绝对损失）：仅施加于局部点图的L1回归，配合置信度图（L_point_abs）。
        - **课程学习效应**：模型先学习更容易的尺度不变几何，再逐渐恢复度量尺度，避免直接回归大动态范围尺度带来的不稳定。
    - **额外正则化**：引入洗牌法线损失（shuffled normal loss）促经跨帧全局几何一致性；常规法线损失保持局部平滑性。
- **公式说明**：总体损失 L = L_cam_rel + L_point_rel + L_point_abs + L_snormal + L_normal。

### 3. 实验设计：数据集、基准、对比方法
- **数据集**：训练集包含21个公共度量尺度数据集，涵盖室内（ScanNet++、ARKitScenes等）、室外（Waymo、Mapfree等）、物体（WildRGBD、Omniobject3D）、人类（HuMMan）等多种场景，真实和合成数据兼有。测试使用10个基准：7-Scenes, NRGBD, DTU, Sintel, TUM-Dynamic, ScanNetv2, Bonn, ETH3D, KITTI, NYUv2。
- **任务与基准**：覆盖7个代表性任务：多视角重建、相机位姿估计、视频深度估计、单目深度估计、长时程感知、多模态重建、深度补全。每个任务在2~3个数据集上评测。
- **对比方法**：6种基线：VGGT (离线、尺度不变)、π3 (离线、尺度不变)、MapAnything (离线、度量、多模态)、DepthAnything3 (离线、度量、多模态)、CUT3R (在线、度量)、StreamVGGT (在线、尺度不变)。对比同时涵盖尺度不变和度量尺度两种设置。
- **评估指标**：点图：Accuracy, Completion, Normal Consistency；位姿：ATE, RPE_trans, RPE_rot；深度：AbsRel, RMSE, δ<1.25。最终汇总平均排名（Rank）。

### 6. 主要结论与发现
- UniT是**首个统一在线感知、离线重建、多模态融合、长时程可扩展、度量尺度估计**这五大能力的单一前馈模型，并在10个基准的7个任务上达到**最先进（SOTA）** 水平，尤其在度量尺度设置下优势明显。
- 分组自回归配方（通过调节组大小G）能自然衔接在线和离线行为，避免冗余计算或重复重计算。
- 队列式KV缓存结合无锚点设计，在不牺牲性能的前提下将长时程推理的内存复杂度从O(N)降低到O(Q)，实现了实际有界内存。
- 尺度自适应损失自动产生课程学习效果，显著提升了度量尺度泛化，相比直接回归损失提升巨大（表X）。

### 7. 优点：方法或实验设计上的亮点
- **统一性极强**：单个模型覆盖从单目深度到多视角重建、从短序列到长时程、从纯RGB到多模态的多种设置，无需切换模型或后处理。
- **设计简洁高效**：仅增加约3%参数（模态注意力层），即可灵活集成多模态信息。队列KV缓存思想简单（丢弃过时KV），但利用无锚点设计从根本上消除了长依赖，而非采用复杂压缩。
- **实验设计全面**：对每个关键设计（模态注意力、损失函数、KV缓存策略、组大小、队列容量）都进行了系统消融，且消融实验在低分辨率下进行，公平对比。
- **可复现性好**：代码和模型已开源，训练和数据采样策略详细描述。

### 8. 不足与局限
- **全模态场景优势不绝对**：当深度图、内参、外参全部提供时，MapAnything在多数指标上略优于UniT。可能因为UniT以VGGT预训练初始化，而VGGT是纯RGB模型，对多模态联合建模可能不够充分。此外，多模态采样概率（50%独立）可能导致对全模态场景的过拟合不足。
- **长时程精度仍有退化**：在线模式（G=1）长时程（>300帧）时，位姿和深度误差随帧数增加而缓慢增长，虽然比CUT3R鲁棒，但相比离线模式仍有明显差距。队列容量Q越大性能越好，但Q过大又增加显存；当前策略（Q=N/3）是折中。
- **动态场景与运动模糊**：论文未专门讨论动态物体或运动模糊处理，而测试集（如TUM-Dynamic、Bonn）包含动态内容，但方法本身并未显式建模运动，可能依赖多视图一致性鲁棒处理。对于严重运动场景，性能未分析。
- **算力需求高**：训练需要64块H100运行7天，对一般研究组门槛较高。尽管推理可以在单卡RTX 4090上运行，但高分辨率（518）下显存开销仍较大。

## 摘要
草图绘制被广泛应用于犯罪现场调查（CSI），用于记录、解释和传达空间信息。然而，它通常在二维介质上进行，这限制了其表现三维空间关系的能力。我们提出了 HolmeSketcher，这是一个生成式三维草图绘制系统，它将前端三维绘图界面与后端深度学习流水线相结合，以支持扩展现实中的物体生成和场景重建。在一项受试者内用户研究（N = 15）中，与基于纸张的二维草图绘制相比，HolmeSketcher 提高了重建场景的空间准确性和可解释性，但同时也带来了更高的任务负荷和更低的可用性。通过整合用户研究和专家访谈（N = 3）的结果，我们进一步为下一代犯罪现场调查三维草图绘制工具提出了三项设计启示。

## Abstract
Sketch mapping is widely used in crime scene investigation (CSI) to document, interpret, and communicate spatial information. However, it is typically performed on 2D media, which limits its ability to represent 3D spatial relationships. We present HolmeSketcher, a generative 3D sketch mapping system that combines a front-end 3D drawing interface with a back-end deep learning pipeline to support object generation and scene reconstruction in extended reality. In a within-subject user study (N = 15), HolmeSketcher improved the spatial accuracy and interpretability of reconstructed scenes, but with a clear trade-off of higher task load and lower usability compared with paper-based 2D sketch mapping. By integrating findings from the user study and expert interviews (N = 3), we further derive three design implications for next-generation 3D sketch mapping tools for CSI.

### 6. 主要结论与发现
*   **空间性能**：3D 草图在**准确性（位置和尺寸）和可解释性**上显著优于 2D 草图。3D 系统的 ODA 和 OPA 得分是 2D 方法的两倍以上。
*   **用户体验权衡**：尽管 3D 系统更具创新性（Novelty）和趣味性，但其**可用性（SUS）显著低于 2D 纸笔**。
*   **任务负荷**：3D 绘制带来了更高的心理负荷、物理负荷和时间压力。用户在 3D 环境中进行物体精细操作（如旋转和对齐）时感到困难。
*   **专家观点**：专家认为 3D 草图在法庭展示和空间推理中有巨大潜力，但目前尚不能完全取代物理重建，且学习成本是实际部署的主要障碍。

### 7. 优点
*   **技术融合**：成功将生成式 AI（草图转 3D）与沉浸式 XR 结合，解决了非专业人士“难以画出 3D 模型”的痛点。
*   **多模态交互**：引入语音生成材质和环境配置（天气、光照），极大地丰富了犯罪现场重建的上下文信息。
*   **认知支持**：通过 1:1 比例的虚拟环境，增强了用户的空间记忆提取能力。

### 8. 不足与局限
*   **交互复杂性**：物体操作（Gizmo 交互）在 VR 中不够直观，导致任务负荷过重。
*   **动态表达缺失**：目前系统主要针对静态物体和布局，无法有效重建动态的犯罪过程（如打斗动作、逃跑轨迹）。
*   **硬件依赖**：系统依赖高性能工作站和 VR 设备，且存在晕动症风险，限制了其在真实一线调查中的快速部署。
*   **样本局限**：受试者多为年轻人，缺乏对老年证人或低技术素养人群的覆盖。

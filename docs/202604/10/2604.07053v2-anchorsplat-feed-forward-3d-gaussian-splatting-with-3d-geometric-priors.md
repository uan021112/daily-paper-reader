## 摘要
最近的前馈高斯重建模型采用了像素对齐的方案，将每个2D像素映射到3D高斯，使高斯表示与输入图像紧密耦合。在本文中，我们提出了 AnchorSplat，这是一种用于场景级重建的新型前馈3DGS框架，直接在3D空间中表示场景。AnchorSplat 引入了由3D几何先验（如稀疏点云、体素或 RGB-D 点云）引导的锚点对齐高斯表示，从而实现了更具几何感知能力且独立于图像分辨率和视图数量的可渲染3D高斯。这种设计显著减少了所需的高斯数量，在提高计算效率的同时增强了重建保真度。除了锚点对齐设计外，我们还利用高斯精炼器（Gaussian Refiner）通过仅几次前馈过程来调整中间高斯。在 ScanNet++ v2 NVS 基准测试上的实验证明了其 SOTA 性能，在视图一致性更好且高斯原语数量显著减少的情况下优于以往的方法。

## Abstract
Recent feed-forward Gaussian reconstruction models adopt a pixel-aligned formulation that maps each 2D pixel to a 3D Gaussian, entangling Gaussian representations tightly with the input images. In this paper, we propose AnchorSplat, a novel feed-forward 3DGS framework for scene-level reconstruction that represents the scene directly in 3D space. AnchorSplat introduces an anchor-aligned Gaussian representation guided by 3D geometric priors (e.g., sparse point clouds, voxels, or RGB-D point clouds), enabling a more geometry-aware renderable 3D Gaussians that is independent of image resolution and number of views. This design substantially reduces the number of required Gaussians, improving computational efficiency while enhancing reconstruction fidelity. Beyond the anchor-aligned design, we utilize a Gaussian Refiner to adjust the intermediate Gaussiansy via merely a few forward passes. Experiments on the ScanNet++ v2 NVS benchmark demonstrate the SOTA performance, outperforming previous methods with more view-consistent and substantially fewer Gaussian primitives.

*   **核心问题**：现有的前馈（Feed-Forward）3D高斯重建模型大多采用“像素对齐”（Pixel-aligned）方案，即将每个2D像素映射为一个3D高斯。
*   **研究背景与痛点**：
    *   **冗余性**：高斯原语的数量随输入视图和分辨率线性增长，导致计算负担重。
    *   **几何不一致**：由于缺乏3D空间的显式约束，容易产生浮点伪影（floaters）、表面破碎和模糊。
    *   **视图依赖**：在遮挡或低纹理区域，像素对齐方法难以保持跨视图的采样一致性。
*   **研究目标**：提出一种直接在3D空间中表示场景、且与图像分辨率和视图数量解耦的重建框架。

### 2. 方法论
AnchorSplat 的核心思想是**锚点对齐（Anchor-aligned）**，通过3D几何先验引导高斯分布。其流程包含三个关键组件：
*   **锚点预测器 (Anchor Predictor)**：
    *   利用预训练的 MVS 模型（如 MapAnything）从无位姿图像中预测深度和相机位姿。
    *   将深度图反投影至3D空间，并使用**最远点采样（FPS）**算法进行下采样，生成稀疏且分布均匀的3D锚点（Anchors）。
*   **高斯解码器 (Gaussian Decoder)**：
    *   使用 CNN 提取多视图特征，并将其投影到3D锚点上。
    *   通过 **Transformer** 建模锚点间的空间交互，预测每个锚点对应的 4 个高斯原语的属性（中心偏移 $\delta\mu$、不透明度 $\alpha$、缩放 $s$、旋转 $r$ 和球谐系数 $sh$）。
*   **高斯精炼器 (Gaussian Refiner)**：
    *   这是一个即插即用的模块。它计算渲染图像与真值的误差（Rendering Error），将误差特征反投影回3D空间。
    *   利用 **Point Transformer** 根据误差反馈微调高斯属性，进一步提升几何边缘的锐利度和颜色一致性。

*   **高效性**：AnchorSplat 使用的高斯数量仅为 AnySplat 的 **1/20** 左右，重建时间缩短了约一半。

*   **解耦设计**：成功将 3D 表示与 2D 像素分辨率解耦，解决了前馈模型中常见的高斯冗余问题。

## 摘要
最近的前馈高斯重建模型采用了像素对齐的表征方式，将每个 2D 像素映射到一个 3D 高斯，使得高斯表示与输入图像紧密耦合。在本文中，我们提出了 AnchorSplat，这是一种用于场景级重建的新型前馈 3DGS 框架，它直接在 3D 空间中表示场景。AnchorSplat 引入了由 3D 几何先验（例如稀疏点云、体素或 RGB-D 点云）引导的锚点对齐高斯表示，从而实现了更具几何感知能力的可渲染 3D 高斯，且该表示独立于图像分辨率和视图数量。这种设计显著减少了所需的高斯数量，在提高计算效率的同时增强了重建保真度。除了锚点对齐设计外，我们还利用高斯精细化器（Gaussian Refiner），仅通过几次前向传递即可调整中间高斯。在 ScanNet++ v2 NVS 基准上的实验证明了其 SOTA 性能，在视图一致性更高且高斯基元数量显著减少的情况下，优于以往的方法。

## Abstract
Recent feed-forward Gaussian reconstruction models adopt a pixel-aligned formulation that maps each 2D pixel to a 3D Gaussian, entangling Gaussian representations tightly with the input images. In this paper, we propose AnchorSplat, a novel feed-forward 3DGS framework for scene-level reconstruction that represents the scene directly in 3D space. AnchorSplat introduces an anchor-aligned Gaussian representation guided by 3D geometric priors (e.g., sparse point clouds, voxels, or RGB-D point clouds), enabling a more geometry-aware renderable 3D Gaussians that is independent of image resolution and number of views. This design substantially reduces the number of required Gaussians, improving computational efficiency while enhancing reconstruction fidelity. Beyond the anchor-aligned design, we utilize a Gaussian Refiner to adjust the intermediate Gaussiansy via merely a few forward passes. Experiments on the ScanNet++ v2 NVS benchmark demonstrate the SOTA performance, outperforming previous methods with more view-consistent and substantially fewer Gaussian primitives.

### 2. 方法论
AnchorSplat 提出了**锚点对齐（Anchor-aligned）**的 3DGS 框架，直接在 3D 空间中表示场景。
*   **核心思想**：利用 3D 几何先验（如稀疏点云）作为“锚点”，在锚点周围生成高斯基元，从而使表示独立于图像分辨率。
*   **关键技术流程**：
    1.  **锚点预测器（Anchor Predictor）**：利用预训练的 MVS 模型（如 MapAnything）预测深度图和相机位姿，通过反投影将像素提升至 3D 空间，并使用最远点采样（FPS）下采样为稀疏锚点。
    2.  **高斯解码器（Gaussian Decoder）**：使用 CNN 提取多视图特征，通过 Transformer 建模锚点间的空间交互，为每个锚点预测 4 个高斯基元的属性（位置偏移、不透明度、缩放、旋转、球谐系数）。
    3.  **高斯精炼器（Gaussian Refiner）**：一个即插即用的模块。它计算当前渲染图与真值的误差，将误差特征反投影回 3D 空间，通过 Point Transformer 进一步微调高斯属性，提升细节保真度。
*   **损失函数**：结合了渲染损失（$L_1$, SSIM, LPIPS）、深度损失以及针对高斯不透明度和体积的正则化项。

### 6. 主要结论与发现
*   **高效性**：AnchorSplat 使用的高斯基元数量仅为 AnySplat 的 **1/20** 左右，但重建质量更高。
*   **几何一致性**：锚点对齐设计显著减少了浮空伪影，生成的几何结构更清晰、更符合物理真实。
*   **稳定性**：由于高斯数量由空间锚点决定，模型性能不会随输入视角增加而导致计算量爆炸，表现出良好的可扩展性。
*   **精炼器作用**：Refiner 模块能有效修复解码器生成的空洞并增强边缘锐度。

### 7. 优点与亮点
*   **架构创新**：打破了像素对齐的思维定式，将前馈重建从 2D 像素索引转向 3D 空间锚点索引。
*   **即插即用**：Gaussian Refiner 可以在不重新训练主网络的情况下提升性能，具有很强的实用性。
*   **性能与效率双赢**：在大幅降低显存占用和渲染压力的同时，刷新了前馈重建的 SOTA 指标。

### 8. 不足与局限
*   **先验依赖**：模型高度依赖初始几何先验（MVS 预测的深度）。如果 MVS 模块在极端环境下失效或产生巨大误差，AnchorSplat 难以在完全空白或错误的区域“生长”出正确的高斯。
*   **空洞覆盖**：由于高斯数量和分布受限于锚点预算，对于先验点云极其稀疏的区域，可能会出现覆盖不足导致的渲染空洞。
*   **动态场景**：目前仅针对静态场景设计，尚未讨论在动态或时序场景中的应用。

## 摘要
关节物体对于具身智能和世界模型至关重要，但从单张闭合状态的图像中推断其运动学仍然具有挑战性，因为关键的运动线索往往被遮挡。现有方法要么需要多状态观测，要么依赖于显式的部件先验、检索或其他部分暴露待推断结构的辅助输入。在这项工作中，我们提出了 DailyArt，它将单张静态图像中的关节估计建模为一个由合成介导的推理问题。DailyArt 并非直接从严重遮挡的观测中回归关节，而是首先在相同相机视角下合成一个最大程度张开的关节状态以暴露关节线索，然后根据观测状态与合成状态之间的差异估计全套关节参数。通过使用集合预测公式，DailyArt 可以同时恢复所有关节，且在测试时无需特定物体的模板、多视图输入或显式的部件标注。以估计的关节为条件，该框架进一步支持部件级的新状态合成作为下游能力。大量实验表明，DailyArt 在关节估计方面表现出色，并支持以关节为条件的部件级新状态合成。项目页面见 https://rangooo123.github.io/DaliyArt.github.io/。

## Abstract
Articulated objects are essential for embodied AI and world models, yet inferring their kinematics from a single closed-state image remains challenging because crucial motion cues are often occluded. Existing methods either require multi-state observations or rely on explicit part priors, retrieval, or other auxiliary inputs that partially expose the structure to be inferred. In this work, we present DailyArt, which formulates articulated joint estimation from a single static image as a synthesis-mediated reasoning problem. Instead of directly regressing joints from a heavily occluded observation, DailyArt first synthesizes a maximally articulated opened state under the same camera view to expose articulation cues, and then estimates the full set of joint parameters from the discrepancy between the observed and synthesized states. Using a set-prediction formulation, DailyArt recovers all joints simultaneously without requiring object-specific templates, multi-view inputs, or explicit part annotations at test time. Taking estimated joints as conditions, the framework further supports part-level novel state synthesis as a downstream capability. Extensive experiments show that DailyArt achieves strong performance in articulated joint estimation and supports part-level novel state synthesis conditioned on joints. Project page is available at https://rangooo123.github.io/DaliyArt.github.io/.

### 2. 方法论
DailyArt 采用了一个三阶段的流水线：
*   **Stage I：新状态合成（发现线索）**
    *   **核心思想**：在不指定具体部件的情况下，将单张闭合图像 $I_0$ 映射到其最大开启状态 $\hat{I}_1$。
    *   **技术细节**：使用预训练的 DINOv2 作为编码器提取语义特征，通过自适应层归一化（AdaLN）注入标量状态索引 $t$（$t=0$ 为闭合，$t=1$ 为全开），由 VAE 解码器生成图像。此阶段旨在暴露潜在的运动部件。
*   **Stage II：3D 感知关节估计（几何推理）**
    *   **核心思想**：对比 $I_0$ 与合成的 $\hat{I}_1$ 之间的差异。
    *   **技术细节**：利用视觉几何变换器（VGGT）将图像对提升为 3D 点图。通过计算每像素的 3D 位移提取“运动种子点”，并使用基于 Transformer 的集合预测（Set-prediction）公式，通过匈牙利匹配（Hungarian matching）同时回归所有关节的参数（类型、原点、轴向、范围）。
*   **Stage III：关节条件合成（下游应用）**
    *   **核心思想**：将 Stage II 估计的精确关节参数作为显式条件反馈给合成模型。
    *   **功能**：支持对特定部件进行精细的、符合物理逻辑的运动序列合成。

### 6. 主要结论与发现
*   **合成辅助推理的有效性**：通过合成开启状态来减少 2D 观测的歧义性，比直接从单图回归关节参数的效果提升了约 24.2% 的成功率。
*   **3D 几何的重要性**：将图像对提升到 3D 空间进行位移分析，能显著提高关节轴和原点估计的准确性，避免了 2D 空间中看似合理但在 3D 中错误的几何预测。
*   **泛化能力**：尽管在合成数据上训练，模型在真实世界图像（AKB-48）上表现出良好的零样本迁移能力。

### 7. 优点（亮点）
*   **无需先验**：测试时不需要部件掩码、文本提示、CAD 模型或多视图输入，仅需单张 RGB 图像。
*   **闭环设计**：估计出的关节参数可以直接用于下游的可控合成，形成了一个从感知到生成的完整闭环。
*   **端到端集合预测**：能够一次性处理具有多个关节的复杂物体，无需预设关节数量。

### 8. 不足与局限
*   **误差累积**：Stage II 的估计精度高度依赖于 Stage I 合成图像的质量，如果合成出现严重的伪影或几何扭曲，会导致关节估计失败。
*   **极端视角限制**：当物体背面朝向相机或运动完全被遮挡时，3D 提升模块可能失效。
*   **状态定义**：假设物体具有明确的“闭合”和“最大开启”状态，对于没有清晰运动边界的物体，跨状态对应关系可能变差。
*   **工业应用**：目前主要针对消费级物体，对于结构极其复杂的工业设备关节可能尚显不足。

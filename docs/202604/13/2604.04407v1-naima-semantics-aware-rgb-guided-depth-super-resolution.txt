# Abstract 

Guided depth super-resolution (GDSR) is a multi-modal approach for depth map super-resolution that relies on a low-resolution depth map and a high-resolution RGB image to restore finer structural details. However, the misleading color and texture cues indicating depth dis-continuities in RGB images often lead to artifacts and blurred depth boundaries in the generated depth map. We propose a solution that introduces global contextual semantic priors, generated from pretrained vision trans-former token embeddings. Our approach to distilling se-mantic knowledge from pretrained token embeddings is motivated by their demonstrated effectiveness in related monocular depth estimation tasks. We introduce a Guided Token Attention (GTA) module, which iteratively aligns encoded RGB spatial features with depth encodings, us-ing cross-attention for selectively injecting global seman-tic context extracted from different layers of a pretrained vision transformer. Additionally, we present an archi-tecture called Neural Attention for Implicit Multi-token Alignment (NAIMA) 1, which integrates DINOv2 with GTA blocks for a semantics-aware GDSR. Our proposed architecture, with its ability to distill semantic knowledge, achieves significant improvements over existing methods across multiple scaling factors and datasets.

# 1 Introduction 

Depth maps are crucial for numerous computer vision tasks, including 3D scene reconstruction, object detec-tion, and semantic segmentation [1, 2, 3]. They are widely utilized in real-world applications such as autonomous driving and industrial manufacturing systems [3, 4, 5]. Compared to RGB sensors, depth sensors still lag in terms of resolution, accuracy, and cost-effectiveness. Hence, RGB-guided depth map super-resolution has emerged as a practical solution to enhance depth resolution without the need for expensive hardware. RGB-guided depth super-resolution is motivated by the observation that RGB images and depth maps provide complementary geometric representations of the same scene [3, 6, 7]. Under this assumption, the objective is to integrate low-frequency structural information from the blurred low-resolution depth map with high-frequency features of the corresponding high-resolution RGB im-age to reconstruct a high-resolution depth map. Several studies have highlighted a fundamental challenge inher-ent to this multi-modal setting, where the abundant color and texture information contained in the RGB image is not always aligned with true geometric discontinuities in the depth domain [1, 8, 9, 10]. Hence, in the absence of semantic guidance, a direct naive fusion of the spatial RGB features with LR depth features can lead to struc-tural inconsistencies, blurred object boundaries, or mis-aligned geometric structures. Monocular depth estimation, which predicts relative depth from a single RGB image, faces the same challenge of RGB feature noise, where prior works have addressed this issue using multi-task learning, typically using se-mantic segmentation as an auxiliary task [11, 12, 13]. DepthAnything introduced a unified approach by replac-ing the auxiliary task with pretrained DINO patch to-kens, which provide rich semantic priors absent in stan-dard RGB encoders [14]. Their work demonstrated that these DINO [15] embeddings can perform well for se-mantic tasks, such as semantic segmentation, even with-out fine-tuning, and can be leveraged to distill strong se-mantic knowledge to guide the monocular depth estima-tion. Motivated by the aforementioned limitations and ideas, we introduce our unified architecture NAIMA (Neural At-tention with Implicit Multi-Token Alignment). The core idea is to integrate semantically rich priors into encoded depth features to enable semantics-aware depth super-resolution. To this end, we propose a Guided Token Alignment (GTA) module, which employs cross-attention to query relevant RGB semantic tokens using depth en-codings, infusing additional semantic information into the depth representations, instead of depending solely on spa-tial RGB features. Our contributions are summarized as follows: • We propose a solution to cross-modal misalignment in guided depth super-resolution (GDSR). • We propose a semantics-driven framework that pro-vides explicit semantic guidance through pretrained token representations, drawn from a vision trans-former. • We introduce NAIMA, a semantics-aware GDSR framework that employs a cross-attention-based GTA module to align depth features with pretrained semantic tokens for enhanced reconstruction. Extensive experimental evaluations show that our ap-proach consistently outperforms existing GDSR methods across multiple datasets and scaling factors. We evaluate NAIMA on the Middlebury [16], Lu [17], NYU v2 [18], and RGBDD [19] datasets against 11 benchmark meth-ods, using root mean squared error (RMSE) in cm 2 as the evaluation metric. Obtaining high-quality depth data is costly and often con-strained by hardware limitations [20]. To address this limitation, prior work has largely focused on multi-modal approaches, where a high-resolution RGB image serves as a guiding modality to recover missing high-frequency details. Despite the recent advancements, the develop-ment of new methods in guided depth super-resolution remains comparatively limited when contrasted with the rapid progress observed in RGB image super-resolution. In recent years, various techniques have been pro-posed for both standard and RGB-guided depth super-resolution, introducing end-to-end deep learning architec-tures to reconstruct high-resolution depth maps from low-resolution inputs. Early works such as [21, 22, 23, 24] were among the first to employ deep neural networks for guided depth super-resolution. These methods typi-cally apply convolutional neural networks to fused fea-tures extracted from the low-resolution depth map and the high-resolution RGB guide to reconstruct a single high-resolution depth output. However, a key challenge in RGB-guided depth super-resolution remains the presence of noise in RGB images, such as misleading color and texture patterns that do not necessarily correspond to actual geometric shapes. This, in turn, can cause blurred boundaries and artifacts in the resultant high-resolution depth map. This issue has been widely recognized in prior work, with many novel archi-tectures proposing mechanisms such as attention-based fusion, auxiliary learning, and constrained feature inte-gration to mitigate noise from RGB guidance [9, 1, 5, 8]. These approaches aim to selectively filter irrelevant RGB information and focus on strongly correlated features to improve reconstruction quality. Prior works have used attention for learned feature fusion mechanisms to better model the correlation be-tween RGB and depth modalities [25, 26, 27]. By lever-aging attention, these approaches aim to suppress irrel-evant texture artifacts, selecting only relevant features, thereby reducing blurred boundaries and reconstruction errors caused by noisy RGB guidance. In addition, many studies adopt multi-task learning strategies to further regulate the interaction between RGB and depth features. For example, monocular depth esti-mation (MDE) has been used as an auxiliary task to en-hance shared feature representations, improving overall GDSR performance [8]. Methods such as SGNet and DORNet introduce gradient–frequency estimation and self-supervised degradation modeling, respectively, im-proving robustness under real-world degraded depth con-ditions. Similarly, another such study [28] incorporates depth completion as an auxiliary task, arguing that such supervision helps capture contextual dependencies and better control feature interactions between RGB and depth modalities. The challenge of noisy texture and color informa-tion in RGB has also been widely studied in monocular depth estimation (MDE). Many approaches adopt multi-task learning frameworks, using semantic segmentation as an auxiliary task to mitigate noisy feature correlations and enforce global semantic consistency from RGB im-ages [11, 12, 13]. These methods argue that incorporating high-level semantic supervision helps reduce the impact of misleading texture cues. Recent architectures such as DepthAnything further extend this idea by leveraging pre-trained vision transformer embeddings trained on large-scale RGB datasets as an alternative to explicit auxiliary tasks, demonstrating that pretrained semantic representa-tions can effectively guide depth estimation. However, the integration of such semantic token representations has not yet been thoroughly explored in guided depth super-resolution (GDSR), despite their proven effectiveness in MDE.

# 4 Experiments 

## 4.1 Datasets 

We follow the same training and evaluation protocols as adopted in prior work [22, 32, 1]. Our model is trained on the NYU v2 benchmark using scaling factors of 4x, 8x, and 16x, where low-resolution depth maps are synthet-ically generated through bicubic downsampling, follow-ing the standard train-test split. For cross-dataset valida-tion, the same models trained exclusively on the NYU v2 5dataset are used without any fine-tuning. Evaluation is conducted on RGBD-D, Middlebury, and LU datasets, us-ing root mean squared error (RMSE) in centimeters for measuring performance. 

## 4.2 Implementation 

For NAIMA, the DINOv2 backbone remains frozen to re-tain the pretrained semantic priors. Consistent with prior work, all models are trained on bicubic downsampled in-puts for each scaling factor and optimized for 200 epochs. We use the Adam optimizer with an initial learning rate of 0.0001, and employ a multi-step learning rate decay scheduler. All experiments are conducted with a batch size of 1 on an NVIDIA RTX 4090 GPU. 

## 4.3 Quantitative Results 

Table 1 reports comparative performance across multiple datasets and scaling factors. Our proposed NAIMA archi-tecture consistently outperforms existing techniques, with significant performance gains at larger scales. The largest performance gains are reported for 16x scale, which rep-resents the most challenging setting, as the largely down-sampled depth maps contain minimal structural informa-tion, and the architectures have to rely more on RGB guid-ance. Our proposed NAIMA architecture’s ability to con-trol the RGB noise using cross attention over semantic priors achieves an improvement of 0.20, 0.21, 0.16, and 0.08 cm 2 on the Middlebury, Lu, NYU v2, and RGBDD datasets, respectively. Furthermore, NAIMA achieves an average RMSE reduction of 4.02% and 4.8% compared to the second-best method for scales 8x and 16x, respec-tively. Moreover, in cases where our model ranks second or third, the performance gap is marginal. We additionally evaluate the performance of our pro-posed NAIMA on the relatively newer TOFDSR dataset, which has not been used for comparative analysis in pre-vious works. For a fair and transparent comparison, we selected SGNet and DCTNet, both reporting competitive performance on standard benchmarks and providing pub-licly available pretrained weights. The results of the com-parison are presented in table 2. These results validate our hypothesis that incorporating semantic priors effectively enhances reconstruction qual-ity and reduces errors introduced by noisy RGB guidance. 

## 4.4 Qualitative Results 

Figure 4 presents a qualitative comparison of our pro-posed method with existing benchmark techniques across multiple datasets. It can be observed that NAIMA recov-ered finer structural details while being robust to RGB feature noise. Particularly, it performs better near depth boundaries, producing sharper edges and well-defined corner structures, solidifying the use of semantic priors in mitigating the impact of RGB noise and cross-modal mis-alignment, which often lead to blurred depth boundaries in competing approaches. Figure 6 presents a side-by-side comparison of the RGB encoded features and their refined counterparts after pro-cessing through the GTA module at different levels of the NAIMA architecture. This refinement enables NAIMA to produce sharper depth boundaries while mitigating the adverse effects of RGB noise.

# 6 Conclusion 

We introduce a novel architecture for guided depth super-resolution (GDSR) that incorporates additional seman-tic priors derived from pretrained vision transformer em-beddings. Unlike existing approaches, our framework leverages semantically rich token representations as an explicit modality for guided depth super-resolution, en-abling more faithful utilization of RGB information. Our proposed architecture achieves state-of-the-art per-formance across different scaling factors and datasets. Furthermore, we propose a novel Guided Token Align-ment (GTA) module that selectively transfers relevant se-mantic information across multiple levels of token ab-straction into the depth representation. ## A.2 Semantics Encoder Settings 

As mentioned, we used DINOv2 as the semantic feature extractor for the input RGB image. Specifically, we used the ViT-S/14 variant containing 21 million parameters. The model is pretrained on a diverse corpus consisting of 142 million unlabeled images, capturing robust semantic features that benefit our downstream task of depth super-resolution. The encoder’s weights are kept frozen during training to preserve these generalized, rich semantic rep-resentations learned from the large-scale pretraining.

## A.4 Additional Implementation Choices 

Next, we discuss additional configurations, including patch size selection and training protocols. 

Patch Size: We use a patch size of 420 for 4x scaling factor, and 448 for 8x and 16x scaling factors. These val-ues are selected to ensure compatibility with the DINOv2 semantics encoder, which requires the input image size to be divisible by 14 for its patch tokenization. Additionally, a smaller patch size was chosen for the 4x scaling factor due to GPU memory limitations. 

Training Protocols: We employ a step-based learning rate decay scheduler, using 0.3 as the decay factor, re-ducing the learning rate after every 50 epochs. All RGB images are standardized using ImageNet normalization statistics. For reproducibility, the complete data process-ing pipeline and training code are provided in the code repository 2. Additionally, data preparation code is also available publicly 3.

Inference Time Padding: As aforementioned, our model required input images having a size divisible by 14 for the DINOv2 encoder. During inference, the input images whose spatial dimensions are not divisible by 14 are padded with additional zeros at the bottom right cor-ner before being processed by the model. After the infer-ence, the predicted depth map is cropped back to the origi-nal resolution, prior to being evaluated using RMSE. This step is necessary to satisfy the architectural constraints of DINOv2, although it may introduce a minor bias in the reported RMSE due to padding artifacts, slightly adding to the reported error.

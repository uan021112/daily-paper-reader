## 摘要
自回归 Transformer 的最新进展在生成艺术家级网格方面展现了巨大潜力。然而，现有方法采用的 Token 排序策略通常无法达到专业艺术家的标准：基于坐标的排序会导致序列过长且效率低下，而基于补丁的启发式方法则会破坏高质量建模所必需的连续边流和结构规则性。为了解决这些局限性，我们提出了 Strips as Tokens (SATO)，这是一个采用受三角形条带启发的 Token 排序策略的新颖框架。通过将序列构建为显式编码 UV 边界的面连接链，我们的方法自然地保留了艺术家创作网格所特有的有序边流和语义布局。该方案的一个关键优势在于其统一的表示形式，使得相同的 Token 序列可以被解码为三角形或四边形网格。这种灵活性促进了对两种数据类型的联合训练：大规模三角形数据提供了基础结构先验，而高质量四边形数据则增强了输出的几何规则性。大量实验表明，SATO 在几何质量、结构连贯性和 UV 分割方面始终优于以往的方法。

## Abstract
Recent advancements in autoregressive transformers have demonstrated remarkable potential for generating artist-quality meshes. However, the token ordering strategies employed by existing methods typically fail to meet professional artist standards, where coordinate-based sorting yields inefficiently long sequences, and patch-based heuristics disrupt the continuous edge flow and structural regularity essential for high-quality modeling. To address these limitations, we propose Strips as Tokens (SATO), a novel framework with a token ordering strategy inspired by triangle strips. By constructing the sequence as a connected chain of faces that explicitly encodes UV boundaries, our method naturally preserves the organized edge flow and semantic layout characteristic of artist-created meshes. A key advantage of this formulation is its unified representation, enabling the same token sequence to be decoded into either a triangle or quadrilateral mesh. This flexibility facilitates joint training on both data types: large-scale triangle data provides fundamental structural priors, while high-quality quad data enhances the geometric regularity of the outputs. Extensive experiments demonstrate that SATO consistently outperforms prior methods in terms of geometric quality, structural coherence, and UV segmentation.

#### 1. 核心问题与整体含义（研究动机和背景）

*   **排序策略不佳**：基于坐标排序的序列过长，而基于补丁（Patch）的启发式排序会破坏表面的连续边缘流（Edge Flow）和结构规则性。
*   **缺乏语义布局**：现有方法通常忽略了 UV 分割和展开，导致生成的模型难以直接进入纹理贴图、动画绑定等下游管线。
*   **拓扑单一**：大多数模型仅支持三角形网格，难以生成艺术家更青睐的四边形（Quad）网格。

#### 2. 方法论：核心思想与关键技术
*   **核心思想**：将网格序列化为受“三角形带（Triangle Strips）”启发的连续面链。每个新顶点与前两个顶点结合形成新面，这种“拉链式”增长天然符合艺术家的建模习惯。
*   **关键技术细节**：
    *   **条带标记化（Strip-based Serialization）**：通过贪婪的最低坐标优先策略提取拓扑路径，将复杂的网格图转化为相连的顶点流。
    *   **统一解码协议（Unified Decoding）**：通过调整步长参数 $\delta$（三角为 1，四边形为 2），同一套 Token 序列可以被解码为三角形或四边形网格。

*   **算法流程**：输入点云条件 $\rightarrow$ 点云编码器 $\rightarrow$ Hourglass Transformer 自回归生成 Token $\rightarrow$ 根据步长解码为三角或四边形网格 $\rightarrow$ 应用 UV 分割。

#### 6. 主要结论与发现
*   **条带表示法的优越性**：相比 Patch 排序，条带排序具有更高的压缩率（减少了约 15% 的 Token 长度）且更易于模型学习。

*   **端到端 UV 预测可行**：通过在序列中嵌入边界标记，模型能够生成比传统分割工具更符合几何特征、变形更小的 UV 布局。

#### 7. 优点
*   **工业友好性**：生成的网格具有清晰的边缘流和原生的 UV 分割，极大地降低了下游管线的处理成本。
*   **统一性**：一个模型框架同时解决三角、四边形和 UV 三大任务，具有极高的灵活性。
*   **效率高**：条带化策略减少了序列中的“断点”，提高了 Transformer 处理长序列的效率。

*   **四边形纯度**：生成的四边形网格本质上是“四边形主导”，在极少数拓扑复杂处（如条带长度为奇数）仍可能出现少量三角形。

*   **特定形状偏差**：在处理球体等高度对称且各向同性的形状时，边缘流路由偶尔会出现不够规则的情况。

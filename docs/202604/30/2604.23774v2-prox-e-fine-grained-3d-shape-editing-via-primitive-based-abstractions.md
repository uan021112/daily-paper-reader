## 摘要
基于文本的 2D 图像编辑模型近期已达到令人印象深刻的成熟水平，这激发了大量工作严重依赖这些模型来驱动 3D 编辑。虽然这些以 2D 为中心的 3D 编辑流水线在基于外观的修改方面非常有效，但在细粒度 3D 编辑方面往往表现不佳，因为细粒度编辑需要在严格保留物体整体身份的同时应用局部结构变化。为了解决这一局限性，我们提出了 Prox-E，这是一个无需训练的框架，通过显式的、基于原语的几何抽象来实现细粒度的 3D 控制。我们的框架首先将输入的 3D 形状抽象为一组紧凑的几何原语。然后，预训练的视觉语言模型 (VLM) 编辑该抽象以指定原语级别的更改。这些结构编辑随后被用于引导 3D 生成模型，从而在保留原始形状未更改区域的同时，实现细粒度的局部修改。通过广泛的实验，我们证明了与包括基于 2D 的 3D 编辑器和基于训练的方法在内的各种现有方法相比，我们的方法在平衡身份保留、形状质量和指令保真度方面始终更加有效。

## Abstract
Text-based 2D image editing models have recently reached an impressive level of maturity, motivating a growing body of work that heavily depends on these models to drive 3D edits. While effective for appearance-based modifications, such 2D-centric 3D editing pipelines often struggle with fine-grained 3D editing, where localized structural changes must be applied while strictly preserving an object's overall identity. To address this limitation, we propose Prox-E, a training-free framework that enables fine-grained 3D control through an explicit, primitive-based geometric abstraction. Our framework first abstracts an input 3D shape into a compact set of geometric primitives. A pretrained vision-language model (VLM) then edits this abstraction to specify primitive-level changes. These structural edits are subsequently used to guide a 3D generative model, enabling fine-grained, localized modifications while preserving unchanged regions of the original shape. Through extensive experiments, we demonstrate that our method consistently balances identity preservation, shape quality, and instruction fidelity more effectively than various existing approaches, including 2D-based 3D editors and training-based methods.

#### 1. 核心问题与整体含义（研究动机和背景）
现有的 3D 编辑方法主要依赖 2D 图像编辑模型（如 Diffusion 模型）将编辑效果“提升”到 3D 空间。然而，这种以 2D 为中心的方法在处理**细粒度结构修改**（如“将椅腿加长 1.5 倍”或“旋转车轮 45 度”）时存在显著缺陷：
*   **缺乏度量推理**：像素级 2D 模型难以理解 3D 空间的度量属性和精确几何变换。
*   **身份保持困难**：在进行局部修改时，往往会破坏物体未修改部分的原始特征（身份）。
*   **研究动机**：开发一种能够结合视觉语言模型（VLM）的推理能力与显式几何表示的框架，实现精确、局部且保持身份的 3D 形状编辑。

#### 2. 方法论：核心思想与技术细节
Prox-E 提出了一种**免训练（Training-free）**的框架，其核心流程如下：
*   **几何抽象（Abstraction）**：利用 SuperDec 等工具将输入的 3D 形状分解为一组紧凑的**超二次曲面（Superquadrics）原语**。每个原语由缩放、旋转、平移和形状指数等参数定义。
*   **VLM 代理编辑**：
    *   将原语参数转化为 JSON 格式，并配合多视图渲染图输入给 VLM（如 Gemini 或 Qwen）。
    *   VLM 作为空间推理代理，根据文本指令修改 JSON 中的原语参数（移动、缩放、添加或删除）。
    *   引入**视觉验证循环**：VLM 渲染编辑后的原语图并自我检查，确保几何修改符合指令。
*   **代理引导的去噪（Proxy-Induced Denoising）**：
    *   基于 TRELLIS（3D 扩散模型）的潜空间进行操作。
    *   **潜空间混合策略**：在去噪过程中，通过空间掩码（Mask）融合三种来源的潜变量：
        1.  **原始形状注入**：在未修改区域保持原始几何。
        2.  **变形形状注入**：对修改区域进行仿射变换（Warping），保留表面细节。
        3.  **原语注入**：在新增区域由编辑后的原语引导生成新结构。
*   **外观精细化**：利用 2D 图像编辑器（如 FLUX）修改渲染视图，再通过 TRELLIS 的外观流模型将纹理映射回 3D 结构。

#### 6. 主要结论与发现
*   **性能领先**：Prox-E 在身份保持（LPIPS/DINO-I）和编辑忠实度（VQAScore）上显著优于现有方法。
*   **显式引导的必要性**：实验证明，仅靠 2D 扩散模型无法处理精确的度量修改，而显式的几何原语能为 3D 扩散模型提供必要的空间约束。
*   **VLM 的潜力**：VLM 能够有效地在符号化（JSON）和视觉化（渲染图）表示之间进行推理，完成复杂的结构化编辑任务。

#### 7. 优点
*   **精确控制**：通过原语参数实现了像素级模型无法达到的度量级控制（如精确的缩放和旋转）。
*   **免训练且通用**：不需要针对特定类别训练，可直接应用于各种 3D 资产。
*   **身份保持**：创新的潜空间混合策略有效解决了“编辑即重绘”的身份丢失问题。
*   **可解释性**：原语编辑过程是透明且可人工干预的。

#### 8. 不足与局限
*   **依赖原语分解质量**：如果初始的形状分解（SuperDec）不准确（例如将两个独立部件识别为一个原语），则无法进行精细的局部控制。
*   **VLM 推理限制**：虽然 VLM 进步很快，但在处理极其复杂的空间逻辑或非常细小的部件时仍可能出错。
*   **分辨率限制**：受限于底层生成模型（TRELLIS）的体素分辨率，难以处理超大规模或极高精度的场景。
*   **运行效率**：10 分钟以上的处理时间对于实时交互应用来说仍然较慢。

## 摘要
最近的前馈高斯重建模型采用了像素对齐的方案，将每个 2D 像素映射到一个 3D 高斯，从而使高斯表示与输入图像紧密耦合。在本文中，我们提出了 AnchorSplat，这是一种用于场景级重建的新型前馈 3DGS 框架，它直接在 3D 空间中表示场景。AnchorSplat 引入了一种由 3D 几何先验（如稀疏点云、体素或 RGB-D 点云）引导的锚点对齐高斯表示，从而实现了更具几何感知能力的可渲染 3D 高斯，且该表示独立于图像分辨率和视图数量。这种设计显著减少了所需的高斯数量，在提高计算效率的同时增强了重建保真度。除了锚点对齐设计外，我们还利用高斯精炼器（Gaussian Refiner）仅通过几次前馈过程来调整中间高斯。在 ScanNet++ v2 NVS 基准测试上的实验表明，该方法达到了 SOTA 性能，在保持更高视图一致性的同时，使用了显著更少的高斯原语，优于以往的方法。

## Abstract
Recent feed-forward Gaussian reconstruction models adopt a pixel-aligned formulation that maps each 2D pixel to a 3D Gaussian, entangling Gaussian representations tightly with the input images. In this paper, we propose AnchorSplat, a novel feed-forward 3DGS framework for scene-level reconstruction that represents the scene directly in 3D space. AnchorSplat introduces an anchor-aligned Gaussian representation guided by 3D geometric priors (e.g., sparse point clouds, voxels, or RGB-D point clouds), enabling a more geometry-aware renderable 3D Gaussians that is independent of image resolution and number of views. This design substantially reduces the number of required Gaussians, improving computational efficiency while enhancing reconstruction fidelity. Beyond the anchor-aligned design, we utilize a Gaussian Refiner to adjust the intermediate Gaussiansy via merely a few forward passes. Experiments on the ScanNet++ v2 NVS benchmark demonstrate the SOTA performance, outperforming previous methods with more view-consistent and substantially fewer Gaussian primitives.

### 1. 核心问题与研究动机
*   **核心问题**：现有的前馈（Feed-Forward）3D高斯重建模型大多采用“像素对齐”（Pixel-aligned）方案，即将每个2D像素映射为一个3D高斯。
*   **研究动机**：
    *   **冗余性**：高斯数量随分辨率和视图数线性增长，导致计算开销巨大。
    *   **几何不一致**：像素特征在3D空间交互弱，容易产生浮点伪影（floaters）、表面破碎和几何不连贯。
    *   **视图依赖**：重建质量高度依赖预测深度图的准确性，在遮挡或低纹理区域表现差。
    *   **目标**：提出一种直接在3D空间中表示场景、独立于图像分辨率且几何感知的重建框架。

### 2. 方法论
AnchorSplat 放弃了像素对齐，转而采用**锚点对齐（Anchor-aligned）**的架构，主要包含三个阶段：
*   **锚点预测器（Anchor Predictor）**：
    *   利用预训练的多视图立体（MVS）模型（如 MapAnything）预测深度和相机位姿。
    *   将深度图反投影到3D空间，并使用最远点采样（FPS）进行下采样，生成稀疏且分布均匀的3D锚点（Anchors）。
*   **高斯解码器（Gaussian Decoder）**：
    *   使用轻量级 U-Net 提取多视图图像特征，并将其投影到3D锚点上。
    *   通过 Transformer 建模锚点间的空间交互，由 MLP 预测每个锚点对应的 4 个高斯属性（中心偏移、不透明度、缩放、旋转、球谐系数）。
*   **高斯精炼器（Gaussian Refiner）**：
    *   **即插即用模块**：计算渲染图像与真值的误差，将2D误差反投影回3D空间。
    *   利用 Point Transformer 根据误差特征微调高斯属性，进一步提升细节保真度和视图一致性。
*   **损失函数**：结合了渲染损失（L1, SSIM, LPIPS）、深度损失以及针对高斯不透明度和体积的正则化项。

*   **Benchmark 设置**：分别测试了 32、48、64 个输入视图下的新视角合成（NVS）效果。
*   **对比方法**：

### 6. 主要结论与发现
*   **高效性**：AnchorSplat 使用的高斯数量仅为 AnySplat 的 **1/20** 左右，重建速度提升约一倍。
*   **几何质量**：锚点对齐设计显著减少了浮点伪影和重影，生成的深度图更锐利，几何一致性更强。
*   **鲁棒性**：在极稀疏视图（3-5张）下表现稳定，且在视图增加时，高斯数量保持恒定，不会像像素对齐方法那样导致内存溢出（OOM）。

### 7. 优点（亮点）
*   **解耦设计**：将高斯表示与输入分辨率解耦，解决了前馈模型中常见的冗余问题。
*   **几何先验引导**：通过引入 3D 锚点，使模型在处理弱纹理或复杂遮挡区域时具有更好的空间约束。
*   **精炼模块**：提出的高斯精炼器不依赖重新训练主网络，通过误差反馈有效修复了渲染中的“空洞”和模糊。

### 8. 不足与局限
*   **先验依赖**：高度依赖预训练 MVS 模型提供的几何先验。如果初始深度预测存在严重偏差或缺失，模型难以在完全空白的区域“生长”出正确的高斯。
*   **固定预算限制**：虽然锚点数量可控，但在极度复杂的局部细节处，固定的锚点密度可能导致重建精度遇到瓶颈。
*   **动态场景**：目前仅针对静态场景设计，尚未讨论在动态或时序场景中的应用。

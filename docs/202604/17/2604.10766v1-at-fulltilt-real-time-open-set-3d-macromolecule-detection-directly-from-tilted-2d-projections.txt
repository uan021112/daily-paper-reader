Abstract 

Open-set 3D macromolecule detection in cryogenic electron tomography eliminates the need for target-specific model retraining. However, strict VRAM constraints prohibit processing an entire 3D tomogram, forcing current methods to rely on slow sliding-window inference over extracted subvolumes. To overcome this, we propose FullTilt, an end-to-end framework that redefines 3D detection by operating directly on aligned 2D tilt-series. Because a tilt-series contains significantly fewer images than slices in a reconstructed tomogram, FullTilt eliminates redundant volumetric computation, accelerating inference by orders of magnitude. To process the entire tilt-series simultaneously, we introduce a tilt-series encoder to efficiently fuse cross-view information. We further propose a multiclass visual prompt en-coder for flexible prompting, a tilt-aware query initializer to effectively anchor 3D queries, and an auxiliary geometric primitives module to enhance the model’s understanding of multi-view geometry while improving robustness to adverse imag-ing artifacts. Extensive evaluations on three real-world datasets demonstrate that FullTilt achieves state-of-the-art zero-shot performance while drastically reducing runtime and VRAM requirements, paving the way for rapid, large-scale visual proteomics analysis. All code and data will be publicly available upon publication. 

1 Introduction 

Determining the 3D structures of macromolecules (hereafter referred to as particles ) is critical for understanding biological mechanisms and advancing drug development [ 1 , 2]. Cryogenic electron tomography (cryo-ET) is an imaging technique that enables in situ 3D structural visualization [ 3, 4]. Using this method, cellular samples are rotated and imaged by capturing a series of 2D projections at different angles. The resulting tilt-series is aligned along the y-axis and subsequently reconstructed into a 3D tomogram. Off-the-shelf computational methods are applied to detect target particles [ 5 ], followed by subtomogram averaging and refinement to produce high-resolution structures [4]. While conventional closed-set detection methods [ 6 – 8] require laborious data annotation and re-training for every new target, recent work has explored open-set detection paradigms [ 9– 12 ]. By supplying a visual prompt (e.g., a reference subvolume), these frameworks localize arbitrary targets by comparing prompt embeddings against 3D subvolumes extracted from the tomogram. This enables 

zero-shot detection without target-specific retraining. However, strict VRAM constraints prohibit loading an entire 3D tomogram into memory. As a result, current methods rely on sliding-window inference approaches [ 9, 11 ], resulting in long inference runtimes (Figs. 1 and 2a). Others apply SAM [ 13 ] to 2D slices along each orthogonal axis of the 3D tomogram [ 10 ], but they still exhibit high computational complexity that scales with the tomogram’s volumetric dimensions (Figs. 1 and 2b). Considering that a 3D tomogram is reconstructed from the aligned 2D tilt-series [ 14 , 15 ], the tilt-series intrinsically contains at least as much information as the tomogram. This indicates that tomograms contain redundant information, which wastes computational resources and slows down processing, raising the critical question: Can we conduct open-set 3D macromolecule detection directly on the aligned 2D tilt-series to expedite runtime? 

To answer this question, we propose FullTilt , an open-set 3D macromolecule detection framework that processes the 2D tilt-series in an end-to-end manner (Fig. 2d). By dramatically reducing the input dimensionality, our framework operates at full tilt —running hundreds to thousands of times faster than current state-of-the-art methods (Fig. 1). FullTilt adopts a DETR-like architecture [ 16 ]. We first utilize a 2D backbone to extract initial 2D features [ 17 ]. Our tilt-series encoder ( ETilt ) then merges these feature maps into 3D-aware features with an efficient global row attention, followed by a multiclass visual prompt encoder ( EPrompt ) that extracts prototypes for the user’s requested classes. We then effectively initialize 3D queries with a tilt-aware query initializer ( ITilt ) and leverage a 3D decoder to obtain the 3D coordinates and labels of the targeted particles. To enhance the model’s understanding of multi-view geometry and improve its robustness to adverse imaging artifacts such as occlusion and non-uniform illumination (Fig. 3), we introduce an auxiliary geometric primitives module ( G) that generates geometric data on-the-fly .Although 3D macromolecule detection from 2D tilt-series shares similarities with multi-view 3D ob-ject detection in autonomous driving [ 18 ], directly applying existing frameworks (e.g., DETR3D [ 19 ]or PETR [ 20 ]) yields poor results, rendering such adaptations non-trivial. This is due to the extremely low signal-to-noise ratio (SNR) characteristic of cryo-ET data [ 21 ]. Unlike natural images where objects are clearly visible in a single frame, macromolecules often only become distinguishable when observing consecutive frames across the tilt-series (Fig. 3). For the same reason, naive implemen-tations, such as applying a 2D detector to each tilt image independently and back-projecting the results into 3D space [ 22 ], also fail (Figs. 1 and 2c). FullTilt successfully addresses this challenge by introducing a dedicated tilt-series encoder, enabling the model to holistically contextualize all tilts at once. Evaluated on three public, real-world cryo-ET datasets as well as simulated datasets, FullTilt achieves comparable or superior zero-shot performance against existing open-set baselines, while dramatically reducing runtime and VRAM requirements. 2 Related Work 

3D Macromolecule Detection for Cryo-ET. Deep learning has largely superseded template matching in cryo-ET [ 7]. Methods like crYOLO [ 6 ] and PickYOLO [ 23 ] merge 2D slice detections into 3D coordinates. Zeng’s method [ 22 ] back-projects 2D tilt-series detections into 3D, but suffers from high projection errors and lacks cross-tilt fusion. Volumetric frameworks (e.g., DeepFinder [ 7], DeePiCt [ 24 ], VP-Detector [ 25 ]) utilize 3D CNNs, requiring sliding-window inference and segmen-tation clustering. To reduce annotation costs, weakly supervised learning (WSL) methods (Huang’s method [ 26 ], TomoPicker [ 12 ]) employ positive-unlabeled learning [ 27 ]. Crucially, all these ap-proaches operate in a closed-set paradigm, requiring target-specific model retraining for novel particle classes. 

Open-Set 3D Macromolecule Detection. To bypass retraining, open-set detection paradigms have recently emerged [ 9 , 11 , 10 ]. TomoTwin [ 9 ] compares a target subvolume’s embedding against all 3D tomogram subvolumes. ProPicker [ 11 ] frames this as conditional segmentation to improve speed, while CryoSAM [ 10 ] clusters masks generated by applying 2D SAM [ 13 ] to orthogonal tomogram slices. However, all these methods rely on exhaustive 2D or 3D tomogram patching, which severely bottlenecks inference via repetitive model I/O. Instead, FullTilt directly processes the 2D tilt-series end-to-end, completely bypassing 3D tomogram inference, thus drastically accelerating detection. 3Figure 4: FullTilt architecture. Multi-View 3D Object Detection. End-to-end multi-view 3D object detection is prominent in autonomous driving [ 28 ]. DETR3D [ 19 ] extends DETR [ 16 ] for multi-view data, while PETR [ 20 ]replaces DETR3D’s 3D-to-2D projection with a 3D positional encoder. Subsequent models like PETRv2 [29] incorporate temporal modeling. FullTilt combines DETR3D’s training efficiency (via 3D-to-2D projection) with PETR’s idea of 3D-aware positional encoding. Crucially, to handle high-noise cryo-ET data and enable open-set detection, FullTilt further introduces four novel components: a tilt-series encoder ( ETilt ), a multiclass visual prompt encoder ( EPrompt ), a tilt-aware query initializer (ITilt ), and an auxiliary geometric primitives module ( G).

3.2 Architecture Overview 

FullTilt follows the DETR design principles [ 16 ] and consists of five sequential modules (Fig. 4a). First, a 2D backbone independently extracts multi-level 2D feature maps from each tilt image. A tilt-series encoder ( ETilt ) then collectively processes these features, merging cross-tilt information into 3D-aware features. Subsequently, a multiclass visual prompt encoder ( EPrompt ) extracts prototype tokens for the requested classes based on the 2D prompts. Using these prototypes and the 3D-aware features, a tilt-aware query initializer ( ITilt ) establishes the 3D queries. Finally, a 3D decoder 

refines these queries to estimate the targeted 3D coordinates, diameters, and labels. 43.3 Architecture Details 2D Backbone. 3.4 Training Objective 

Following standard DETR architectures [ 32 , 33 ], we compute an L1 loss for bounding box regression and a focal loss for classification, applied after bipartite matching between model outputs and ground truth. To expedite convergence, we follow previous work [ 33 ] by incorporating auxiliary losses at each decoder layer and employing a contrastive denoising strategy [34]. 

3.5 Auxiliary Geometric Primitives ( G)

Learning multi-view geometry directly from 2D tilt-series is exceptionally challenging, especially given adverse imaging artifacts (Fig. 3). As physics-based cryo-ET simulations are computationally prohibitive and scale poorly, we introduce an auxiliary geometric primitives module ( G). This module generates synthetic 2D projections (e.g., circles of varying sizes) alongside their exact 3D ground truth coordinates on-the-fly , eliminating I/O storage bottlenecks. G simulates diverse object distributions (e.g., dense clusters in the central z-plane) and mimics adverse imaging artifacts such as physical occlusion and non-uniform illumination. During training, we alternate between this dynamically generated data and the simulated cryo-ET data at every epoch. 

3.6 Inference and Interaction 

FullTilt’s flexible design supports multiple interactive workflows. In a standard setup, a user selects a few target particles (e.g., 1–4 per class) from a reconstructed 3D tomogram to serve as prompts. For high-throughput analysis, users can extract class prototypes from one tomogram and reuse them across an entire dataset, enabling rapid zero-shot inference without manual re-prompting. In both cases, FullTilt automatically projects the 3D coordinates onto the 2D tilt images and seamlessly handles varying prompt quantities across views. In addition, users can successfully conduct full tilt-series inference by directly providing 2D prompts solely on the 0◦ tilt image.

Evaluation Dataset. We evaluate FullTilt’s zero-shot performance on three public cryo-ET datasets: CZII’s CryoET Object Identification Challenge (CZII) [ 36 ], EMPIAR-10304 [ 37 ], and EMPIAR-10499 [ 38 ]. We additionally evaluate on a simulated dataset containing 10 unseen proteins alongside a training subset. 

Metrics. We report mean Average Precision (mAP) at distance thresholds of 0.5 and 1.0 times the target radius (mAP@0.5r, mAP@1.0r) [ 26 ]. As some baselines lack confidence scores, we supplement this with an F1 score (1.0 radius threshold). In all cases, inference runtime (T.) and peak VRAM (M.) are measured on a single NVIDIA RTX 6000 Ada GPU and 48-core CPU.

4.3 Intra-Instance Detection 

For intra-instance detection (where prompts and targets reside in the same instance), we evaluate FullTilt against zero-shot open-set methods (TomoTwin [ 9], CryoSAM [ 10 ], ProPicker [ 11 ]). To benchmark against 2D back-projection (Zeng’s method [ 22 ]) and general multi-view frameworks (DETR3D [ 19 ], PETR [ 20 ]), we integrated our prompt module ( EPrompt ) into their architectures. Fully supervised methods are omitted as these datasets lack separate, fully annotated training sets. We evaluate 1, 2, and 4 prompts over 10 independent trials to account for variance (Tables 1–3). Expectedly, providing more prompts enhances performance. FullTilt achieves comparable or superior real-world performance across all metrics, reducing inference to under one second with the lowest VRAM requirements. Notably, FullTilt’s margin over TomoTwin grows on challenging native cellular (EMPIAR-10499) versus purified (EMPIAR-10304) ribosomes. Among the baselines, TomoTwin outperforms ProPicker but incurs significantly longer runtimes, whereas CryoSAM struggles because its natural-image pre-training translates poorly to cryo-ET [ 11 ]. Despite being trained on identical data, general multi-view models (DETR3D, PETR) fail to effectively fuse tilt-series information. PETR’s particularly poor performance indicates that attention across all tilts in the decoder cannot capture critical features. 

4.4 Cross-Instance Detection 

For cross-instance detection, a prompt from one instance is used for dataset-wide inference. We compare FullTilt against the top open-set baselines (TomoTwin [ 9], ProPicker [ 11 ]) and WSL methods (Huang’s method [ 26 ], TomoPicker [ 12 ]). We randomly select one instance per dataset to extract embeddings or train WSL models using 1, 2, or 4 prompts. Experiments span 10 trials (limited to 3 for WSL models due to extensive training times). As shown in Table 4, FullTilt outperforms baselines across all real-world datasets, except for a marginally lower mAP@0.5r than TomoTwin on EMPIAR-10304. As expected, WSL methods fail to generalize effectively with so few prompts, though TomoPicker outpaces Huang’s method.

Component Ablation. We ablate our primary contributions: the tilt-series encoder ( ETilt ), tilt-aware query initializer ( ITilt ), and auxiliary geometric primitives module ( G) (Table 7). Without these modules, FullTilt fails. Among single-module configurations, only ITilt yields viable detections. Integrating ETilt subsequently drives a dramatic performance surge, underscoring the absolute necessity of cross-tilt fusion for cryo-ET data. Finally, adding G further boosts performance by enhancing the model’s understanding of multi-view geometry and improving its robustness to adverse imaging artifacts. 

5 Conclusion 

In this work, we introduced FullTilt, an end-to-end framework that redefines open-set 3D macro-molecule detection by operating directly on aligned 2D cryo-ET tilt-series. By bypassing computation-ally expensive 3D sliding-window inference, FullTilt eliminates redundant volumetric computation, reducing runtime by orders of magnitude while strictly limiting VRAM usage. Extensive evaluations demonstrate that FullTilt achieves state-of-the-art zero-shot detection performance on real-world datasets. We anticipate that demonstrating the efficiency and effectiveness of this direct 2D-to-3D detection paradigm will facilitate rapid, large-scale object detection in structural biology and beyond. 

Limitations and Future Work. FullTilt currently struggles with particularly small macromolecules (a known limitation of DETR-like architectures [ 16 ]) and suboptimally localizes highly non-globular particles due to its simplified single-diameter formulation. Future work will address these constraints and integrate our macromolecule detection framework into downstream subtomogram averaging [ 4]. 9References

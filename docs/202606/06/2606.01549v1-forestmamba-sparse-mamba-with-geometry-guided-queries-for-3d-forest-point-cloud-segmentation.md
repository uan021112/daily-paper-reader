## 摘要
基于人工智能的地面和无人机LiDAR点云的语义与实例分割正逐渐成为一种变革性方法，用于将森林的复杂三维结构转化为可用于森林监测和生物多样性评估的实用信息。然而，森林LiDAR场景仍然极具挑战性，因为它们数据量大、采样密度不规则、冠层结构重叠且复杂，以及地理变异性。现有的基于稀疏卷积或Transformer的方法取得了有希望的结果，但存在两个关键限制：注意力的二次复杂度难以扩展到大型森林场景；通用的上下文建模未能利用森林结构先验，限制了复杂区域中的树木分离。为了解决这些挑战，我们提出了ForestMamba，一种结构感知方法，它将森林特定的先验融入特征编码、查询生成和查询精化中，同时用线性时间状态空间建模替代二次注意力。首先，我们引入了一个具有垂直优先平板序列化的稀疏编码器，它将稀疏体素组织成垂直连贯的序列，以实现高效的长程上下文建模。其次，我们提出了一种基于即时多尺度冠层高度模型（CHM）的几何引导查询初始化策略，其中冠层最大值提供生态上有意义的查询种子，并辅以最远点采样（FPS）以覆盖下层树木。第三，我们设计了一个基于Mamba的查询解码器，它结合了局部kNN体素聚合和空间双路径Mamba，以线性计算复杂度进行查询精化。在七个森林区域的广泛实验表明，ForestMamba在两项分割任务中始终优于现有基线，同时相比基于Transformer的方法实现了3倍更快的推理速度和2.3倍的GPU内存降低。

## Abstract
AI-based semantic and instance segmentation of terrestrial and drone LiDAR point clouds is emerging as a transformative approach for converting the complex 3D structure of forests into actionable information for forest monitoring and biodiversity assessment. However, forest LiDAR scenes remain highly challenging due to their large data volumes, irregular sampling density, overlapping and complex canopy structure, and geographic variability. Existing methods based on sparse convolutions or Transformers achieve promising results, but suffer from two key limitations: Quadratic complexity of attention scales poorly to large forest scenes, and Generic context modeling does not exploit forest structural priors, limiting tree separation in complex regions. To address these challenges, we propose ForestMamba, a structure-aware method that incorporates forest-specific priors into feature encoding, query generation, and query refinement, while replacing quadratic attention with linear-time state-space modeling. First, we introduce a sparse encoder with vertical-priority slab serialization that organizes sparse voxels into vertically coherent sequences for efficient long-range context modeling. Second, we propose a geometry-guided query initialization strategy based on an on-the-fly multi-scale Canopy Height Model (CHM), where canopy maxima provide ecologically meaningful query seeds, supplemented by Farthest Point Sampling (FPS) to cover understory trees. Third, we design a Mamba-based query decoder that combines local kNN voxel aggregation with a spatial dual-path Mamba for query refinement with linear computational complexity. Extensive experiments across seven forest regions demonstrate that ForestMamba consistently outperforms existing baselines in both segmentation tasks, while achieving 3 times faster inference and 2.3 times lower GPU memory than Transformer-based methods.

## 2. 方法论：核心思想、关键技术细节

ForestMamba包含三个主要组件：

### 2.1 结构感知稀疏编码器
- **动机**：树木具有从地面到冠层的强垂直组织，通用空间序列化（如空间填充曲线）未编码此生态先验。
- **实现**：
  - 采用U-Net风格的稀疏卷积骨干（5级下采样+上采样），每级包含稀疏卷积残差块和Mamba块。
  - **垂直优先平板序列化**：将每个占用体素分配到垂直平板（slab），平板索引s_n = floor(c_z_n / τ)，其中τ为平板厚度（5体素单元）。按(s_n, c_y_n, c_x_n)字典序排序，得到垂直连贯的序列。处理后通过逆置换恢复体素顺序。
  - Mamba块对序列化特征进行线性时间的长程依赖建模（层归一化+残差连接）。

### 2.2 几何引导查询初始化
- **动机**：通用查询初始化（如FPS）忽视森林几何结构，产生冗余或遗漏。利用冠层高度模型（CHM）提供生态上有意义的种子。
- **步骤**：
  1. **辅助分支**：从编码器特征通过两个MLP头分别预测树/背景二分类logits和实例判别嵌入（受判别损失约束）。
  2. **多尺度CHM峰值检测**：
     - 对每个分辨率r（0.3m和0.7m），构建2D CHM，每个网格存储树体素的最大高度。
     - 使用异速生长抑制窗口：半径w_r(h)=ceil(α h^β / r)，其中α=0.25, β=0.5（从训练集经验调优）。
     - 对每个检测到的局部峰值，记录二维坐标和CHM高度，多尺度候选通过贪心空间非极大抑制去重，按CHM高度排序保留最多K个。
  3. **圆柱特征汇聚**：对每个峰值，在半径r_p=1.5m的圆柱内汇聚附近树体素的编码器特征，作为查询特征。
  4. **FPS补充**：若CHM峰值数<K，则对树体素的判别嵌入进行FPS，选取剩余查询。最终查询集Q为CHM查询与FPS查询的并集，共K=300个。

### 2.3 基于Mamba的查询解码器
- 给定编码器体素特征F和初始查询集Q，通过6层解码器迭代精化。
- **每层结构**：
  1. **局部特征聚合**：每个查询关联一个3D锚点位置μ_k（初始为峰值或FPS体素中心，后续更新为预测掩码质心）。使用kNN（κ个近邻）在3D空间中选取邻近体素，通过缩放点积注意力计算权重，结合元素级门控聚合局部证据，并残差更新。
  2. **空间双路径Mamba**：将更新后的查询按其锚点位置进行两种互补的平板遍历（自底向上和自顶向下），分别通过共享Mamba处理，然后取平均融合，再经前馈网络输出。
  3. **掩码预测**：每层输出对象性分数和实例掩码logits（查询嵌入与体素特征的归一化点积）。
- **复杂度**：所有操作线性于体素数或查询数，避免二次注意力。

### 2.4 训练目标
- 多任务损失：λ_ins L_ins + λ_sem L_sem + λ_bin L_bin + λ_dis L_dis
- L_sem：语义分割交叉熵损失；L_bin：树/非树分类损失；L_dis：判别嵌入损失（类内紧凑、类间分离）。
- L_ins：通过一对多匹配（one-to-many matching）对每层解码器输出计算对象性损失、BCE掩码损失和Dice损失，逐层平均。
- 优化器：AdamW（lr=1e-4, weight decay 0.05），多项式学习率衰减（power 0.9），训练3000 epochs，梯度裁剪10。

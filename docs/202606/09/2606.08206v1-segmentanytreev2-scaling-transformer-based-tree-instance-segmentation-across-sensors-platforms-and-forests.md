## 摘要
我们提出了SegmentAnyTreeV2，一个传感器和平台无关的框架，用于森林点云的语义和实例分割。该模型结合了基于序列化的Point Transformer v3骨干网络、轻量级语义头以及一个专注于树木的交叉注意力掩码解码器。语义预测将实例解码限制为树木类体素，而实例感知的查询初始化、一对多种子监督和非对称掩码评分改善了在密集和结构复杂的林分中的分离效果。我们进一步引入了FOR-instance v3，一个扩展的基准数据集，包含来自不同生物群落、森林结构和LiDAR平台的427个场景和26,496棵标注树木。在FOR-instanceV2测试集上，SegmentAnyTreeV2达到了90.5%的精确率、80.2%的召回率、85.0%的F1分数、90.7%的覆盖率和87.6%的语义mIoU，在实例检测和掩码完整性方面均优于以前基于学习的方法。在独立场地上的零样本评估进一步证明了强大的跨域泛化能力。

## Abstract
We present SegmentAnyTreeV2, a sensor- and platform-agnostic framework for semantic and instance segmentation of forest point clouds. The model combines a serialization-based Point Transformer v3 backbone with a lightweight semantic head and a tree-focused cross-attention mask decoder. Semantic predictions restrict instance decoding to tree-class voxels, while instance-aware query initialization, one-to-many seed supervision, and asymmetric mask scoring improve separation in dense and structurally complex stands. We further introduce FOR-instance v3, an expanded benchmark comprising 427 scenes and 26,496 annotated trees across diverse biomes, forest structures, and LiDAR platforms. On the FOR-instanceV2 test split, SegmentAnyTreeV2 achieves 90.5% precision, 80.2% recall, 85.0% F1, 90.7% coverage, and 87.6% semantic mIoU, outperforming previous learning-based methods in both instance detection and mask completeness. Zero-shot evaluation on independent sites further demonstrates strong cross-domain generalization.

## 2. 方法论：核心思想与关键技术细节

- **整体架构**：柱状裁剪 → 体素化 → PTv3 编码器-解码器骨干 → 轻量级语义头 + 实例感知采样 (ISA) → 交叉注意力实例解码器 → 输出二进制掩码和置信度。  
  - 核心设计原则：将语义和实例分割解耦为不对称路径。语义由骨干特征上的 MLP 直接预测，实例解码器只处理树木类体素。

- **关键技术细节**：
  - **Point Transformer v3 (PTv3) 骨干**：采用序列化（Z-order、Hilbert 等）实现线性复杂度，U-Net 结构，深度在第四阶段最重（12层），防止过早压缩。  
  - **语义头**：两层 MLP，预测 ground/wood/leaf 三类。训练时使用加权交叉熵 + Lovász-Softmax。  
  - **实例感知采样 (ISA)**：  
    - 辅助嵌入头将骨干特征投影到 5D 空间，用判别性损失（方差拉近、距离推远）使同类体素聚集。  
    - 在树木体素候选池上对 5D 嵌入执行最远点采样，选出 Q=400 个种子点，每个种子点的骨干特征作为初始查询。  
    - 课程学习：前 30 个 epoch 禁用 ISA，使用随机初始化查询，让语义头和嵌入头先成熟。  
  - **实例解码器**：  
    - **树点过滤**：仅保留语义预测为木材或树叶的体素，降低注意力计算量并聚焦前景。  
    - **双重特征投影**：分别用于注意力和掩码评分，掩码评分分支更深（含层归一化）。  
    - **迭代注意力掩码**：借鉴 Mask2Former，每层基于上一层的掩码 logits 生成二值掩码，限制每个查询的注意力范围。  
    - **输出头**：掩码头（点积 + sigmoid）、评分头（两层 MLP 预测 mask-IoU 回归）。  
  - **训练损失**：  
    - 语义损失：0.5 CE + 0.5 Lovász。  
    - 实例损失：Dice + Focal + Score MSE（+ 分类损失禁用）。  
    - 使用一对多匹配：来自同一 GT 树木的多个查询都监督该 GT 掩码，提高大树召回。  
    - 深度监督：中间 5 层也计算实例损失，权重 0.5。  
  - **推理管线**：滑动窗口（半径 16m，步长 4m）→ 流式推理 → 局部后处理（边界/置信度/地面高度过滤）→ 全局贪心合并（按置信度排序，重叠阈值 0.1473）→ 最终后处理（大小过滤、地面覆盖重写）。

## 6. 主要结论与发现

- **SOTA 性能**：在 FOR-instance v2 测试集上，SaTv2 的 F1 达到 85.0%（比 ForestFormer3D 高 2.2 pp），覆盖率 90.7%（高 9.5 pp），语义 mIoU 87.6%。

- **覆盖率的巨大提升**（+9.5 pp）主要来自架构改进而非数据量增加（v2→v3 数据仅贡献 11% 的覆盖率增益），说明 PTv3 骨干和树聚焦解码器能产生更完整的掩码。

- **实用性**：在管理良好的针叶林中已接近运营部署（F1 > 96%）。

## 7. 优点

- **传感器与平台无关**：单模型可处理 ULS、TLS、MLS 数据，零样本可跨生物群落迁移。
- **架构创新**：序列化 Transformer 骨干避免固定邻域搜索，自然捕获长距离上下文；解耦语义与实例，减少解码器负担；树点过滤降低注意力成本并聚焦前景。
- **一对多匹配 + ISA**：提高大树的召回率，同时保持小树精确性；课程学习稳定训练。

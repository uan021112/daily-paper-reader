## 摘要
我们提出TriFlow，一种新的生成方法，能够直接从输入几何条件（如符号距离场）生成具有艺术家风格三角形拓扑的紧凑3D网格。我们的关键见解是将网格拓扑表示为定义在表面上的最近顶点向量场（NVF），其中每个点编码其在局部重心坐标系中与最近三角形顶点的关联。我们训练一个潜在流匹配模型来合成该场，从而实现基于输入几何条件的拓扑生成。为了提取连贯的网格，我们使用生成的NVF对表面区域进行聚类，并通过拓扑感知优化引导约束二次误差度量（QEM）网格简化。这使得输出网格在几何上与输入高度匹配，同时表现出结构化的、类似艺术家的连接性。实验表明，与最先进的基于学习方法相比，TriFlow实现了更强的泛化能力和显著改善的拓扑质量，同时Chamfer距离降低了90%，速度提升了8倍。

## Abstract
We present TriFlow, a new generative approach for producing compact 3D meshes with artist-like triangle topology directly from input geometry conditions such as signed distance fields. Our key insight is to represent mesh topology as a nearest-vertex vector field (NVF) defined over the surface, where each point encodes its association to the nearest triangle vertex in the local barycentric frame. We train a latent flow-matching model to synthesize this field, enabling topology generation conditioned on the input geometry. To extract a coherent mesh, we cluster surface regions using the generated NVF and guide a constrained quadric error metric (QEM) mesh simplification with topology-aware optimization. This yields output meshes that closely match the input geometry while exhibiting structured, artist-like connectivity. Experiments demonstrate that TriFlow achieves stronger generalization and significantly improved topology quality compared to state-of-the-art learning-based approaches, alongside 90% lower Chamfer Distance and an 8x speedup.

## 1. 论文的核心问题与整体含义

**研究动机与背景**：  
三角形网格是计算机图形学与视觉中的基础表示，但现有方法（如Marching Cubes）提取的网格过度细分且拓扑杂乱，不适合下游处理（例如实时渲染、编辑、物理模拟）。艺术家手工创建的网格具有紧凑面数、均匀顶点分布、边缘对齐几何特征等“艺术家风格”拓扑。目前基于自回归序列建模的方法（如MeshGPT、TreeMeshGPT）虽然能生成一定质量的网格，但存在推理速度慢、误差累积、泛化性差等问题。因此，本文旨在提出一种能直接从输入几何（如SDF）生成紧凑且具有艺术家风格拓扑的高效方法。

## 2. 论文提出的方法论

**核心思想**：将网格拓扑表示为定义在表面上的**最近顶点向量场（NVF）**，从而将离散的拓扑预测转化为连续场建模，避免自回归序列的缺陷。

**关键技术细节**：
1. **最近顶点向量场（NVF）定义**：对于表面上任意点 \( p \)，其NVF值 \( \mathbf{t}(p) = \mathbf{v}_n - p \)，其中 \( \mathbf{v}_n \) 是包含 \( p \) 的三角形中重心坐标权重最大的顶点。该场将每个三角形划分为三个区域，隐式编码了顶点连接关系。
2. **稀疏体素化**：将NVF离散化到512³稀疏体素网格上，每个体素中心计算对应的NVF向量。
3. **潜在流匹配模型**：训练一个SDF自编码器编码输入几何为潜在 \( \mathbf{z}_{\text{SDF}} \)，一个NVF自编码器编码NVF为潜在 \( \mathbf{z}_{\text{NVF}} \)。使用条件流匹配（flow matching）网络，以SDF潜在和拓扑控制参数（目标面数、四边形比例）为条件，生成NVF潜在。流匹配采用线性插值路径，损失函数为 \( \mathcal{L}_{\text{flow}} = \mathbb{E}[\| \mathbf{u}_\theta(\mathbf{z}(i), i, \mathbf{z}_{\text{SDF}}, \mathbf{c}) - (\epsilon - \mathbf{z}_0) \|_1] \)。
4. **网格提取**：
   - **双边滤波**平滑预测的NVF。
   - **NVF转移**：将体素化的NVF映射到从SDF用Marching Cubes提取的过度细分网格 \( \mathcal{M}_d \) 的顶点上。
   - **分水岭算法**：以位移幅度小的顶点为根，按预测目标位置欧氏距离进行区域增长，得到与NVF对应的连通簇。
   - **约束QEM简化**：在QEM简化中，禁止跨不同簇的边折叠，并对涉及根顶点的折叠添加位置约束二次型 \( \mathbf{Q}_t \)，使收缩点靠近预测目标位置。
5. **几何数据增强**：训练时对几何施加随机3D变形场，使模型对扫描或生成中的噪声更鲁棒。

**流程概要**：输入SDF → 编码为潜在 → 流匹配生成NVF潜在 → 解码为体素NVF → 双边滤波 → 转移到稠密网格 → 分水岭聚类 → 约束QEM简化 → 输出艺术家风格网格。

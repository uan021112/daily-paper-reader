## 摘要
单目3D形状恢复是几何理解的基础，但在任意视角和未见物体类别下实现鲁棒的泛化仍然是一个重大挑战。本文提出了一种可泛化的形变学习框架，通过显式地将类别级形状模板形变以匹配目标观测来重建3D物体。为了解决模板与目标之间的复杂形状变化，我们引入了一种几何引导的特征建模机制。该过程首先利用模板拓扑丰富基础特征，得到几何感知表示，然后将其与目标观测显式关联以指导精确形变。此外，为了弥合固定模板与任意目标视角之间的差异，我们提出了一种视图自适应特征聚合模块。该模块利用多视角模板特征及其对应的相机位姿来丰富规范模板表示，确保无论目标视角如何都能实现鲁棒的特征对齐。大量实验表明，我们的方法在处理大幅形状变化和多样化视角方面显著优于现有最先进方法，展现出对新颖类别的强泛化能力，并有效支持下游真实世界的灵巧机器人操作任务。项目主页：https://GODeform.github.io/

## Abstract
Monocular 3D shape recovery is fundamental to geometric understanding, yet achieving robust generalization across arbitrary viewpoints and unseen object categories remains a significant challenge. In this paper, we present a generalizable deformation learning framework that reconstructs 3D objects by explicitly deforming a category-level shape template to match the target observation. To address complex shape variations between the template and the target, we introduce a geometry-guided feature modeling mechanism. This process first enriches foundation features with template topology to yield a geometry-aware representation, which is then explicitly correlated with the target observation to guide precise deformation. Furthermore, to bridge the disparity between the fixed template and arbitrary target views, we propose a view-adaptive feature aggregation module. This module leverages multi-view template features and their corresponding camera poses to enrich the canonical template representation, ensuring robust feature alignment regardless of the target's perspective. Extensive experiments demonstrate that our approach significantly outperforms state-of-the-art methods in handling large shape variations and diverse viewpoints, exhibiting strong generalization to novel categories and effectively supporting downstream real-world dexterous robotic manipulation tasks. ### 2. 论文提出的方法论：核心思想、关键技术细节

-   **核心思想**：将单视图3D重建问题建模为**条件流匹配问题**。给定一个目标图像和一个类别级3D几何模板，学习一个连续的形变场，将模板的顶点流形地“推”向目标的3D形状。其关键在于使用**几何引导的基础特征建模**和**视图自适应特征聚合**来生成稳定且信息丰富的条件信号，指导形变网络。
-   **方法流程**：
    1.  **任务公式化**：输入目标图像 `I_T` 和3D模板 `S` (N个点)，目标是预测每个点的形变场 `D` (Nx3) 使得 `T_hat = S + D`。采用流匹配（Flow Matching）框架，直接预测形变场，实现单步推理。
    2.  **几何引导的视觉基础特征建模 (Geometry-Guided Feature Modeling)**:
        -   **特征传播 (Geometry-Guided Feature Propagation)**：解决2D特征仅覆盖模板可见表面，而形变需要全形状引导的问题。利用3D编码器获取模板所有点的几何嵌入 `G`，计算所有点与可见点之间的几何相似性，通过加权聚合将可见点的2D语义特征传播到整个模板表面，得到完备特征 `F_complete`。
        -   **语义对齐 (Semantic-Aware Feature Alignment)**：桥接模板3D点特征 `F_complete` (作为Query) 与目标图像2D特征 `F_T` (作为Key/Value) 之间的模态和空间差异。通过交叉注意力机制，让每个模板点动态地“关注”其在目标图像中的语义对应区域，生成对齐后的特征 `F_aligned`，形成最终的调节信号`c`。
    3.  **视图自适应的特征聚合 (View-Adaptive Feature Aggregation)**:
        -   **多视图相机位姿编码 (Multi-View Camera Pose Encoding)**：解决固定模板视图与任意目标视图视角差异导致特征漂移的问题。从K个预渲染的模板多视图中，选择与目标图像DINOv3特征最相似的一个作为主视图，其余为辅助视图。计算辅助视图相对于主视图的相对相机位姿 `P_rel`，并编码为特征。
        -   **位姿感知的跨视图特征聚合 (Pose-Aware Cross-View Feature Aggregation)**：将主视图和辅助视图的可见点特征分别与其对应的位姿编码相加，实现几何对齐。然后，以主视图调制后的特征作为Query，以所有视图特征作为Key/Value进行交叉注意力，选择性地融合来自不同视角的互补信息。最终通过残差连接得到视角不变的特征 `F_hat`。
    4.  **训练目标 (Training Objectives)**：总损失函数 `L` 结合了流匹配损失 `L_FM` 用于监督形变场，以及多个几何正则化项：Chamfer距离损失 `L_CD`（全局对齐）、Laplacian平滑损失 `L_Lap`（局部平滑）、ARAP损失 `L_ARAP`（局部刚性）、正则化项 `L_reg`（限制形变幅度）和轮廓损失 `L_sil`（多视角几何监督）。

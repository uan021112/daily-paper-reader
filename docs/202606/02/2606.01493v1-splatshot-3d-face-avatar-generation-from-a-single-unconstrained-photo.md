## 摘要
从单张无约束照片重建逼真的3D人脸头像具有挑战性：前馈式3D高斯泼溅（3DGS）模型在分布外输入上会退化，而预训练的扩散模型生成高保真图像但缺乏多视图一致性。我们观察到这些范式本质上是互补的：显式3D表示保证几何一致性，而2D扩散先验确保照片真实感。基于此，我们提出SplatShot，一种无训练框架，直接在去噪过程中耦合这些表示。给定一个基础3DGS人脸模型和一张参考图像，我们使用每步3D反馈回路联合去噪所有目标视图。在每个时间步，我们从噪声潜变量预测干净图像，将3DGS重新拟合到这些多视图预测，并将3DGS重新渲染与2D预测之间的光度差异反向传播到噪声估计中。这将采样轨迹引导向严格3D一致、身份忠实的结果。在多样化的野外图像上的实验表明，SplatShot生成具有优越身份保持、照片真实感和多视图一致性的3D头像。

## Abstract
Reconstructing a photorealistic 3D face avatar from a single unconstrained photograph is challenging: feed-forward 3D Gaussian Splatting (3DGS) models degrade on out-of-distribution inputs, while pretrained diffusion models produce high-fidelity images but lack multi-view consistency. We observe that these paradigms are fundamentally complementary: explicit 3D representations guarantee geometric consistency, whereas 2D diffusion priors ensure photorealism. Building on this, we propose SplatShot, a training-free framework that couples these representations directly within the denoising process. Given a base 3DGS face model and a single reference image, we jointly denoise all target views using a per-step 3D feedback loop. At each timestep, we predict clean images from the noisy latents, refit the 3DGS to these multi-view predictions, and back-propagate the photometric discrepancy between the 3DGS re-renderings and 2D predictions into the noise estimate. This steers the sampling trajectory toward strictly 3D-coherent, identity-faithful outputs. Experiments on diverse in-the-wild images demonstrate that SplatShot produces 3D avatars with superior identity preservation, photorealism, and multi-view consistency.

## 2. 方法论：核心思想、关键技术细节
- **核心思想**：在扩散模型降噪的每一步，用当前预测的多视角图像更新3DGS模型，再将3DGS重渲染与预测图之间的光度差异反向传播回噪声估计，从而将采样轨迹引导向严格3D一致且身份忠实的结果。
- **关键技术细节**：
  1. **初始化**：从一组预训练“基础”3DGS模型（来自NeRSemble）中自动选择与输入照片最匹配的一个，渲染V=16个视角的图像，加噪至起始时间步`ts = 0.6*T`（保留部分结构）。
  2. **每步迭代（三个子步骤）**：
     - **Step 1: 扩散生成**：用IP-Adapter（身份条件）和ControlNet（姿态条件）对各视角独立预测噪声，解码得到预测干净图像`^I_v`。
     - **Step 2: 3DGS更新**：用所有视角的`^I_v`对当前3DGS模型进行多步优化（首步1000次，后续200次），通过光度损失（L1 + SSIM）拟合。
     - **Step 3: 噪声调整**：计算3DGS重渲染与扩散预测之间的光度损失`g`，通过链式法则计算梯度`∂g/∂ε^v_t`，更新噪声估计`ε^v_t ← ε^v_t - λ·∂g/∂ε^v_t`。
  3. **噪声混合机制**：早期的预测干净图像严重损坏，无法有效监督3DGS。因此将预测噪声与初始加入的真实噪声按权重`w=0.4`混合（`ε_mix = w·ε_pred + (1-w)·ε_gt`），在保持可微性的同时提供稳定的几何锚点，防止3DGS崩溃。
- **关键公式**：
  - 预测干净图像：`^x_0 = (x_t - σ_t ε_mix)/α_t`
  - 噪声梯度：`∂g/∂ε^v_t = (∂^x_0/∂ε)·(∂g/∂^x_0)`，其中`∂^x_0/∂ε = -σ_t/α_t`

## 6. 主要结论与发现
- SplatShot在**身份保真、照片真实感、多视图一致性**上均显著优于所有现有方法，尤其是显式3D一致性指标（CV-CSIM +0.03~0.25）和感知质量（CLIP-IQA最高）。
- 消融证实**3D反馈回路**（λ>0）和**噪声混合**（w<1）是必需的，两者缺一都会导致CV-CSIM锐降或渲染质量崩溃。
- 基础模型主要影响**发型**和**头形**，但身份特征（面部结构、肤色）可通过反馈回路有效迁移。
- 方法在**极端姿态、遮挡、不同年龄、种族**上均表现稳定，具备较强的野外泛化能力。

## 7. 优点
- **无需训练**：完全基于现成预训练模型，无任何微调或额外数据需求，即插即用。
- **天然3D一致**：通过显式3DGS作为输出，直接保证多视角几何一致性，避免了扩散输出各自为政的问题。
- **强身份保持**：巧妙利用图像适配器（IP-Adapter）与3D反馈，实现身份忠实迁移。
- **噪声混合设计**：以极简方式解决了早期扩散预测不可靠导致的3DGS崩溃问题，同时保持梯度可微。
- **公平性验证**：主动报告了种族和年龄组指标，表明方法不存在显著偏见。

## 8. 不足与局限
- **速度较慢**：~3分钟/身份，远慢于前馈方法（秒级），虽快于某些优化方法，但对实时/大规模应用仍有瓶颈。
- **几何锚定受限**：基础模型仅覆盖前180°（NeRSemble数据），无法生成后视图；也无法处理帽子等罕见物品（训练集中缺失）。
- **发型继承问题**：身份编码器通常裁剪掉头发，导致发型主要由基础模型决定，若匹配不当会产生差异。
- **配件局限**：太阳镜、耳环等能处理，但帽子是典型失败案例（图12）。
- **可扩展性**：若基础模型库不包含与输入在发型/头形上相近的模型，质量会下降；对基础模型的数量和质量有一定依赖。
- **无对比增强**：与SDS类方法的对比较少（文中仅DreamGaussian），且未讨论与最新3D-aware GAN或NeRF方法的对比。

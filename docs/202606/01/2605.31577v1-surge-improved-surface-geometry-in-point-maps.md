## 摘要
最近的馈送前向3D重建方法能够预测点图并相当好地估计全局3D几何。然而，它们的预测仍然显示出不准确的局部表面几何，这在定性上清晰可见，但在常见指标中仅微弱反映。为了在评估中更明确地显示这些误差，我们引入了一个点图法线度量，评估由相邻3D预测引起的局部表面方向。为了减少这些误差，我们提出了两个互补组件：一个点梯度匹配损失，用于监督深度归一化的3D有限差分，以及一个邻域注意力解码器（NAD），逐步上采样特征并使用邻域注意力进行局部特征混合。在八个零样本单目几何基准上，我们的模型SurGe在全局点图AbsRel上达到了最佳平均排名，并一致地改进了局部点图和点图法线评估。

## Abstract
Recent feedforward 3D reconstruction methods predict point maps and estimate global 3D geometry remarkably well. However, their predictions still exhibit inaccurate local surface geometry, which is clearly visible qualitatively but only weakly reflected in common metrics. To make these errors more explicit in evaluation, we introduce a point map normal metric that evaluates the local surface orientation induced by neighboring 3D predictions. To reduce these errors, we propose two complementary components: a point gradient matching loss that supervises depth-normalized 3D finite differences, and a Neighborhood Attention Decoder (NAD) that progressively upsamples features and uses Neighborhood Attention for local feature mixing. Across eight zero-shot monocular geometry benchmarks, our model, SurGe, achieves the best average rank for global point map AbsRel and consistently improves local point map and point map normal evaluations.

## 1. 论文的核心问题与整体含义（研究动机和背景）

- **研究动机**：现有的前馈3D重建方法（如MoGe、DUSt3R等）在全局3D几何估计上表现良好，但预测的点图在**局部表面几何**上存在明显的不准确性，表现为薄结构弯曲、振荡、表面不平整等伪影。这些伪影在定性观察中清晰可见，但在常用的点度量（如全局AbsRel）中仅被微弱反映，因为点度量通过平均点位置误差掩盖了局部不一致性。
- **背景**：点图表示（per-pixel 3D point）已被广泛应用于单目/多视图几何估计。当前最先进模型（如MoGe-2、InfiniDepth）虽然整体准确，但局部表面质量仍是瓶颈。作者指出，现有评估指标不能充分捕捉局部表面伪影，且现有损失函数（如点法线损失、log-depth梯度匹配）在局部监督上存在局限。

## 2. 论文提出的方法论：核心思想、关键技术细节

### 核心思想
SurGe旨在**改进点图的局部表面几何**，通过两个互补组件实现：
- **新的评估度量**：点图法线平均角度误差（MAE normal），更直接地反映局部表面方向的一致性。
- **新的损失函数**：点梯度匹配损失（Lpgm），通过深度归一化的3D有限差分对局部表面进行尺度不变的监督。
- **新的解码器架构**：邻域注意力解码器（NAD），使用邻域注意力替代卷积，在渐进式上采样过程中进行内容依赖的局部特征混合。

### 关键技术细节

1. **点梯度匹配损失（Lpgm）**：
   - 受log-depth梯度匹配损失（Li & Snavely）启发，但扩展到向量值的点图。
   - 定义深度归一化3D有限差分：\(\widetilde{\Delta}_x \mathbf{Q}_{ij} = \frac{\Delta_x \mathbf{Q}_{ij}}{\min([\mathbf{Q}_{ij}]_z, [\mathbf{Q}_{i,j+1}]_z)}\)，对垂直方向类似。
   - 损失计算预测与真值归一化差分的欧氏距离，平均所有有效像素对。
   - 特点：克服了log-depth梯度匹配不能直接用于点图的限制；通过深度归一化实现局部尺度不变性；同时监督局部方向与幅度。

2. **邻域注意力解码器（NAD）**：
   - 结构：从DINOv2 ViT-Large编码器的低分辨率token网格出发，经过5个阶段逐步上采样（2×），每个阶段包含 \(n_\ell\) 个NAD块。
   - NAD块：类似Transformer的残差块，但将全局自注意力替换为**邻域注意力**（Neighborhood Attention, window size k=9）。去除标准LayerNorm，使用QK归一化稳定训练。加入窗口匹配的旋转位置编码（RoPE）。
   - 动机：邻域注意力提供内容依赖的局部聚合，优于卷积的固定核；同时避免全分辨率自注意力的高昂成本和ViT解码器的块状伪影。
   - 最终输出投影为 \((\xi, \eta, \rho)\)，映射为 \((\xi e^\rho, \eta e^\rho, e^\rho)\) 得到点。

3. **总损失**：
   - 使用MoGe风格的全局和局部点损失（L_glob, L_loc4, L_loc16, L_loc64），再加上10倍权重的Lpgm。
   - 根据标签质量调整损失项：合成标签用全部，SfM标签去掉L_loc64和Lpgm，LiDAR只用L_glob和L_loc4。

## 6. 论文的主要结论与发现

1. **现有点度量不能充分捕捉局部表面伪影**，提出的MAE normal能有效区分高频与低频扰动。
2. **点梯度匹配损失（Lpgm）** 优于现有表面损失（点法线损失、log-depth梯度匹配），在提升局部几何的同时不损害全局精度（表7-9）。
3. **邻域注意力解码器（NAD）** 相比卷积解码器和ViT解码器，显著改善薄结构和局部表面质量，在所有局部和全局指标上取得最优或接近最优（表4-6）。
4. **SurGe整体在八个零样本基准上达到全局点图最优平均排名，并在局部评估上一致领先**。
5. 定性结果表明，SurGe预测的点图更平滑、薄结构更直、表面细节更锐利。

## 7. 优点

- **问题导向明确**：聚焦长期被忽视的局部表面几何问题，并提供了显式的评估度量（MAE normal）。
- **方法论创新**：
  - 损失函数Lpgm聪明地扩展了log-depth梯度匹配思想，通过深度归一化保持尺度不变性，同时适用于向量场。
  - 解码器NAD首次在密集点图预测中引入邻域注意力，避免了卷积的固定核限制和ViT的块状伪影，设计精巧（去除LayerNorm、窗口匹配RoPE等）。
- **实验严谨**：严格控制变量；消融实验验证了每个组件贡献；训练协议大规模且考虑了高分辨率阶段。
- **性能优异**：在多个指标和基准上达到SOTA，尤其局部指标大幅领先。

## 8. 不足与局限

- **计算成本**：NAD解码器相比ConvStack-L延迟增大约1.3~1.5倍（完整模型），峰值内存增加8%~8%，推理效率有待优化。作者在附录E承认并指出这是未来方向。
- **依赖标签质量**：监督方案需根据标签类型抽取不同损失项，对于混合标签场景需要先验分类，可能带来复杂性。
- **绝对深度恢复**：SurGe仍预测尺度模糊的点图（输出为\(e^\rho\)形式），依赖ROE对齐评估，不直接输出度量深度。但与其他方法一致。
- **未见场景泛化**：零样本评估仅涵盖8个常见基准，未在更具挑战性的野外大尺度或动态场景上测试。
- **对比方法局限性**：InfiniDepth和PPD需要借助MoGe-2进行对齐，可能引入额外误差，最优比较不够纯粹。
- **表面损失仅在合成标签上使用Lpgm**，对于真实数据（SfM、LiDAR）未使用，限制了潜在改进。

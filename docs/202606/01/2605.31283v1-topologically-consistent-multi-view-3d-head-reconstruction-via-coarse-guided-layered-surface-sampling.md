## 摘要
我们提出了SHELLS（基于分层局部采样的语义头部估计），一个高效的前馈框架，用于从多视图图像中重建具有密集语义对应的3D头部。现有方法通常通过局部特征体素独立优化顶点，这种方法将内存密集的特征采样与网格分辨率耦合，限制了密集拓扑（>1万个顶点）的可扩展性，并引入表面噪声。相比之下，SHELLS通过分层采样策略将特征提取与网格分辨率解耦。我们使用带有LoRA适配的DINOv2主干网络提取多视图特征，投影采样稀疏全局特征云，并预测中间粗网格。这个粗先验指导了分层、表面感知的采样壳的构建，这些采样壳作为最终重建的离散搜索空间。SHELLS保持了表面一致性，同时推理GPU内存比体积基线减少了88%（2.4GB对比20GB）。对于18k顶点网格，它将中值配准误差降低了21%到29%，推理速度提升了3.5倍（0.08秒对比0.29秒）。值得注意的是，我们的模型仅使用合成数据进行训练，却能有效泛化到真实世界捕获，消除了先前工作中常见的昂贵、预先配准的多视图数据集的需求。

## Abstract
We present SHELLS (Semantic Head Estimation via Layered Local Sampling), an efficient feed-forward framework for 3D head reconstruction in dense semantic correspondence from multi-view images. Existing methods typically refine vertices independently via localized feature volumes. This approach couples memory-intensive feature sampling to mesh resolution, which limits scalability for dense topologies (> 10k vertices) and introduces surface noise. In contrast, SHELLS decouples feature extraction from mesh resolution via a hierarchical sampling strategy. We extract multi-view features using a DINOv2 backbone with LoRA adaptation, projectively sample a sparse global feature cloud, and predict an intermediate coarse mesh. This coarse prior guides the construction of layered, surface-aware sampling shells that serve as a discrete search space for the final reconstruction. SHELLS maintains surface consistency while using 88% less inference GPU memory (2.4GB vs. 20GB) than volumetric baselines. It reduces median registration error by 21% to 29% with a 3.5x inference speedup (0.08s vs. 0.29s) for 18k-vertex meshes. Notably, our model is trained exclusively on synthetic data yet generalizes effectively to real-world captures, eliminating the need for the costly, pre-registered multi-view datasets common in prior work.

## 1. 核心问题与整体含义（研究动机和背景）

- **问题定义**：从多视图图像中高效重建具有密集语义对应的3D头部网格，是构建数字人类的关键基础。传统方法依赖于多视图立体（MVS）扫描后非刚性配准，流程繁琐、计算量大（每帧数分钟到数小时），且受扫描噪声、空洞和过平滑问题影响。
- **现有方法瓶颈**：前馈方法（如ToFu、TEMPEH）虽实现了亚秒级重建，但存在两个根本性问题：
  - **内存与分辨率耦合**：使用体积特征采样，每个顶点需要512个邻域采样点，总采样数达9×10⁶，内存随网格分辨率线性增长，限制在低顶点数（~3k-5k）。
  - **缺乏全局一致性**：独立逐顶点细化导致表面噪声和拓扑不一致，尤其在遮挡区域（头发、衣物）产生伪影。
- **本文目标**：设计一种内存高效、可扩展到高密度拓扑（18k顶点）、保持表面一致性、且能从合成数据泛化到真实场景的端到端前馈重建框架。

## 2. 方法论

### 核心思想
通过**分层采样策略**将特征提取与网格分辨率解耦：先预测一个低分辨率粗网格，再由粗网格引导构造表面感知的采样壳，作为精细重建的离散搜索空间。整个框架由共享的Transformer（XCiT）在两阶段中分别处理稀疏全局采样和稠密局部采样。

### 关键技术细节

**（1）特征提取**
- 使用**DINOv2-B**作为共享图像编码器，冻结主干，在每层线性层插入**LoRA**适配（rank=5）。
- 拼接4个均匀间隔的DINOv2层特征，通过1×1卷积和像素重组投影到d_f=98维，输出特征图尺寸为27×18（输入384×256）。
- 每张图像独立处理，输出多视图特征图{F_k}。

**（2）粗网格预测（Graph Stage）**
- **稀疏全局采样**：定义采样点云S_g为16层同心二十面体近似球面（经两次细分），共2,592个点。
- **特征融合**：将每个3D点透视投影到所有视图特征图，进行双线性采样，计算视图间均值μ和方差σ²，拼接为2d_f维特征向量。
- **Transformer预测**：
  - 将下采样模板（3,000顶点）经MLP生成模板令牌Z_t。
  - 采样点坐标经MLP生成几何位置嵌入，与融合特征相加后线性投影，得到特征令牌Z_g。
  - 联合令牌Z_c = [Z_t; Z_g]送入**XCiT**（12层，6头，d_m=384）。XCiT采用跨协方差注意力（XCA），复杂度从O(N²)降为O(d_m²)，适合处理大量点。
  - 输出划分为查询令牌Q_c和键令牌K_c，粗顶点坐标通过注意力加权和回归：V_c = Softmax(Q_c K_c^T / √d_m) S_g。

**（3）精细网格预测（Shell Stage）**
- **表面感知采样壳构造**：基于粗网格顶点V_c及其法线N_c，构建三层采样壳S_l = [V_c; V_c + d_l N_c; V_c - d_l N_c]，位移d_l=4mm，总采样点9,000个。
- **可见性感知特征融合**：采用TEMPEH的可见性加权策略。对每个采样点，计算每个视图的权重φ_k = Softplus(δ_k·cosθ_k)，其中δ_k为可见性（深度缓存检查），cosθ_k为法线与视线方向点积。然后计算加权均值μ和加权方差σ²。
- **Transformer预测**：
  - 模板顶点（17,821个）经MLP生成全分辨率模板令牌Z_t。
  - 定义静态模板壳S_t（由下采样模板沿法线偏移得到），用于编码空间关系。S_t坐标经MLP生成位置嵌入，与融合特征F_l相加后线性投影得Z_l。
  - 联合令牌Z_f = [Z_t; Z_l]送入共享XCiT（与粗阶段权重共享），输出Q_f和K_f，最终顶点V_f = Softmax(Q_f K_f^T / √d_m) S_l。

**（4）损失函数**
- **顶点到顶点损失（L_v2v）**：使用对角权重矩阵Ω（面部特征区域权重更高：嘴唇/眼睑5.0，皮肤/眉毛/耳朵/鼻子3.0，其余1.0）。L_v2v = λ_c ||ΩΔV_c||²_F + λ_f ||ΩΔV_f||²_F。
- **顶点到平面损失（L_v2p）**：仅惩罚沿法线方向的位移分量，允许切线滑动：L_v2p = λ_c ||Ω(ΔV_c ⊙ N_gt)1_3||²₂ + λ_f ||Ω(ΔV_f ⊙ N_gt)1_3||²₂。
- **总损失**：L_total = λ_v2v L_v2v + λ_v2p L_v2p，权重均为1.0。

### 算法流程（文字说明）
输入n_i张同步图像和对应相机参数 → DINOv2+LoRA提取多视图特征图 → 将稀疏球面点云投影到各视图，融合特征得到全局特征点云 → XCiT处理后通过注意力加权和回归粗网格 → 沿粗网格法线位移构造采样壳 → 对壳点进行可见性加权特征融合 → 共享XCiT处理全分辨率模板令牌和壳特征令牌 → 注意力加权和回归最终18k顶点网格。

## 3. 实验设计

### 数据集

- **真实测试集**：9,617帧，来自303个主体，记录约50个静态表情，使用13相机校准系统。参考几何通过深度MVS扫描后由3DMM配准获得（包含原始扫描和配准结果）。

### 基准方法
- **TEMPEH**（Bolkart et al. 2023）：前馈多视图配准，在同样合成数据上训练1.4M步（约30天，H100 GPU）。
- **3DMM回归**：预测116个身份系数和197个表情系数，从每视图检测密集地标，多视图取均值。
- **3DMM拟合**（Wood et al. 2022）：对多视图每帧联合拟合3DMM到密集地标，约35秒/帧。

### 评估指标
- **V2V误差**（顶点到顶点）：在合成数据上计算预测顶点与GT顶点的欧氏距离；真实数据中与参考配准顶点比较。
- **P2S误差**（点对面）：计算MVS扫描中皮肤区域顶点到预测网格的最小距离。
- **网格质量**：三角形面积变形（与GT面积差的均值）、三角形翻转率（逆法向占比）。

## 6. 主要结论与发现

- **性能提升**：在合成数据上，SHELLS中值V2V误差（面部）为1.22mm，比TEMPEH（1.71mm）**降低29%**；在真实数据上中值V2V为1.50mm，比TEMPEH（1.90mm）**降低21%**。
- **内存与速度**：推理内存仅2.4GB（TEMPEH 20GB），推理时间0.08秒（TEMPEH 0.29秒），实现高密度拓扑（18k顶点）的高效重建。
- **合成到真实泛化**：仅用合成数据训练即可直接泛化到真实多视图捕获，无需昂贵真实配准数据。
- **全局一致性**：Transformer全局注意力优于逐顶点细化，在遮挡区域（头发、口腔）保持表面平滑和拓扑一致。
- **网格质量**：三角形变形0.38（TEMPEH 0.55），翻转率0.08%（TEMPEH 0.15%），显著优于TEMPEH。

## 7. 优点

- **创新性设计**：粗引导分层采样壳巧妙地将3D搜索空间从全局体积缩小到局部表面附近，总采样点仅11,592个（粗阶段2,592 + 精细阶段9,000），实现内存效率与几何精度的平衡。
- **解耦策略**：特征提取与网格分辨率解耦，使方法可扩展到更高密度拓扑而不增加采样负担。
- **Transformer全局建模**：用XCiT替代独立逐顶点细化，避免了噪声和伪影，提升了对严重遮挡的鲁棒性（通过联合注意力将可见区域特征传播到不可见区域）。
- **合成数据训练**：展示了程序化合成数据（含多样化身份、表情、头发、服饰）足以支持下游真实场景重建，消除了对真实标注数据的依赖，具有重要实用价值。
- **综合评估**：不仅报告V2V/P2S误差，还分析了网格质量（变形、翻转），提供了比单纯几何精度更全面的性能画像。
- **消融完备**：从特征提取和网格预测两个维度系统地验证了每个组件的重要性。

## 8. 不足与局限

- **数据集覆盖局限**：合成训练数据中舌头表达多样性不足，导致对极端舌头动作重建失败。未包含张嘴/说话等动态序列的专门评估。
- **几何细节有限**：18k顶点网格仅能捕获中频特征（宏观形状和中等褶皱），缺乏微观细节（毛孔、细纹、皮肤血管）。距照片级真实感渲染尚有差距。
- **非皮肤表面建模**：仅预测皮肤表面，头发和衣物下的实际体积未被建模。对于需要头发/胡子外轮廓的神经虚拟化身应用，需扩展训练数据标签。
- **P2S指标劣势**：在真实数据上P2S中值误差（0.76mm）高于TEMPEH（0.62mm），表明在某些局部几何精确度上不如逐顶点细化方法。这可能是因为全局注意力牺牲了局部极值拟合能力。
- **单视图性能**：虽然两视图结果尚可，但单视图情况下重建存在病态性，缺乏明确量化评估。
- **基线对比不完整**：未与GRAPE、MOCHI等后续方法或传统优化方法（如VHAP）对比，削弱了在最新SOTA中的定位。
- **计算资源需求**：虽推理轻量，但训练仍需单张H100约2周，对学术实验室的硬件门槛仍然较高。未提供不同规模骨干的扩展性分析。
- **缺乏统计学显著性分析**：未报告方差或置信区间，难以判断性能提升的统计可靠性。

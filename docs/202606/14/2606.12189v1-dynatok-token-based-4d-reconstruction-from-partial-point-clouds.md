## 摘要
我们解决了从部分点云序列进行4D重建的问题，其中深度传感器观测是不完整的、无序的，并且缺乏显式的时间对应关系。这种仅有几何信息的设置由于缺失观测和模糊的动态而具有挑战性。虽然最近的进展主要依赖于基于图像的方法，但现有的基于点的方法通常关注单个物体，假设相对完整的输入，或需要显式的对应关系。为了解决这些限制，我们提出了 DynaTok，一个基于点的框架，用于从部分点云序列进行无对应关系的4D重建，无需图像。DynaTok 将帧编码为紧凑的潜在 token，通过基于 Transformer 的时空编码器聚合随时间变化的不完整观测，并通过残差 token 在统一模型中解耦几何和运动。然后，一个流匹配解码器根据潜在 token 重建完整、时间一致的4D点云序列。在物体和场景级基准测试上的实验表明，从部分点云观测中重建质量和时间一致性得到了改善。项目页面：https://wrchen530.github.io/dynatok/。

## Abstract
We address 4D reconstruction from partial point cloud sequences, where depth-sensor observations are incomplete, unordered, and lack explicit temporal correspondences. This geometry-only setting is challenging due to missing observations and ambiguous dynamics. While recent progress has largely relied on image-based methods, existing point-based approaches typically focus on single objects, assume relatively complete inputs, or require explicit correspondences. To address these limitations, we propose DynaTok, a point-based framework for correspondence-free 4D reconstruction from partial point cloud sequences without images. DynaTok encodes frames into compact latent tokens, aggregates incomplete observations over time with a Transformer-based spatiotemporal encoder, and decouples geometry and motion through residual tokens in a unified model. A flow-matching decoder then reconstructs complete, temporally consistent 4D point-cloud sequences conditioned on the latent tokens. Experiments on object- and scene-level benchmarks demonstrate improved reconstruction quality and temporal coherence from partial point cloud observations. Project page: https://wrchen530.github.io/dynatok/.

## 1. 论文的核心问题与整体含义

- **研究动机**：动态3D场景的4D重建（随时间变化的几何重建）是计算机视觉的基础问题，广泛应用于机器人、AR/VR和场景理解。现实场景中，深度传感器或多视图几何管线通常只能输出**部分点云**——这些点云由于遮挡、视角变化而不完整、无序，且缺乏帧间点对应关系。传统方法大多依赖完整图像或视频输入、水密网格监督或显式对应，难以直接应用于这种“仅有几何”的挑战性设置。
- **核心问题**：如何从**部分、无序、无对应关系的点云序列**中，恢复出**完整、时间一致**的4D几何？难点在于：①单帧可能完全缺失大区域几何；②无图像外观或跟踪线索，难以区分静态结构与动态物体；③无显式对应关系，简单时间融合会导致不一致。
- **整体含义**：本文提出DynaTok，一种纯点云驱动的、无需图像和显式对应的4D重建框架，通过**潜空间时间聚合**来补偿缺失观测，从而在物体和场景尺度上实现高质量、时间连贯的重建。

## 2. 论文提出的方法论

- **核心思想**：将每帧部分点云编码为紧凑的潜在token，通过Transformer进行时空对齐与融合，得到一组共享的潜表征；利用残差token机制在统一模型中解耦时间不变的几何（参考帧）与时变的运动；最后用条件流匹配解码器生成完整点云。
- **关键技术细节**：
  1. **逐帧token提取**：对每帧输入点云 \(X_s \in \mathbb{R}^{N_{\text{in}} \times 3}\)，使用最远点采样（FPS）选取 \(M=512\) 个点作为锚点，通过交叉注意力（以原始点坐标为key/value）和自注意力层提取每帧潜在token \(F_s \in \mathbb{R}^{M \times D_Z}\)。
  2. **时空对齐模块**：基于Transformer架构，交替进行帧内自注意力和帧间全局注意力。帧内保持空间结构，帧间允许token跨时间传播信息。使用3D旋转位置编码（RoPE）增强相对几何推理。整个模块输出对齐后的统一潜表征。
  3. **潜在残差分解**：将第一帧（s=1）的潜token视为参考表征 \(G\)，其他帧表示为 \(Z_s = G + H_s\)，其中 \(H_s\) 为残差运动token（第一帧 \(H_1=0\)）。这种设计使模型在无监督下自动将持久结构分配到 \(G\)，时变运动分配到 \(H_s\)。
  4. **条件流匹配解码器**：采用线性概率路径 \(x_t = (1-t)\epsilon + t x_1\)（\(\epsilon\) 为均匀立方噪声，\(x_1\) 为目标点云）。学习条件速度场 \(\Phi_{\text{dec}}(x_t, t, Z_s)\)，损失函数为 \(L_{\text{FM}} = \mathbb{E}_{t, x_1, \epsilon, Z_s}[\|\Phi_{\text{dec}}(x_t, t, Z_s) - (x_1 - \epsilon)\|_2^2]\)。推理时从噪声开始ODE积分（25步Euler）得到50k个输出点。
- **架构流程**（文字说明）：输入部分点云序列 → 逐帧token提取 → 时空对齐（帧内+帧间注意力）→ 残差分解得到 \(G\) 和 \(H_s\) → 拼接形成 \(Z_s\) → 条件流匹配解码器 → 输出完整点云 \(Y_s\)。整个框架端到端训练，仅需点云监督。

## 6. 论文的主要结论与发现

- 在DT4D-A和Kubric上，DynaTok在所有指标（Accuracy、Completeness、Normal Consistency）上均**显著优于**现有方法，尤其Completeness提升2-3倍，Normal Consistency提高5-10个百分点。

- **鲁棒性强**：对极稀疏输入（512点）仍能恢复整体结构；对真实噪声（CUT3R预测）也能保持合理质量。

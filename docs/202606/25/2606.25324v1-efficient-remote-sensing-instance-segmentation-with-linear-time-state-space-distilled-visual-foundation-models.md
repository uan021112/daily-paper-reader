## 摘要
Transformer的计算复杂度随着token数量呈二次方增长，这严重制约了视觉模型的效率，尤其是近期基于ViT的视觉基础模型在密集预测任务中的表现。实例分割作为遥感领域典型的密集视觉预测任务，面临类似的挑战。本文受大语言模型中知识蒸馏最新进展的启发，提出了RS4D——一种具有线性计算复杂度的新型遥感实例分割方法，通过蒸馏状态空间建模解决了长序列建模的低效问题。我们提出了一种自适应噪声与掩码知识蒸馏训练方法，用于预训练轻量级SSM骨干网络，该方法有效将来自巨大的自注意力空间的知识压缩到紧凑、密集的线性状态空间中。我们还基于该轻量级视觉编码器设计了一种遥感图像实例分割架构，其中探索了三种不同骨干网络和两种分割头的变体。在多个基准数据集（包括SSDD、WHU和NWPU）上进行了大量实验。与基于ViT的方法相比，我们提出的SSM骨干网实现了参数减少8倍，FLOPs减少9倍，同时保持了与基于ViT和CNN的实例分割方法相当或更优的精度。实现代码已在https://github.com/QinzheYang/RS4D公开。

## Abstract
The computational complexity of Transformers scales quadratically with the number of tokens, which significantly constrains the efficiency of vision models, particularly recent ViT-based foundation models in dense prediction tasks. Instance segmentation, a typical dense visual prediction task in the remote sensing field, faces similar challenges. In this paper, inspired by the recent advances of knowledge distillation in large language models, we introduce RS4D - a new remote sensing instance segmentation method with linear computational complexity, which addresses the inefficiency of long sequence modeling through distilled state space modeling (SSM). We propose an adaptive noise and masking knowledge distillation training method for pre-training lightweight SSM backbones, which effectively compresses knowledge from the vast self-attention space into a compact, dense linear state space. We also design a remote sensing image instance segmentation architecture based on this lightweight visual encoder, where we explore variants of three different backbones and two segmentation heads. Extensive experiments are conducted on multiple benchmark datasets, including SSDD, WHU, and NWPU. Compared to ViT-based approaches, our proposed SSM backbone achieves an 8x reduction in parameters and a 9x reduction in FLOPs while maintaining comparable or superior accuracy to both ViT- and CNN-based instance segmentation methods. The implementation codes have been publicly available at https://github.com/QinzheYang/RS4D.

- **核心问题**：Transformer 架构在视觉任务中表现出色，但其自注意力机制的计算复杂度与输入 token 数量呈**二次方关系**。遥感实例分割作为密集预测任务，需要处理高分辨率图像（如 1024×1024 像素），对应极长的 token 序列（例如 4096 个 token），导致 ViT 类基础模型计算开销巨大、推理缓慢，难以部署于资源受限边缘设备（如在轨处理）。
- **研究动机**：近年状态空间模型（SSM）因具备**线性时间复杂度**而成为处理长序列的新范式。同时，大语言模型中的知识蒸馏技术已证明能将大模型知识高效压缩至轻量架构。本文旨在将两者结合，开发一种**既保留基础模型（SAM）分割先验、又实现线性计算效率**的遥感实例分割方法。
- **整体含义**：通过蒸馏 ViT 基础模型的知识到 SSM 轻量骨干中，实现 **8 倍参数压缩、9 倍 FLOPs 下降**，同时保持与 ViT 和 CNN 方法相当甚至更优的精度，为遥感密集预测提供高效新范式。

## 2. 方法论：核心思想、关键技术细节、公式或算法流程

### 核心思想
- 采用**知识蒸馏框架**，教师模型为预训练的 SAM 视觉编码器（ViT-Base），学生模型为轻量级 SSM 网络。
- 在蒸馏过程中引入**自适应噪声与掩码增强**，帮助学生模型学习鲁棒、形状导向的表示，并缓解自然图像（SA-1B）与遥感图像之间的领域差距。
- 下游微调时，将蒸馏后的 SSM 骨干与多级特征融合颈（Neck）及实例分割头组合，在遥感数据集上端到端训练。

### 关键技术细节

#### （1）SSM 学生骨干架构
作者探索了三种架构，最终选用 **ScanningMamba** 作为默认骨干：
- **VanillaMamba**：12 层 Vision Mamba（ViM）块，结构简单但长序列学习能力有限。
- **TransMamba**：混合 Mamba-Transformer 块（TRVim），保留跳跃连接和 GELU 激活以增强优化稳定性，但参数量较大。
- **ScanningMamba**：24 层双向扫描 Mamba 块，同时进行前向和反向扫描，并对结果取平均，以更完整地融合 2D 空间上下文，维持线性复杂度。

关键公式（以 ScanningMamba 为例）：
- 前向：`X_forward_c = Mambamixer(X_{c-1})`
- 反向：`X_backward_c = Mambamixer(flip(X_{c-1}))`
- 合并：`X_c = X_{c-1} + (X_backward_c + flip(X_forward_c)) / 2`

#### （2）蒸馏过程（自适应噪声与掩码）
- 使用 SA-1B 子集（0.1% ~ 10% 样本）作为无标注蒸馏数据。
- 对输入图像施加**随机掩码**和**自适应噪声**：噪声振幅根据上一轮学生输出最大绝对值动态调节：
  - `α = min(max(|Y_{e-1}|), λ_const)`
  - `X' = mask(X_input) + α * ε`，其中 `ε ~ Uniform(0,1)`
- 蒸馏损失为教师与学生输出之间的 **L1 距离**：
  - `L_kd = (1/(b×l×d)) * Σ|y_ij - y_pred_ij|`

#### （3）实例分割架构
- **Neck**：包含下采样特征层、级联残差、简单 FPN（SimpleFPN），从 SSM 骨干的多个中间层选择特征进行融合，而非传统 ResNet+FPN 方式，更适合遥感小目标。
- **Head**：两个变种
  - **Box-based head**：类 Mask R-CNN 结构，使用 RPN + RoI + FCN 掩码头，损失为 `L = L_rpn_cls + L_rpn_bbox + L_roi_cls + L_roi_bbox + L_fcn_mask`。
  - **Prompt-based head**：利用 RoI 特征生成学习到的提示 token，送入 SAM 的 prompt 编码器与解码器生成掩码，损失中 `L_decoder_mask` 替代 FCN 损失。

### 算法流程（文字说明）
1. **阶段一**：在 SA-1B 子集上，以 SAM 编码器为教师，对 SSM 学生骨干进行知识蒸馏，采用自适应噪声与掩码输入，L1 损失监督。

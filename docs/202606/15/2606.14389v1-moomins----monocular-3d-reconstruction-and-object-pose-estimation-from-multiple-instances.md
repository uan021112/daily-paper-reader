## 摘要
从单张单目图像同时进行3D重建和6D物体姿态估计是一个固有的病态问题。然而，在工业场景中，物体的多个实例通常随机堆放在料箱中，这隐式地在单张图像中提供了同一物体的多个视角。我们展示了可以利用这种隐式的多视图几何来同时进行物体的3D重建并估计每个可见物体实例的6D姿态。我们提出了MooMIns，一种新的基于高斯泼溅的方法，它反转了原始的高斯泼溅公式：不是从多个相机渲染单个场景，而是从单个相机渲染多个物体实例。我们的方法使用SAM3实例分割掩码和修改后的运动恢复结构（SfM）流程进行初始化。与学习到的单目深度估计不同，我们基于图像证据进行真正的基于几何的重建，避免了由训练数据先验引起的幻觉。我们在合成和真实的料箱抓取场景中评估MooMIns，并展示了对之前未见物体的精确重建以及单个实例的可靠姿态估计。

## Abstract
Simultaneous 3D reconstruction and 6D object pose estimation from a single monocular image is an inherently ill-posed problem. In industrial settings, however, multiple instances of an object are often randomly arranged in bins, implicitly providing several views of the same object within a single image. We show that this implicit multi-view geometry can be exploited to simultaneously reconstruct the object in 3D and estimate the 6D pose of each visible object instance. We present MooMIns, a new Gaussian-splatting-based approach that inverts the original Gaussian splatting formulation: instead of rendering a single scene from multiple cameras, we render multiple object instances from a single camera. Our method is initialized with SAM3 instance segmentation masks and a modified Structure from Motion (SfM) pipeline. In contrast to learned monocular depth estimation, we perform true geometry-based reconstruction from image evidence, avoiding hallucinations caused by training data priors. We evaluate MooMIns on synthetic and real bin-picking scenarios, and demonstrate accurate reconstruction of previously unseen objects as well as reliable pose estimation of individual instance

- **整体含义**：定义了一个新的机器视觉任务——从单张多实例图像零样本地同时完成3D重建和实例位姿估计，并提出了首个解决方案MooMIns。

## 2. 论文提出的方法论

### 核心思想
- 反转经典高斯泼溅（GS）公式：经典GS是多相机渲染单场景，MooMIns改为**单相机渲染多个物体实例**（将多个实例放置在不同位姿下，从同一固定相机渲染）。这保证了全局光照一致性、隐式引入多视图约束，并允许每次迭代使用全图所有实例信息，收敛更稳定。

### 关键技术细节
- **基础框架**：基于2D高斯泼溅（2D-GS）和反射式球面Voronoi（SV）参数化（分离几何与光照，支持延迟渲染）。
- **初始化流程**：
  1. 使用SAM3进行可见实例分割（通过多模态LLM提示物体描述）。
  2. 用每个物体的掩码图像输入修改后的SfM（COLMAP）生成初始稀疏点云和虚拟相机位姿。
- **修改的GS管道**：
  - **场景生成**：将规范物体高斯集`G_canon`变换到每个实例位姿`CP_O`，组合成`G_multi`（公式2）。
  - **自适应密度控制（ADC）**：修改为考虑每个高斯在多次实例渲染中的可见次数（公式3），以适应单图像多次渲染的情况。
  - **实例丢弃（Instance Dropout）**：随机丢弃60%的实例，防止多个实例外观编码进同一规范物体，并利用丢弃实例的掩码避免跨实例梯度污染。
  - **鲁棒实例移除**：基于每实例中位损失，使用稳健标准差剔除高误差实例（最多移除20%）。
  - **损失函数**：包括光度损失（掩码版本）、背景二值交叉熵损失`L_m`、2D-GS正则化项。
  - **姿态优化**：使用6D旋转表示（Hempel等），9参数，Adam优化器，学习率从0.001衰减到0.00001。
  - **稀疏视图处理**：对规范高斯施加常数30%的丢弃层，防止过拟合。
  - **光照建模**：远场照明用立方体图，近场照明用随机分布在`G_multi`包围盒内的光探头，且只建模单通道强度。
- **渲染过程**：仅计算重叠实例的掩码区域，背景每迭代随机着色，避免黑色高斯“躲藏”在背景中。

### 算法流程（文字说明）
输入：单张多实例RGB图像 + 相机内参 → 初始化（SAM3分割 + SfM点云和位姿）→ 修改的GS管道迭代优化：每次迭代中，将规范高斯变换到当前所有（或随机保留的子集）实例位姿，从固定相机渲染，计算掩码后的光度损失和背景损失，反向传播更新规范高斯参数、位姿和光照 → 输出：规范物体3D模型 + 所有实例6D位姿。

## 6. 主要结论与发现

1. **MooMIns能够有效利用单张多实例图像中的隐含多视图几何**，以基于证据的方式同时进行3D重建和6D位姿估计，避免深度估计幻觉。
2. **对于纹理丰富的非遮挡物体，MooMIns在零样本设定下取得精确结果**（SYN1：MRE 0.98°, MTE 14.17mm, d_c 1.31mm，远优于初始化）。
3. **对于无纹理、反射、遮挡的工业物体，在给定合理初始位姿和几何时，MooMIns仍能大幅改善重建质量和位姿精度**（XYZ1-5：MRE从5°降至0.46~4.82°，MTE从30mm降至0.75~7.98mm，d_c从30~47mm降至0.78~1.66mm）。
4. **所有提出的组件对真实数据至关重要**：无背景损失、无实例丢弃时方法无法收敛或结果退化。
5. **修改后GS管道的收敛半径较大**（旋转噪声≤15°，平移噪声≤150mm），但对较大初始误差敏感。

## 7. 优点

- **任务新颖**：首次明确定义并解决单张多实例图像的零样本3D重建+位姿估计，有实际工业应用价值。
- **方法创新**：巧妙反转高斯泼溅公式，解决光照一致性和多视图约束问题；设计了针对单图像多次渲染的ADC、实例丢弃、鲁棒实例移除等实用技巧。
- **避免学习先验**：完全基于图像证据重建，无训练数据漂移风险，适用于未见物体。
- **架构灵活**：可同时优化光照、几何和位姿，支持反射物体（通过SV参数化）。
- **实验设计合理**：消融实验证明各组件必要性，鲁棒性实验给出收敛半径指导。

## 8. 不足与局限

- **初始化依赖**：整体零样本流程依赖SAM3分割和SfM，在无纹理/反射物体上SfM失败（当前需额外初始信息）。
- **额外约束**：需要已知相机内参，与任务定义（仅RGB图像）有差距。
- **模态掩码缺失**：SAM3只能提供可见掩码，无法处理遮挡情况下的amodal掩码，当前通过实例丢弃部分缓解但未完全解决。
- **几何重建缺陷**：在物体不可见侧面（如仅从上观察的扁平区域）会出现几何错误，这是GS在弱纹理区域的常见问题。
- **实验覆盖有限**：仅评估5张工业图像；未与其他单目重建/位姿方法比较；未报告训练时间或迭代次数。
- **对称物体局限**：在旋转对称物体（XYZ5）上MRE改善有限，暗示方法对对称性敏感。
- **不能直接处理杂乱背景**：目前需使用掩码排除背景，但背景的随机着色技巧可能遗漏一些支持信息。

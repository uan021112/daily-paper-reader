## 摘要
实时多相机三维重建是沉浸式媒体、远程交互和空间计算的关键基础。虽然同步相机阵列被广泛采用，但实现几何一致且可扩展的实时重建仍具挑战。一个关键挑战是外参标定、多视角融合和全局优化之间的紧密耦合，导致重建结果波动、累积误差和系统可扩展性差。我们提出了一种用于标定和无状态实时多视角点云融合的解耦框架（FUSE-Flow），该框架包含两个协作组件：几何对齐的多视角外参标定（GMAC）和可靠性引导的多视角点云融合（FUSE）。这种分离设计避免了目标优化冲突，实现了针对性改进。GMAC模块通过几何约束和多视角重建Transformer精化相机外参，无需标定目标、密集图像或全局集束调整即可实现精确的稀疏视角标定。FUSE模块采用置信度加权和自适应空间哈希实现无状态融合，确保线性的时间和内存消耗。两个模块相互促进：精确的相机位姿提升融合精度，而置信度感知融合纠正标定偏差。在公开数据集和真实相机设置上的验证表明，FUSE-Flow在视觉效果、动态稳定性和可扩展性上优于主流实时重建方法，为大尺度实时三维重建提供了实用解决方案。

## Abstract
Real-time multi-camera 3D reconstruction is a key foundation for immersive media, remote interaction and spatial computing. While synchronized camera arrays are widely adopted, achieving geometrically consistent and scalable real-time reconstruction remains challenging. A key challenge is the close linkage among extrinsic calibration, multi-view fusion and global optimization, which causes fluctuating reconstruction results, cumulative errors and poor system expandability. We propose a decoupled framework for calibration and stateless real-time multi-view point cloud fusion (FUSE-Flow), a framework with two collaborative components: geometry-aligned multi-view extrinsic calibration (GMAC) and reliability-guided multi-view point cloud fusion (FUSE). This split design avoids conflicting optimization objectives for targeted improvement. The GMAC module refines camera extrinsics via geometric constraints and multi-view reconstruction transformers, enabling accurate sparse-view calibration without calibration targets, dense images or global bundle adjustment. The FUSE module integrates confidence weighting and adaptive spatial hashing for stateless fusion, ensuring linear time and memory consumption. The two modules mutually reinforce each other: accurate camera poses boost fusion accuracy, and confidence-aware fusion corrects calibration biases. Validated on public datasets and real camera setups, FUSE-Flow outperforms mainstream real-time reconstruction methods in visual effect, dynamic stability and scalability, providing a practical solution for large-scale real-time 3D reconstruction.

### 2. 论文提出的方法论

- **核心思想**：提出**FUSE-Flow**框架，将实时多相机重建解耦为两个模块：
  1.  **GMAC (Geometry-aligned Multi-view Extrinsic Calibration)**：负责自动、可扩展的外参精化。
  2.  **FUSE (Reliability-guided Multi-view Point Cloud Fusion)**：负责高效、鲁棒的点云融合。
  - 两个模块独立运行，相互促进：GMAC提供的精确位姿提升FUSE的融合质量，而FUSE的置信度感知机制又能吸收GMAC产生的残余标定误差。
- **关键技术细节**：
  - **GMAC模块**:
      1.  **初始位姿预测**：利用预训练的**多视角重建Transformer**（如VGGT）从多张RGB图像中预测出**全局一致的相对位姿**（但缺乏物理尺度）。
      2.  **图一致公制度量提升**：通过一个**加权最小二乘闭式求解器**，对全局共享的单一尺度因子 \( s \) 进行最优估计。关键在于构造**重投影一致性（RC）** 约束，通过最小化所有观测点对的深度残差来求解 \( s \)，并引入基于特征相似性和传感器有效性的权重，确保估计的鲁棒性。
      3.  **循环一致性（CC）验证**：通过检查三元组相机间的循环误差来剔除异常边，进一步精化尺度估计。
      4.  **结果**：最终将相对位姿乘以 \( s \)，得到可直接用于RGB-D系统的公制尺度外参。
  - **FUSE模块**:
      1.  **测量置信度建模（MCM）**：为每个深度像素点计算一个基于**相对深度梯度**和**归一化局部深度方差**的置信度分数，用于衡量单视角观测的可靠性。
      2.  **跨视角几何一致性（CVGC）**：对于每个3D点，仅在一组预计算的、**视野重叠的有限邻域相机**中评估其重投影深度残差，将该残差归一化后聚合，得到跨视角的几何一致性权重。
      3.  **联合融合权重**：使用单视角置信度和跨视角一致性权重的**调和平均**作为最终权重，确保只有在两方面都可靠的点才能获得高权重。
      4.  **自适应空间哈希（ASH）**：构建一个两级的空间哈希结构：先划分粗网格估计点密度，再在稠密区域自适应划分更细的网格，避免了全局状态维护，实现了线性复杂度。
      5.  **代表点选择（RPS）**：在每个细网格单元内，根据密度自适应地保留 \( k \) 个代表点，并通过**置信度加权的远点采样**来挑选，确保所选点既可靠又空间分布均匀。
- **算法流程（文字说明）**：
  1.  **输入**：N个相机的同步RGB-D图像流。
  2.  **GMAC步骤**：a. 将多视角RGB图像输入预训练Transformer，获取相对位姿。b. 构建图，基于重投影深度一致性，使用闭式最小二乘法求解全局尺度因子。c. 通过循环一致性验证优化图结构并更新尺度。d. 输出公制尺度的精确外参。
  3.  **FUSE步骤**：a. 为每个深度像素计算MCM置信度。b. 对于每个有效3D点，在有限邻域相机内评估CVGC。c. 通过调和平均计算最终融合权重。d. 构建两级自适应空间哈希对点云进行组织。e. 在每个哈希单元内，基于权重进行置信度加权的远点采样，选出代表点。
  4.  **输出**：单帧、高精度、低噪声的融合点云。

### 6. 论文的主要结论与发现

1.  **解耦设计有效**：将外参标定和点云融合解耦可以避免任务间干扰，使得两个模块可以独立优化以达到更好的整体性能。
2.  **GMAC显著提升标定精度**：通过图一致公制度量提升，GMAC在所有测试的骨干网络上均显著降低了旋转和平移误差（最高降低32%以上），并能在真实世界中实现0.67%的相对平移精度。
3.  **FUSE实现线性可扩展**：FUSE的线性时间复杂度（\( O(NHW) \)）和静态、无状态设计使其在相机数量增加时，GPU内存占用远低于R3D3等对比方法，同时保持了高帧率（8台相机环境下达到26.3 FPS）。
4.  **FUSE在动态场景下更鲁棒**：由于没有帧间状态累积，FUSE在处理动态物体时优于R3D3，有效避免了重影、运动轨迹等伪影。
5.  **外参精度直接影响重建质量**：通过联合流水线实验证实，GMAC带来的外参精度提升直接转化为重建质量的提升，证明了模块间存在明确的“正向增益传递”关系。

### 7. 优点

- **模块化与解耦设计**：这是最大的亮点。将复杂的MRR问题分解为两个可独立优化、可重用、可替换的模块，极大地提高了系统的灵活性、可维护性和可扩展性。
- **无需标定目标或重训练**：GMAC摆脱了对棋盘格等标定目标的依赖，FUSE是一个无需训练的后处理算法，这使得该系统易于部署到各种不同的硬件和场景。
- **高度的计算效率**：FUSE的静态、线性复杂度设计是其核心工程优势，使得系统在资源有限的硬件上也能实现实时运行，并具有优良的硬件可扩展性。
- **强大的鲁棒性**：通过置信度建模和自适应空间哈希，FUSE对中等程度的标定误差具有容忍度，展现了良好的鲁棒性。这在实际应用中至关重要。

### 8. 不足与局限

- **对极端场景的敏感度**：GMAC的性能在**弱纹理、运动模糊、反射表面、以及相机间重叠区域小**的场景下会下降。FUSE的融合质量依赖深度输入的有效性，在**薄结构、小且远的物体、或深度缺失区域**效果不佳。
- **对预训练骨干网络的依赖**：GMAC的整体表现上限高度依赖于其所使用的**多视角重建Transformer**的性能。如果骨干网络预测的相对位姿本身质量很差，GMAC也无法得到良好的结果。
- **限于实时应用场景**：为了保持线性复杂度和低延迟，FUSE的设计有意回避了全局状态维护和全局优化。这意味着它不适用于需要**亚毫米级工业精密测量**的任务。
- **感知不确定性**：论文提到，GMAC的特征不一致性和**单尺度标度设计**可能限制了其在超大规模相机阵列上的扩展性。
- **局限性总结**：该方法主要面向**沉浸式媒体和大型交互式应用**，追求的是实时性和鲁棒性，而不是绝对的最高几何精度。在一个统一的框架下，性能与效率之间进行了权衡。

## 摘要
基于文本的三维生成在图像和孤立物体上取得了快速进展，但生成手物网格仍具挑战性：输出必须保持语言语义、跨视图一致性、物体几何、关节手部形状以及物理上合理的接触。我们提出TextHOI-3D，一个分阶段框架，利用生成的多视图观察作为文本条件视觉生成与几何感知手物恢复之间的显式接口。TextHOI-3D学习固定视角手物观测的紧凑VQ令牌空间，通过CLIP条件视觉自回归模型从文本预测多视图视觉令牌，并通过先验初始化、多视图联合优化和抗穿透细化恢复统一的手物网格。该设计将语义生成与几何恢复分离，同时通过离散多视图表示保持两个阶段的连接。在HO3D衍生评估中，与单视图方法相比，多视图设置将物体CD从17.26毫米降至4.92毫米，穿透体积从5.3721立方厘米降至0.2193立方厘米，同时改善了手部误差和表面F分数。这些结果支持多视图视觉令牌作为文本驱动三维手物网格创建的有效中间表示。

## Abstract
Text-conditioned 3D generation has progressed rapidly for images and isolated objects, but producing a hand-object mesh remains challenging: the output must preserve language semantics, cross-view consistency, object geometry, articulated hand shape, and physically plausible contact. We present TextHOI-3D, a staged framework that uses generated multi-view observations as an explicit interface between text-conditioned visual generation and geometry-aware hand-object recovery. TextHOI-3D learns a compact VQ token space for fixed-camera hand-object observations, predicts multi-view visual tokens from text with a CLIP-conditioned visual autoregressive model, and recovers a unified hand-object mesh through prior initialization, multi-view joint optimization, and anti-penetration refinement. The design separates semantic generation from geometric recovery while keeping both stages connected by a discrete multi-view representation. On HO3D-derived evaluations, the multi-view setting reduces object CD from 17.26 mm to 4.92 mm and penetration volume from 5.3721 cm^3 to 0.2193 cm^3 compared with a single-view counterpart, while improving hand errors and surface F-scores. These results support multi-view visual tokens as an effective intermediate representation for text-driven 3D hand-object mesh creation.

## 1. 核心问题与整体含义
- **研究动机**：文本到3D生成在图像和孤立物体上进展迅速，但生成手与物体交互（Hand-Object Interaction, HOI）的3D网格仍极具挑战。一个合理的结果必须同时满足：语言语义保持、跨视图一致性、物体几何合理性、手部关节有效性、物理上合理的接触（无穿透或漂浮）。
- **核心问题**：现有直接文本到网格的方法难以同时满足上述所有要求，尤其是手物接触区域常被遮挡，且带配对文本和网格的交互数据有限。
- **整体含义**：提出一种分阶段框架，先由文本生成一致的多视图图像作为中间表示，再恢复3D网格。多视图比网格更容易生成，且能保留轮廓、遮挡模式和接触线索，用于驱动3D恢复。

## 2. 方法论：核心思想、关键技术细节
- **核心思想**：使用离散多视图视觉令牌作为文本条件生成与几何恢复之间的显式接口。将语义生成与几何恢复分离，通过紧凑的VQ令牌空间连接两个阶段。
- **关键技术细节**：
  - **阶段一：离散多视图视觉表示学习（VQ-VAE）**
    - 对同一手物场景的多个固定视角RGB视图（N=6），沿通道维度堆叠为张量X ∈ R^{H×W×3N}。
    - 使用VQ-VAE编码器映射到连续潜特征，通过含4096个条目的码本进行最近邻量化，解码器重建并拆分回各视图。
    - 训练目标：L_VQ = ||X - X̂||² + ||sg[z_e] - z_q||² + β||z_e - sg[z_q]||²，其中β=0.25。
    - 最高分辨率令牌图为16×16，每令牌代表多视图场景单元。
  - **阶段二：文本条件多视图生成（CLIP + VAR）**
    - 使用Visual Autoregressive Modeling (VAR)的下一尺度预测范式：p(R|y)=∏_{k=1}^{K} p(r_k | r_{<k}, y)。
    - CLIP文本编码器提取全局特征c_g（通过AdaLN调制视觉隐藏状态）和令牌级序列c_seq（通过交叉注意力融入局部文本属性）。
    - 训练采用教师强制（teacher forcing），推理时使用分类器无引导（classifier-free guidance）和top-k/top-p采样。
  - **阶段三：手物网格恢复**
    - 分割：HSV颜色空间提取手部和物体掩膜。
    - 补全：Stable Diffusion LoRA进行双补全（恢复被遮挡的手部和物体区域）。
    - 先验初始化：物体分支使用InstantMesh重建；手部分支使用OmniHands风格手部先验和MANO参数模型。
    - 网格修复和SDF滤波去除不稳定几何。
    - 多视图联合优化：变量Ξ = {Θ_h, T_h, R_o, t_o, ΔM_o}，目标函数：
      Ξ* = arg min (λ_mv L_mv + λ_c L_contact + λ_p L_pen + λ_r L_reg)
      - L_contact鼓励手部候选接触点到物体表面的距离在阈值内；
      - L_pen惩罚手部顶点落入物体内部的穿透；
      - 优化分三阶段：全局多视图对齐 → 交互校正 → 局部细化与抗穿透后处理。

## 3. 实验设计

## 6. 主要结论与发现
- 多视图离散视觉令牌是文本驱动3D手物网格生成的有效中间表示。
- 多视图相比单视图显著改善：
  - 物体几何：CD从17.26mm降至4.92mm（降低71.5%），F@5从46.3%升至92.7%，F@10从70.6%升至98.6%。
  - 手部误差：MPJPE从1.46mm降至0.65mm，MPVPE从1.54mm降至0.70mm。
  - 穿透抑制：穿透体积从5.3721 cm³降至0.2193 cm³（减少95.9%），穿透顶点百分比从1.80%降至0.67%。
- 交互诊断显示最终输出保持低穿透深度（平均4.26mm）且维持非零接触比率（CR@2=0.37%），表明优化并非简单分离手物。

## 7. 优点
- **分阶段设计清晰**：将语义生成（多视图令牌）与几何恢复（网格优化）分离，通过离散中间表示连接，易于定位失败原因。
- **紧凑中间表示**：VQ-VAE将多视图压缩为16×16令牌图，码本仅4096项，生成效率高（单帧50步时1张GPU可达40视图/秒）。
- **粗到细生成策略**：VAR先建立全局布局再细化局部细节，CLIP全局AdaLN和局部门槛注意力协同控制语义和属性。
- **多视图联合优化**：结合分割、补全、先验和多种损失（重投影、掩码、接触、穿透），有效提升几何和交互合理性。
- **实验控制严格**：单/多视图对比中所有其他条件固定，公平体现多视图贡献。
- **全面评估指标**：不仅包括常见几何误差，还专门测量穿透体积、深度和接触比率，关注物理合理性。

## 8. 不足与局限

## 摘要
在物流自动化中，对未见物体的精确分割对于在杂乱环境中实现高效的机器人操作至关重要。诸如料箱拣选和货架拣选等任务需要强大的感知能力来处理遮挡、变化的物体形状和复杂的空间排列。传统的基于RGB的方法由于依赖纹理往往过度分割物体，而基于深度的方法则因主要关注几何特征而常常分割不足。为了解决这些局限性，我们提出了DA-Fusion，一种基于可变形注意力的RGB-D融合Transformer，专为未见物体实例分割设计。DA-Fusion有效结合了RGB和深度数据的优势，提高了在杂乱和多层物体环境中的分割精度。我们还引入了物体杂乱料箱数据集（OCBD），这是一个专门为评估自上而下视图中的料箱拣选场景而设计的基准数据集。大量实验表明，DA-Fusion在各种环境中均优于最先进的方法，使其特别适用于现实世界的物流任务。

## Abstract
In logistics automation, precise segmentation of unseen objects is crucial for efficient robotic manipulation in cluttered environments. Tasks such as bin-picking and shelf-picking require robust perception to handle occlusions, varying object shapes, and complex spatial arrangements. Traditional RGB-based methods tend to over-segment objects due to their reliance on texture, while depth-based methods often under-segment by focusing primarily on geometric features. To address these limitations, we propose DA-Fusion, a deformable attention-based RGB-D fusion Transformer designed for unseen object instance segmentation. DA-Fusion effectively combines the strengths of both RGB and depth data, enhancing segmentation accuracy in cluttered and multi-layered object environments. We also introduce the Object Clutter Bin Dataset (OCBD), a benchmark dataset specifically tailored for evaluating bin-picking scenarios in top-down views. Extensive evaluations demonstrate that DA-Fusion outperforms state-of-the-art methods across diverse environments, making it particularly suited for real-world logistics tasks.

- **研究背景**：现有UOIS方法多基于两阶段细化网络（如UOAIS、MSMFormer），计算开销大；Transformer架构（如Mask2Former）虽能捕捉全局上下文，但在跨模态融合上仍不充分。可变形注意力（Deformable DETR）通过动态采样关键点可降低计算成本并聚焦重要区域，为本工作提供了基础。
- **整体含义**：提出DA-Fusion，一种基于可变形自注意力和交叉注意力的RGB-D融合Transformer，在多个基准（包括新提出的OCBD数据集）上达到SOTA，并验证了真实机器人场景的有效性。

## 2. 论文提出的方法论

### 2.1 核心思想
- 利用**可变形注意力机制**在多个编码器层级动态融合RGB和深度特征，避免传统早期/晚期融合的局限性。
- 设计两个专用模块：**Deformable Self-Attention (DS)** 用于在单模态内细化特征；**Deformable Cross-Attention (DC)** 用于跨模态交互（RGB query 采样 depth 的 key/value，反之亦然）。
- 整体框架基于**Swin Transformer 骨干**（4层）提取多尺度特征，每层输出融合后的特征图 \( f_i \)（i=1..4），输入**Mask Transformer 解码器**（像素解码器 + Transformer解码器）生成最终掩码。

### 2.2 关键技术细节
- **输入**：RGB图像 \( I_{\text{rgb}} \) 和深度图（转换为3通道HHA格式）\( I_d \)。
- **Deformable Self-Attention (DS)**：
  - 对每个模态的特征图 \( f \)，从参考网格生成参考点 \( R \)，通过学习偏移 \( \Delta p = \theta_{\text{offset}}(Q) \) 得到变形采样点 \( D = R + \Delta p \)。
  - 使用双线性插值 \( \varphi \) 在变形点采样 key 和 value，计算多头注意力（公式5-7），输出细化特征 \( f' \)。
  - 每个head使用K=? （原文说K个采样点，未给出具体值，一般为4或8）。
- **Deformable Cross-Attention (DC)**：
  - 输入为DS输出的 \( f'_{\text{rgb}} \) 和 \( f'_d \)。
  - 分别从RGB query预测偏移到depth参考点，从depth query预测偏移到RGB参考点（公式9-10），获取跨模态的key/value。
  - 计算交叉注意力：RGB query 与 depth 的 key/value 交互，depth query 与 RGB 的 key/value 交互（公式13-14）。
  - 输出合并为当前层的融合特征 \( f_i \)。
- **解码器**：沿用Mask2Former的掩码预测机制，采用二元交叉熵 + Dice损失 + 分类损失（前景/背景），损失权重为 \( \lambda_{\text{ce}}=\lambda_{\text{dice}}=5.0 \)，\( \lambda_{\text{cls}} \) 分别为2.0（前景）和0.1（背景）。

### 2.3 算法流程（文字说明）
1. 输入RGB和HHA深度图，分别送入两个并行的Swin Transformer骨干。
2. 在骨干每一层（共4层）依次执行：DS细化（单模态内）→ DC融合（跨模态交叉注意力），得到当前层融合特征 \( f_i \)。
3. 将所有4层融合特征送入Mask Transformer的像素解码器（通常为多尺度特征金字塔）。
4. Transformer解码器基于可学习查询预测掩码嵌入，经像素解码器上采样后得到边界掩码 \( M \)（阈值化后为二值掩码）。

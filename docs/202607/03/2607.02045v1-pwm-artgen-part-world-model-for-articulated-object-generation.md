## 摘要
从单张图像生成铰接3D物体的关键挑战在于准确预测潜在的运动学结构。现有方法要么直接从缺乏动态部件级运动关系的静态图像中推断运动学参数，要么从单张图像生成的视觉动态中估计参数，这容易产生两步累积误差。此外，现有标注数据集的规模和多样性有限，进一步阻碍了向复杂真实世界物体的泛化。为克服这些限制，我们提出学习视觉动态与运动学参数的联合分布。认识到铰接物体可表示为动态系统，我们提出了一个统一的部分世界模型，称为PWM-ArtGen。为利用未标注数据，该模型将动作扩散和图像扩散与独立扩散时间步耦合，从而实现视觉分支的协同训练。我们还精心构建了一个包含19.7k对部件级图像对且无运动学标注的逼真数据集，以支持协同训练。实验表明，PWM-ArtGen在静态状态下显著优于现有基线，并对分布外物体展现出强大的零样本泛化能力。

## Abstract
The key challenge in articulated 3D object generation from a single image is accurately predicting the underlying kinematic structure. Existing methods either infer kinematic parameters directly from a static image that lacks dynamic part-level kinematic relationships, or estimate parameters from visual dynamics generated from a single image, which is prone to accumulated errors of two steps. Moreover, the limited scale and diversity of existing annotated datasets further hinder generalization to complex, real-world objects. To overcome these limitations, we propose to learn the joint distribution of visual dynamics and kinematic parameters. Recognizing that articulated objects can be formulated as dynamic systems, we propose a unified Part World Model called PWM-ArtGen. To leverage unannotated data, this model couples action diffusion and image diffusion with independent diffusion timesteps, which enables visual branch co-training. We further curate a photorealistic dataset of 19.7k part-level image pairs without kinematic annotations, to support co-training. Experiments demonstrate that PWM-ArtGen substantially outperforms existing baselines in the resting state and exhibits strong zero-shot generalization to out-of-distribution objects.

### 1. 论文的核心问题与整体含义（研究动机和背景）

- **核心问题**：从单张静态图像生成铰接3D物体时，准确预测其隐藏的运动学结构（如关节类型、轴位置、运动范围等）非常困难。现有方法存在两类缺陷：
  - 直接基于静态图像推理运动学参数，缺乏部件级动态运动关系，可辨识性差。
  - 先由单张图像生成视觉动态（如视频），再估计参数，容易累积两阶段误差。
  - 此外，现有标注数据集（如PartNet-Mobility）规模小、缺乏真实感，限制了向复杂真实场景的泛化；而真实感数据集（如ACD）规模又太小。
- **整体含义**：视觉动态与运动学参数本质上是相互依赖的——观测到的运动约束了可行的关节类型与轴，而运动学参数解释了运动模式。因此，联合学习二者的分布有望克服上述局限，并可以利用大量无标注的视觉数据。

### 2. 论文提出的方法论：核心思想、关键技术细节、公式或算法流程

- **核心思想**：提出一个统一的部分世界模型 **PWM-ArtGen**，通过耦合动作扩散（action diffusion）与图像扩散（image diffusion），且二者使用独立扩散时间步，学习视觉动态（后动作观测图像 \(o'\)）与运动学参数 \(a\) 的联合分布 \(p(o', a \mid o, p_{\text{ref}})\)，其中 \(o\) 为输入静态图像，\(p_{\text{ref}}\) 为目标部件结构指引。
- **关键技术细节**：
  - **数据表示**：对每个铰接部件，动态视觉为256×256 RGB图像；运动学属性扩增为 \(a = \{t, r, l, d, b, b_{\text{base}}\}\)，包括关节类型、运动范围、轴位置、轴方向、部件包围盒、基座包围盒；部件级结构指引 \(p_{\text{ref}}\) 由部件裁剪图像、2D包围盒等组成，经DINOv2编码为嵌入 \(f_{\text{ref}}\)。
  - **模型架构**：基于Diffusion Transformer，将静态观测的全局特征 \(f_o\) 与参考部件嵌入 \(f_{\text{ref}}\) 拼接，并注入独立扩散时间步 \(t_a, t_{o'}\) 作为条件。视觉分支使用冻结的SDXL VAE编码后 patchify；运动学分支经MLP投影为20个token，与视觉token拼接后送入Transformer。
  - **视觉动态正则化器（VDR）**：受REPA启发，将去噪过程中前 \(l\) 层的视觉隐藏状态与DINOv2的patch特征进行余弦相似度对齐，损失函数为：
    \[
    \mathcal{L}_{\text{VDR}}(\theta) := -\mathbb{E}\left[\frac{1}{N}\sum_{n=1}^N \text{sim}\left(y_{o'}^{[n]}, h_{t_{o'}}^{[n]}\right)\right]
    \]
    促使视觉表示保持与动作一致的运动动态。
  - **协同训练（Co-Training）**：在标注数据 \(D_{\text{paired}}\) 上正常优化 \(\mathcal{L}_{\epsilon_a, \epsilon_{o'}}\)；在无动作标注数据（仅有 \(o'\)）\(D_{\text{free}}\) 上，将动作扩散时间步固定为最大 \(T\)（即用纯噪声impute动作），仅优化图像部分的噪声预测，并通过权重 \(\gamma\) 平衡。

- **算法流程**：给定输入图像 \(o\)，先由SAM+GPT-4o生成部件掩码和连接图；对每个部件执行PWM-ArtGen采样 \((o'_i, a_i) \sim p(o', a \mid o, p_{\text{ref}}^i)\)；然后基于共享基座包围盒进行装配；最后检索数据库中最匹配的网格以规整几何与运动。

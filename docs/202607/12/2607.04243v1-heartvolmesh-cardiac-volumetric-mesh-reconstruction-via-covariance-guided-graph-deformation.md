## 摘要
准确的病人特异性四面体心脏网格对于硅内试验至关重要，但常见的分割-然后建模管道可能会模糊薄壁解剖结构并提供有限的跨案例对应性。我们提出了HeartVolMesh，它将每个模板顶点提升为各向异性高斯核，并使用3D CNN-GNN从体积图像预测每个顶点的位移和Cholesky参数化的协方差。训练通过协方差感知的负对数似然损失和轻量级网格正则化来指导。对于体积网格划分，我们通过分阶段对齐、非刚性配准和变形传播将固定的四面体模板扭曲到重建表面，通过构造保持连通性和对应性，分辨率由模板密度控制。实验表明，在表面网格精度和体积网格保真度方面，与基于变形的基线相比有一致的改进。

## Abstract
Accurate patient-specific tetrahedral cardiac meshes are essential for in-silico trials, yet common segmentation-then-modelling pipelines can blur thin-wall anatomy and offer limited cross-case correspondence. We propose HeartVolMesh, which lifts each template vertex to an anisotropic Gaussian kernel and uses a 3D CNN-GNN to predict per-vertex displacements and Cholesky-parameterized covariances from volumetric images. Training is guided by a covariance-aware negative log-likelihood loss with lightweight mesh regularization. For volumetric meshing, we warp a fixed tetrahedral template to the reconstructed surface via staged alignment, non-rigid registration, and deformation propagation, preserving connectivity and correspondence by construction, with resolution controlled by template density. Experiments show consistent gains over deformation-based baselines in surface mesh accuracy and volumetric mesh fidelity.

## 2. 论文提出的方法论：核心思想、关键技术细节

**核心思想**：将每个模板顶点提升为各向异性高斯核，通过预测位移和协方差矩阵来捕捉边界不确定性；然后利用协方差感知的负对数似然损失训练；最后通过模板驱动的体积-表面配准策略将固定四面体模板变形到重建表面，从而获得具有对应性的四面体网格。

**关键技术细节**：

- **协方差引导的图变形（Covariance-guided Graph Deformation）**
  - 每个网格顶点不再是一个确定性点，而是一个各向异性高斯核：均值为预测的变形后顶点位置，协方差矩阵Σ（3×3）由网络预测。
  - 为保证Σ始终对称正定，采用Cholesky参数化：Σ = LL<sup>T</sup> + εI，其中L为下三角矩阵（对角元经softplus正约束），ε为小常数。
  - 网络架构：3D CNN编码器提取多尺度体素特征，插值到当前顶点位置得到顶点特征；GNN在网格图上传播特征，保持空间一致性和拓扑一致性；采用粗到细的多级细化（渐进上采样和更新）；最终头部输出逐顶点位移和Cholesky参数。

- **模板驱动的体积网格生成（Template-driven Volumetric Mesh Generation）**
  - 给定从四面体模板提取的边界表面S，以及协方差变形预测的表面T。
  - 先进行全局相似对齐（Umeyama方法，旋转、平移、缩放）。
  - 再应用平滑的非刚性表面配准（Coherent Point Drift）得到S上每个顶点的逐点位移。
  - 将表面位移转换为连续的三维变形场：栅格化到体素网格上，插值/上采样，可选高斯模糊平滑。
  - 多结构场景：每个结构单独计算变形场，合并为共享网格上的单一密集变形场。
  - 最后将四面体模板的所有顶点根据变形场进行变形：v<sub>T</sub> = v<sub>S</sub> + D(v<sub>S</sub>)，得到病人特异性的四面体网格，其边界匹配重建表面，且保留模板的连接性和对应性。

- **损失函数**
  - 表面匹配损失采用协方差引导的双向最近邻负对数似然：ℓ(x,y;Σ) = ½(x−y)<sup>T</sup>Σ⁻¹(x−y) + β log det(Σ)。第一项为马氏距离，第二项防止协方差膨胀。
  - 加上轻量级网格正则化：边缘长度正则化（L<sub>edge</sub>）、拉普拉斯平滑（L<sub>lap</sub>）、法向一致性（L<sub>norm</sub>），总损失：L = L<sub>surf</sub> + λ<sub>edge</sub>L<sub>edge</sub> + λ<sub>lap</sub>L<sub>lap</sub> + λ<sub>norm</sub>L<sub>norm</sub>。

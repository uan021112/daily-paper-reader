## 摘要
我们提出了 ViewSplat，这是一种视角自适应的 3D 高斯泼溅网络，用于从无位姿图像中进行新视角合成。虽然最近的前馈 3D 高斯泼溅通过绕过逐场景优化显著加速了 3D 场景重建，但仍存在根本性的保真度差距。我们将这一瓶颈归因于单步前馈网络在回归满足所有视角的静态高斯原语方面的能力有限。为了解决这一局限性，我们将范式从静态原语回归转向视角自适应的动态泼溅。我们的流水线学习的是一种视角可自适应的潜表示，而非僵化的静态高斯表示。具体而言，ViewSplat 最初预测基础高斯原语以及动态 MLP 的权重。在渲染过程中，这些 MLP 以目标视角坐标作为输入，并预测每个高斯属性（即 3D 位置、缩放、旋转、不透明度和颜色）的视角相关残差更新。我们将这种机制称为视角自适应动态泼溅，它允许每个原语纠正初始估计误差，从而有效地捕捉高保真外观。大量实验表明，ViewSplat 在保持快速推理（17 FPS）和实时渲染（154 FPS）的同时，实现了最先进的保真度。

## Abstract
We present ViewSplat, a view-adaptive 3D Gaussian splatting network for novel view synthesis from unposed images. While recent feed-forward 3D Gaussian splatting has significantly accelerated 3D scene reconstruction by bypassing per-scene optimization, a fundamental fidelity gap remains. We attribute this bottleneck to the limited capacity of single-step feed-forward networks to regress static Gaussian primitives that satisfy all viewpoints. To address this limitation, we shift the paradigm from static primitive regression to view-adaptive dynamic splatting. Instead of a rigid Gaussian representation, our pipeline learns a view-adaptable latent representation. Specifically, ViewSplat initially predicts base Gaussian primitives alongside the weights of dynamic MLPs. During rendering, these MLPs take target view coordinates as input and predict view-dependent residual updates for each Gaussian attribute (i.e., 3D position, scale, rotation, opacity, and color). This mechanism, which we term view-adaptive dynamic splatting, allows each primitive to rectify initial estimation errors, effectively capturing high-fidelity appearances. Extensive experiments demonstrate that ViewSplat achieves state-of-the-art fidelity while maintaining fast inference (17 FPS) and real-time rendering (154 FPS).

*   **核心问题**：当前的前馈（Feed-Forward）3D高斯泼溅（3DGS）模型虽然重建速度快，但在图像保真度上与基于逐场景优化的方法存在显著差距。

*   **整体含义**：提出一种从“静态基元回归”向“视角自适应动态泼溅”的范式转变，通过目标视角驱动的动态微调来提升合成质量。

### 2. 方法论
*   **核心思想**：不要求网络预测完美的全局高斯属性，而是预测一组“基础高斯”和一组“动态MLP权重”。在渲染特定视角时，利用动态MLP根据目标视角实时计算高斯属性的残差修正。
*   **关键技术细节**：
    *   **视角相关头（View-Dependent Head）**：采用超网络（Hypernetwork）架构，根据输入图像的特征令牌（Tokens）为每个像素生成一个轻量级的“视角MLP（View MLP）”。

    *   **动态属性更新**：视角MLP输出针对每个高斯基元的5类属性残差：位置（$\Delta\mu$）、不透明度（$\Delta\alpha$）、旋转（$\Delta r$）、缩放（$\Delta s$）和球谐系数（$\Delta c$）。

### 6. 主要结论与发现
*   **性能领先**：ViewSplat 在 RE10K 和 ACID 上均刷新了 SOTA 记录，尤其在处理镜面反射和高频细节方面优势明显。
*   **正交改进**：该视角自适应机制在不同的主干网络（SPFSplat, V2, V2-L）上均能带来一致的性能提升，证明了其通用性。
*   **效率平衡**：虽然引入动态更新略微降低了纯渲染速度（从300+ FPS降至150+ FPS），但仍远超实时要求（30 FPS），且显著提升了画质。

### 7. 优点
*   **高保真度**：有效解决了前馈模型难以捕捉非朗伯体效应（如反光、高光）的顽疾。
*   **无需位姿**：直接从无标定图像重建，极大地扩展了野外（In-the-wild）应用场景。
*   **动态修正**：通过残差机制允许模型在推理时“纠正”几何误差，增强了系统的鲁棒性。

### 8. 不足与局限
*   **缺乏生成先验**：作为一种重建方法，它无法“幻觉”出未观察到的区域，在稀疏视角下未观测区域仍会出现模糊或空洞。
*   **计算开销增加**：相比于原始静态3DGS，每个视角的渲染都需要经过一次MLP前馈，对于极低功耗设备可能存在压力。
*   **依赖初始几何**：如果基础几何预测（Base Gaussians）偏差过大，仅靠残差修正可能难以完全恢复正确的结构。

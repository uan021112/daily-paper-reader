## 摘要
前馈 3D 基础模型面临一个关键挑战：全局注意力机制带来的二次计算成本，这严重限制了模型随输入长度增加的可扩展性。现有的加速方法（如 token 合并）主要在 token 级别运行。虽然它们能节省局部开销，但所需的最近邻搜索会引入额外的负担。因此，这些技术未能解决密集采集数据中占主导地位的结构冗余这一根本问题。在这项工作中，我们提出了 S-VGGT，这是一种在结构帧级别处理冗余的新方法，极大地转变了优化重点。我们首先利用初始特征构建一个密集场景图，用以表征场景的结构冗余并指导后续的场景划分。利用该图，我们将帧软分配给少量的子场景，从而保证了平衡的分组和顺滑的几何过渡。核心创新在于设计子场景以共享一个公共参考帧，建立一个并行的几何桥梁，从而在无需显式几何对齐的情况下实现独立且高效的处理。这种结构重组通过从源头上削减全局注意力成本，提供了强大的内在加速。至关重要的是，S-VGGT 与 token 级加速方法完全正交，允许两者无缝结合以实现复合加速，且不损害重建保真度。代码可在 https://github.com/Powertony102/S-VGGT 获取。

## Abstract
Feed-forward 3D foundation models face a key challenge: the quadratic computational cost introduced by global attention, which severely limits scalability as input length increases. Concurrent acceleration methods, such as token merging, operate at the token level. While they offer local savings, the required nearest-neighbor searches introduce undesirable overhead. Consequently, these techniques fail to tackle the fundamental issue of structural redundancy dominant in dense capture data. In this work, we introduce \textbf{S-VGGT}, a novel approach that addresses redundancy at the structural frame level, drastically shifting the optimization focus. We first leverage the initial features to build a dense scene graph, which characterizes structural scene redundancy and guides the subsequent scene partitioning. Using this graph, we softly assign frames to a small number of subscenes, guaranteeing balanced groups and smooth geometric transitions. The core innovation lies in designing the subscenes to share a common reference frame, establishing a parallel geometric bridge that enables independent and highly efficient processing without explicit geometric alignment. This structural reorganization provides strong intrinsic acceleration by cutting the global attention cost at its source. Crucially, S-VGGT is entirely orthogonal to token-level acceleration methods, allowing the two to be seamlessly combined for compounded speedups without compromising reconstruction fidelity. Code is available at https://github.com/Powertony102/S-VGGT.

前馈 3D 基础模型（如 VGGT）虽然实现了无需优化的快速 3D 重建，但面临一个致命的瓶颈：**全局注意力机制（Global Attention）的计算成本随输入帧数 $N$ 呈二次方增长（$O(N^2)$）**。
*   **现有痛点**：在处理长序列或密集采集的视频时，计算量和显存占用激增。现有的加速方案（如 Token Merging）主要在 Token 级别进行压缩，不仅引入了额外的搜索开销，还可能因特征分布偏移导致几何精度下降。
*   **研究动机**：作者观察到密集序列的冗余主要存在于“帧级别”。通过将大场景分解为多个子场景并并行处理，可以从源头上解决二次方复杂度问题。

#### 2. 论文提出的方法论
S-VGGT 的核心思想是**结构感知的子场景分解**，通过将长序列划分为多个共享参考帧的子场景来实现并行加速。

*   **场景图构建与密度评估**：利用 VGGT 提取的轻量级帧特征计算两两之间的余弦相似度，构建相似度矩阵 $S$。根据场景的观测密度（重叠程度）动态决定子场景的数量 $K$。
*   **基于软分配（Soft Assignment）的分组**：
    *   构建分配矩阵 $A \in \mathbb{R}^{N \times K}$，通过轻量级优化（梯度下降）来确定每帧所属的子场景。
    *   **损失函数设计**：
        1.  **一致性损失 ($L_{coh}$)**：确保子场景内部具有良好的几何连接性。
        2.  **平衡损失 ($L_{bal}$)**：防止某些子场景过大，确保并行效率。
        3.  **锐化损失 ($L_{sharp}$)**：促使软分配趋向于确定的硬分配（One-hot）。
*   **锚帧共享机制（Anchor Frame Sharing）**：这是实现无需显式对齐的关键。所有子场景强制包含“第 0 帧”作为公共参考点。由于 VGGT 以第一帧为坐标系原点，这种设计使得所有子场景的输出天然处于统一的全局坐标系中。
*   **复杂度分析**：将复杂度从 $O((NT)^2)$ 降低到 $O((NT)^2/K)$，其中 $T$ 为每帧的 Token 数。

#### 6. 主要结论与发现
*   **显著加速**：在 500 帧场景下，S-VGGT 达到了约 **10 FPS**，比原始 VGGT* 快 **3.6x - 4x**。
*   **精度不降反升**：在 ScanNet 位姿估计中，S-VGGT 的 ATE（0.145）优于原始 VGGT（0.190）。作者认为这是因为子场景划分过滤了长序列中无关帧带来的噪声干扰。
*   **正交互补性**：S-VGGT（帧级）可与 FastVGGT（Token 级）结合，在 700 帧序列下实现高达 **5.8x** 的复合加速。
*   **内存友好**：通过并行处理子场景，有效规避了长序列导致的显存溢出问题。

#### 7. 优点
*   **架构优雅**：通过“锚帧共享”巧妙解决了子场景对齐难题，避免了传统 SfM 中复杂的 Bundle Adjustment 或后处理对齐。
*   **即插即用**：无需重新训练模型，对现有基于全局注意力的 3D 基础模型具有普适性。
*   **高效性**：针对密集数据的结构冗余进行优化，比 Token 级压缩更直接、更有效。

#### 8. 不足与局限
*   **锚帧依赖**：高度依赖第 0 帧的质量。如果起始帧模糊或视角极差，可能会影响所有子场景的对齐质量。
*   **帧级开销**：虽然 $O(N^2)$ 的帧级相似度计算远小于 Token 级，但当 $N$ 达到数万帧时，场景图构建本身可能成为新的瓶颈。
*   **应用限制**：主要针对具有显著重叠的密集采集场景，对于视角跨度极大、重叠度极低的稀疏输入，子场景划分的效果可能受限。

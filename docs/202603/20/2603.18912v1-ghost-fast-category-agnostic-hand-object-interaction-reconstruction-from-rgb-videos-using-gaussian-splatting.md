## 摘要
从单目 RGB 视频中理解真实的手-物交互对于 AR/VR、机器人技术和具身智能至关重要。现有方法依赖于特定类别的模板或繁重的计算，但在 3D 空间中产生的手-物对齐仍存在物理上的不一致性。我们提出了 GHOST（Gaussian Hand-Object Splatting），这是一个利用 2D 高斯泼溅重建动态手-物交互的快速、类别无关框架。GHOST 将手和物体都表示为稠密且视角一致的高斯圆盘，并引入了三项关键创新：(1) 几何先验检索和一致性损失，用于补全被遮挡的物体区域；(2) 抓取感知对齐，通过细化手部平移和物体尺度以确保真实的接触；(3) 手部感知背景损失，防止对被手部遮挡的物体区域进行惩罚。GHOST 能够从单段 RGB 视频中实现完整、物理一致且可动画化的重建，同时运行速度比之前的类别无关方法快一个数量级。在 ARCTIC、HO3D 和野外数据集上的广泛实验表明，该方法在 3D 重建和 2D 渲染质量方面达到了最先进的精度，确立了 GHOST 作为真实手-物交互建模的高效且鲁棒的解决方案。代码可在 https://github.com/ATAboukhadra/GHOST 获取。

## Abstract
Understanding realistic hand-object interactions from monocular RGB videos is essential for AR/VR, robotics, and embodied AI. Existing methods rely on category-specific templates or heavy computation, yet still produce physically inconsistent hand-object alignment in 3D. We introduce GHOST (Gaussian Hand-Object Splatting), a fast, category-agnostic framework for reconstructing dynamic hand-object interactions using 2D Gaussian Splatting. GHOST represents both hands and objects as dense, view-consistent Gaussian discs and introduces three key innovations: (1) a geometric-prior retrieval and consistency loss that completes occluded object regions, (2) a grasp-aware alignment that refines hand translations and object scale to ensure realistic contact, and (3) a hand-aware background loss that prevents penalizing hand-occluded object regions. GHOST achieves complete, physically consistent, and animatable reconstructions from a single RGB video while running an order of magnitude faster than prior category-agnostic methods. Extensive experiments on ARCTIC, HO3D, and in-the-wild datasets demonstrate state-of-the-art accuracy in 3D reconstruction and 2D rendering quality, establishing GHOST as an efficient and robust solution for realistic hand-object interaction modeling. Code is available at https://github.com/ATAboukhadra/GHOST.

#### 1. 核心问题与整体含义（研究动机和背景）
从单目 RGB 视频中重建 3D 手-物交互（HOI）是 AR/VR、机器人和具身智能的基础。然而，该领域面临三大挑战：
*   **相互遮挡：** 手部和物体之间频繁的遮挡导致几何缺失。
*   **深度歧义：** 单目视角难以确定准确的物体尺度和手部位置，导致物理接触不真实（如穿模或悬空）。
*   **计算效率：** 现有方法要么依赖特定物体模板（泛化性差），要么基于 NeRF 优化（耗时数小时），难以实际应用。
**GHOST** 旨在通过 2D 高斯泼溅（Gaussian Splatting）技术，实现一个**快速、无需物体模板（类别无关）且物理一致**的重建框架。

#### 2. 方法论：核心思想与关键技术
GHOST 将手和物体均表示为稠密的 2D 高斯圆盘，通过三个阶段实现重建：

*   **预处理与先验检索：**
    *   使用 SAM2 进行分割，SfM（如 VGGSfM）估计相机位姿。
    *   利用视觉语言模型（InternVL）生成物体描述，通过 **OpenShape** 在 Objaverse 数据库中检索最接近的 3D 几何先验。
    *   使用 HaMeR 初始化手部网格，并进行抖动检测与插值平滑。
*   **手-物对齐（HO Alignment）：**
    *   **抓取检测：** 通过比较手部和物体的运动轨迹余弦相似度，识别交互帧。
    *   **联合优化：** 优化物体尺度 $S_o$ 和手部平移 $T_h$。损失函数包含：接触损失（$L_{contact}$，强制手与物体表面接近）、2D 投影损失（保持图像一致性）和时间平滑损失。
*   **高斯泼溅优化：**
    *   **物体端：** 引入**几何一致性损失（$L_{geo}$）**，利用检索到的先验补全遮挡区域；引入**手部感知背景损失（$L_{bkg,h}$）**，确保被手遮挡的物体区域不被错误剔除。
    *   **手部端：** 将高斯点绑定在 MANO 手部网格的每个面片上，通过仿射变换随手部动作变形，实现可动画化的渲染。

#### 3. 实验设计
*   **数据集：**
    *   **ARCTIC：** 复杂的双手交互数据集（Benchmark 核心）。
    *   **HO3D：** 经典的单手-物体交互数据集。
    *   **In-the-wild：** 使用 GoPro 拍摄的野外视频（钻头、书本），验证泛化性。
*   **对比方法：** 
    *   **HOLD：** 基于 NeRF 的类别无关重建方法。
    *   **BIGS：** 基于 3D 高斯泼溅的最新竞争方法。
*   **评价指标：** 3D 指标（MPJPE 手部误差、Chamfer Distance 物体误差）；2D 渲染指标（PSNR、SSIM、LPIPS）。

#### 4. 资源与算力
*   **硬件：** 使用单张 **NVIDIA RTX A6000** GPU。
*   **耗时：** 
    *   GHOST 的总优化时间约为 **1 小时**（针对 300 帧序列）。
    *   相比之下，HOLD 需要约 16 小时，BIGS 需要约 13 小时。GHOST 实现了 **13 倍以上**的加速。

#### 5. 实验数量与充分性
*   **实验规模：** 在 ARCTIC 和 HO3D 的多个序列上进行了定量评估，并提供了大量的定性可视化结果。
*   **消融实验：** 针对 SfM 算法选择（HLoc vs VGGSfM）、几何损失 $L_{geo}$ 的超参数、手部感知背景损失 $L_{bkg,h}$ 的有效性、以及每个面片的高斯点数量 $m$ 进行了详细测试。
*   **充分性：** 实验覆盖了从 3D 几何精度到 2D 渲染质量的全方位对比，且包含野外数据验证，实验设计较为客观、公平。

#### 6. 主要结论与发现
*   **精度领先：** GHOST 在 ARCTIC 数据集上的手-物交互指标（CD）显著优于 HOLD 和 BIGS。
*   **渲染卓越：** 在保持 3D 物理一致性的同时，其 2D 渲染质量（PSNR/LPIPS）在 ARCTIC 上达到了 SOTA。
*   **效率极高：** 成功将此类任务的耗时从“天/小时”级别降低到了“小时/分钟”级别，更具实用价值。
*   **先验的重要性：** 引入外部 3D 数据库先验能有效解决单目视频中严重的自遮挡和相互遮挡问题。

#### 7. 优点（亮点）
*   **类别无关：** 不需要预先知道物体的精确 CAD 模型，通过检索实现通用化。
*   **物理一致性：** 专门的抓取感知对齐阶段解决了单目重建中常见的“手物分离”问题。
*   **可动画化：** 重建的手部不仅是静态点云，而是可以随参数驱动的变形高斯头像。
*   **鲁棒性：** 针对 HaMeR 的预测抖动设计了专门的后处理机制。

#### 8. 不足与局限
*   **先验依赖：** 如果 OpenShape 检索到的几何先验与实际物体差异过大，可能会导致重建结果偏离真实形状。
*   **静态接触假设：** 手部感知背景损失在手部与物体接触点始终不发生改变的情况下（如死死握住不动）可能失效，导致产生“高斯空洞”。
*   **物体类型限制：** 目前主要针对刚性物体，尚未扩展到可变形物体或具有复杂关节的铰接物体。
*   **SfM 敏感性：** 整个流程高度依赖于 SfM 的初始化质量，若相机轨迹估计失败，后续优化难以补救。

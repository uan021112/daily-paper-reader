## 摘要
开放词汇三维语义分割旨在分割训练集之外的任意类别。现有方法主要依赖于从二维开放词汇模型中蒸馏知识。然而，将三维特征对齐到二维表示空间限制了内在的三维几何学习，并继承了二维预测的误差。为了解决这些局限性，我们提出了 GeoGuide，这是一个利用预训练三维模型来整合分层几何-语义一致性的新型开放词汇三维分割框架。具体而言，我们引入了一个基于不确定性的超点蒸馏模块，用于融合几何和语义特征以估计逐点不确定性，自适应地加权超点内的二维特征以抑制噪声，同时保留判别性信息以增强局部语义一致性。此外，我们的实例级掩码重建模块利用几何先验，通过重建完整的实例掩码来强制执行实例内的语义一致性。另外，我们的实例间关系一致性模块对齐几何和语义相似性矩阵，以校准同类别对象之间的跨实例一致性，从而减轻视点诱导的语义漂移。在 ScanNet v2、Matterport3D 和 nuScenes 上的广泛实验证明了 GeoGuide 的卓越性能。

## Abstract
Open-vocabulary 3D semantic segmentation aims to segment arbitrary categories beyond the training set. Existing methods predominantly rely on distilling knowledge from 2D open-vocabulary models. However, aligning 3D features to the 2D representation space restricts intrinsic 3D geometric learning and inherits errors from 2D predictions. To address these limitations, we propose GeoGuide, a novel framework that leverages pretrained 3D models to integrate hierarchical geometry-semantic consistency for open-vocabulary 3D segmentation. Specifically, we introduce an Uncertainty-based Superpoint Distillation module to fuse geometric and semantic features for estimating per-point uncertainty, adaptively weighting 2D features within superpoints to suppress noise while preserving discriminative information to enhance local semantic consistency. Furthermore, our Instance-level Mask Reconstruction module leverages geometric priors to enforce semantic consistency within instances by reconstructing complete instance masks. Additionally, our Inter-Instance Relation Consistency module aligns geometric and semantic similarity matrices to calibrate cross-instance consistency for same-category objects, mitigating viewpoint-induced semantic drift. Extensive experiments on ScanNet v2, Matterport3D, and nuScenes demonstrate the superior performance of GeoGuide.

### 1. 核心问题与研究动机
*   **核心问题**：现有的开放词汇三维（3D）语义分割方法过度依赖从预训练二维（2D）模型（如 CLIP）中蒸馏知识。
*   **研究背景与动机**：
    *   **几何缺失**：将 3D 特征强行对齐到 2D 表示空间，限制了模型学习 3D 内在几何结构的能力。
    *   **误差继承**：2D 模型在处理遮挡、视角变化和边界模糊时容易产生错误预测，3D 模型在蒸馏过程中会继承这些噪声。
    *   **一致性挑战**：如何在缺乏大规模 3D 文本对齐数据的情况下，利用 3D 几何先验来纠正 2D 蒸馏带来的偏差，实现局部到全局的语义一致性。

### 2. 方法论与关键技术
GeoGuide 提出了一个分层几何引导框架，通过三个核心模块在不同粒度上强制执行几何-语义一致性：
*   **基于不确定性的超点蒸馏 (USD)**：
    *   利用超点（几何同质区域）应具有相同语义的特性。
    *   融合 3D 几何特征与 2D 语义特征来估计逐点“可靠性权重”，通过加权聚合抑制 2D 预测中的噪声，增强超点内部的语义一致性。
*   **实例级掩码重建 (IMR)**：
    *   利用类无关的 3D 实例分割生成提案。
    *   通过随机遮蔽部分实例区域并尝试重建完整掩码，迫使模型学习完整的实例几何结构，确保实例内部所有点语义统一。
*   **实例间关系一致性 (IIRC)**：
    *   核心思想是：几何相似的对象（如两把椅子）在语义上也应相似。
    *   计算实例间和超点间的几何相似性矩阵，并将其作为目标来对齐语义相似性矩阵，从而缓解因视角变化导致的语义漂移。

### 6. 主要结论与发现
*   **性能领先**：GeoGuide 在所有基准测试中均达到 SOTA。在 ScanNet v2 上达到 64.8 mIoU，显著超过了之前的 SAS (61.9 mIoU)。
*   **几何纠偏有效**：实验证明，单纯引入 3D 预训练特征提升有限，但通过分层几何约束，可以有效纠正 2D 蒸馏带来的几何不一致问题。
*   **泛化能力强**：在类别数量增加和跨数据集测试中，GeoGuide 的性能下降幅度远小于基准方法，显示出极强的鲁棒性。

### 7. 优点
*   **分层建模**：从局部超点到单个实例，再到跨实例关系，构建了完整的几何引导体系。
*   **不确定性感知**：USD 模块能够自适应地识别并过滤 2D 投影中的噪声点，而非盲目平均。
*   **即插即用**：推理时不需要 2D 模型或几何引导模块，仅需纯 3D 输入，效率较高。

### 8. 不足与局限
*   **依赖预训练质量**：性能在很大程度上取决于预训练 3D 骨干网络（如 Sonata）和类无关实例分割算法的质量。
*   **计算开销**：虽然推理高效，但训练阶段涉及相似性矩阵计算和掩码重建，计算复杂度随实例数量增加而上升。
*   **2D 依赖消除不彻底**：尽管引入了 3D 几何，但语义信息的源头仍是 2D VLM，若 2D 模型对某些罕见类完全无认知，3D 几何也难以凭空修复语义。

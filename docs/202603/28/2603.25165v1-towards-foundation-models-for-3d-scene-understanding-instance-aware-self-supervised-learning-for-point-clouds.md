## 摘要
近期点云自监督学习（SSL）的进展在无需人工标注的情况下显著提升了3D场景理解能力。现有方法通过在增强视图间强制执行特征一致性或通过掩码场景建模来强调语义感知。然而，生成的表示在实例定位方面的迁移效果较差，且通常需要全量微调才能获得较强的性能。实例感知是3D感知的基本组成部分，因此弥补这一差距对于迈向支持3D数据所有下游任务的真正3D基础模型至关重要。在这项工作中，我们介绍了 PointINS，这是一个面向实例的自监督框架，通过几何感知学习丰富了点云表示。PointINS 采用正交偏移分支来共同学习高层语义理解和几何推理，从而产生实例感知。我们确定了对于鲁棒实例定位至关重要的两个一致性属性，并将其制定为互补的正规化策略：偏移分布正规化（ODR），它将预测的偏移与经验观察到的几何先验对齐；以及空间聚类正规化（SCR），它通过使用伪实例掩码正规化偏移来强制执行局部连贯性。通过在五个数据集上的广泛实验，PointINS 在室内实例分割方面平均实现了 +3.5% 的 mAP 提升，在室外全景分割方面实现了 +4.1% 的 PQ 增益，为可扩展的 3D 基础模型铺平了道路。

## Abstract
Recent advances in self-supervised learning (SSL) for point clouds have substantially improved 3D scene understanding without human annotations. Existing approaches emphasize semantic awareness by enforcing feature consistency across augmented views or by masked scene modeling. However, the resulting representations transfer poorly to instance localization, and often require full finetuning for strong performance. Instance awareness is a fundamental component of 3D perception, thus bridging this gap is crucial for progressing toward true 3D foundation models that support all downstream tasks on 3D data. In this work, we introduce PointINS, an instance-oriented self-supervised framework that enriches point cloud representations through geometry-aware learning. PointINS employs an orthogonal offset branch to jointly learn high-level semantic understanding and geometric reasoning, yielding instance awareness. We identify two consistent properties essential for robust instance localization and formulate them as complementary regularization strategies, Offset Distribution Regularization (ODR), which aligns predicted offsets with empirically observed geometric priors, and Spatial Clustering Regularization (SCR), which enforces local coherence by regularizing offsets with pseudo-instance masks. Through extensive experiments across five datasets, PointINS achieves on average +3.5% mAP improvement for indoor instance segmentation and +4.1% PQ gain for outdoor panoptic segmentation, paving the way for scalable 3D foundation models.

*   **研究背景**：在自动驾驶和机器人领域，实例定位（如实例分割、全景分割）至关重要。目前的SSL表示在这些任务上表现不佳，通常需要全量微调才能达到较好效果。
*   **整体含义**：为了构建真正的3D基础模型，SSL必须同时具备语义理解和几何推理能力。本文提出的 **PointINS** 是首个显式设计用于在自监督预训练阶段学习实例级几何偏移的框架。

### 2. 方法论：核心思想与关键技术
PointINS 采用教师-学生（Teacher-Student）蒸馏架构，包含两个正交的分支：
*   **语义分支（Semantic Branch）**：采用基于原型的自蒸馏机制，通过对齐增强视图间的原型分配概率来学习类别级特征。
*   **偏移分支（Offset Branch）**：预测每个点指向其所属实例几何中心的3D向量。为了在无标注情况下稳定训练，提出了两种正则化策略：
    *   **偏移分布正则化（ODR）**：利用概率积分变换（PIT），强制预测的偏移量符合经验观察到的几何先验（偏移长度呈长尾分布，方向在球面上均匀分布），防止模型坍塌。
    *   **空间聚类正则化（SCR）**：利用模型自身的语义特征进行K-means聚类，并在聚类块内通过广度优先搜索（BFS）构建空间连贯的“伪实例掩码”，以此作为局部几何一致性的监督信号。
*   **损失函数**：结合了语义蒸馏的KL散度损失和偏移分支的 $\ell_1$ 距离及余弦相似度损失。

*   **充分性与公平性**：所有对比实验均使用相同的 PTv3 主干网络，并严格遵循相同的评估协议。此外，还进行了标签效率实验（仅用 0.1% 和 1% 标注）和跨数据集探测实验，验证了模型的泛化能力和鲁棒性。

### 6. 主要结论与发现
*   **性能提升**：PointINS 在室内实例分割上平均提升了 **3.5% mAP**，在室外全景分割上提升了 **4.1% PQ**。
*   **语义与几何的互补性**：实验证明，引入实例感知的偏移学习不仅不会损害语义特征，反而能通过增强几何推理能力来提升语义分割的性能。

### 7. 优点与亮点
*   **首创性**：首次将实例偏移学习引入 3D 自监督预训练。
*   **无需标注的正则化**：ODR 和 SCR 巧妙地利用了统计先验和自身特征聚类，解决了无监督下偏移预测易坍塌的问题。
*   **高兼容性**：该方法可以无缝集成到现有的教师-学生 SSL 框架中（如 PSA 或 Sonata）。
*   **无监督潜力**：甚至在完全不进行下游训练的情况下，直接利用预测偏移进行聚类也能获得初步的实例分割结果。

### 8. 不足与局限
*   **性能差距**：尽管大幅领先于其他 SSL 方法，但在线性探测下与全监督性能仍存在一定差距。
*   **计算开销**：引入额外的偏移分支和正则化计算（如 K-means 和 BFS）增加了约 25% 的预训练耗时。
*   **先验依赖**：ODR 依赖于对场景几何分布的经验假设（虽然实验证明其对分布偏差不敏感，但在极端特殊的场景下可能需要调整）。
*   **维度限制**：目前主要针对静态场景，尚未扩展到包含时序信息的 4D 几何推理。

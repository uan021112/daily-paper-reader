## 摘要
我们提出了 UniQueR，这是一个统一的基于查询的前馈框架，用于从无位姿图像中进行高效且准确的 3D 重建。现有的前馈模型（如 DUSt3R、VGGT 和 AnySplat）通常预测逐像素点图或像素对齐的高斯分布，这在本质上仍属于 2.5D，且局限于可见表面。相比之下，UniQueR 将重建建模为一个稀疏的 3D 查询推理问题。我们的模型学习了一组紧凑的 3D 锚点，这些锚点作为显式的几何查询，使网络能够在单次前馈过程中推断场景结构，包括遮挡区域的几何形状。每个查询直接在全局 3D 空间（而非逐帧相机空间）中编码空间和外观先验，并生成一组用于可微分渲染的 3D 高斯。通过利用跨多视图特征的统一查询交互和解耦的交叉注意力设计，UniQueR 在显著降低内存和计算成本的同时，实现了强大的几何表达能力。在 Mip-NeRF 360 和 VR-NeRF 上的实验表明，UniQueR 在渲染质量和几何精度方面均超越了最先进的前馈方法，且使用的基元数量比密集型替代方案少一个数量级。

## Abstract
We present UniQueR, a unified query-based feedforward framework for efficient and accurate 3D reconstruction from unposed images. Existing feedforward models such as DUSt3R, VGGT, and AnySplat typically predict per-pixel point maps or pixel-aligned Gaussians, which remain fundamentally 2.5D and limited to visible surfaces. In contrast, UniQueR formulates reconstruction as a sparse 3D query inference problem. Our model learns a compact set of 3D anchor points that act as explicit geometric queries, enabling the network to infer scene structure, including geometry in occluded regions--in a single forward pass. Each query encodes spatial and appearance priors directly in global 3D space (instead of per-frame camera space) and spawns a set of 3D Gaussians for differentiable rendering. By leveraging unified query interactions across multi-view features and a decoupled cross-attention design, UniQueR achieves strong geometric expressiveness while substantially reducing memory and computational cost. Experiments on Mip-NeRF 360 and VR-NeRF demonstrate that UniQueR surpasses state-of-the-art feedforward methods in both rendering quality and geometric accuracy, using an order of magnitude fewer primitives than dense alternatives.

#### 1. 核心问题与整体含义（研究动机和背景）
传统的 3D 重建方法（如 SfM、MVS）依赖复杂的几何优化，而新兴的神经表示（如 NeRF、3DGS）通常需要针对每个场景进行耗时的单独优化。现有的前馈（Feedforward）模型（如 DUSt3R、AnySplat）虽然实现了快速推理，但大多采用**像素对齐（Pixel-aligned）**的 2.5D 表示。
*   **局限性**：这些模型仅能重建可见表面，在遮挡区域或未观察视角下会产生“空洞”；此外，逐像素预测导致高斯基元数量巨大，计算和内存开销极高。
*   **核心目标**：提出一种基于**稀疏 3D 查询（Query-based）**的框架，将场景表示从相机空间解耦到全局 3D 空间，从而实现更完整、更高效的 3D 重建。

#### 2. 论文提出的方法论
UniQueR 将 3D 重建重新表述为一个稀疏查询推理问题，其核心流程如下：
*   **图像特征提取**：使用 DINOv2 骨干网络提取多视图图像特征，并通过交替注意力（Alternating-attention）Transformer 进行跨视图信息聚合。
*   **几何先验解码**：初步预测相机位姿、点图（Point maps）和置信度，为后续查询提供几何引导。
*   **混合查询初始化**：
    *   一半查询从预测的点图中采样（覆盖可见表面）。
    *   另一半查询作为可学习的 3D 锚点在空间中均匀分布（探索遮挡/未知区域）。
*   **解耦交叉注意力机制**：为了处理多视图输入，设计了解耦注意力：首先是查询对图像特征的**交叉注意力**（吸收视觉信息），然后是查询之间的**自注意力**（保持全局一致性）。这种设计将复杂度从 $O((Q+N)^2)$ 降低到 $O(QN + Q^2)$。
*   **高斯生成（GS Spawning）**：每个 3D 查询生成 $K$ 个 3D 高斯基元（包括位置偏移、缩放、旋转、不透明度和颜色）。
*   **训练监督**：利用可微分渲染，通过 RGB 损失和深度损失进行监督。关键创新在于**新视角监督**：训练时监督的视角多于输入视角，迫使模型推断遮挡区域的几何结构。

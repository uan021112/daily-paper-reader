## 摘要
创建高保真、可定制的 3D 室内场景纹理仍然是一个重大挑战。虽然文本驱动的方法具有灵活性，但它们缺乏细粒度、实例级控制的精度，且生成的纹理往往质量不足，存在伪影和烘焙阴影（baked-in shading）。为了克服这些局限性，我们提出了 CustomTex，这是一个由参考图像驱动的实例级、高保真场景纹理生成新框架。CustomTex 输入一个未贴图的 3D 场景和一组指定每个物体实例所需外观的参考图像，并生成统一的高分辨率纹理贴图。我们方法的核心是一种双重蒸馏方法，它将语义控制与像素级增强分离。我们采用了配备实例交叉注意力（instance cross-attention）的语义级蒸馏，以确保语义合理性和“参考-实例”对齐，并采用像素级蒸馏来增强视觉保真度。两者都统一在变分分数蒸馏（VSD）优化框架内。实验表明，与现有最先进的方法相比，CustomTex 实现了与参考图像精确的实例级一致性，并生成了具有更高清晰度、更少伪影和极少烘焙阴影的纹理。我们的工作为高质量、可定制的 3D 场景外观编辑建立了一条更直接、更用户友好的路径。

## Abstract
The creation of high-fidelity, customizable 3D indoor scene textures remains a significant challenge. While text-driven methods offer flexibility, they lack the precision for fine-grained, instance-level control, and often produce textures with insufficient quality, artifacts, and baked-in shading. To overcome these limitations, we introduce CustomTex, a novel framework for instance-level, high-fidelity scene texturing driven by reference images. CustomTex takes an untextured 3D scene and a set of reference images specifying the desired appearance for each object instance, and generates a unified, high-resolution texture map. The core of our method is a dual-distillation approach that separates semantic control from pixel-level enhancement. We employ semantic-level distillation, equipped with an instance cross-attention, to ensure semantic plausibility and ``reference-instance'' alignment, and pixel-level distillation to enforce high visual fidelity. Both are unified within a Variational Score Distillation (VSD) optimization framework. Experiments demonstrate that CustomTex achieves precise instance-level consistency with reference images and produces textures with superior sharpness, reduced artifacts, and minimal baked-in shading compared to state-of-the-art methods. Our work establishes a more direct and user-friendly path to high-quality, customizable 3D scene appearance editing.

*   **研究动机**：为了实现更高精度、更符合用户特定需求的场景编辑，研究者希望通过**参考图像**（Reference Images）来直接驱动纹理生成，使场景中的每个物体（如沙发、地板、墙壁）都能精准匹配用户提供的视觉素材，同时提升纹理的物理真实感和清晰度。

#### 2. 方法论：核心思想与关键技术
CustomTex 提出了一种基于**双重蒸馏（Dual-distillation）**的优化框架，将语义控制与像素增强分离：
*   **核心思想**：在 UV 空间中优化一个隐式多分辨率哈希网格（Hash Grid）表示的纹理场，通过预训练的扩散模型提供先验指导。
*   **关键技术细节**：
    *   **语义级蒸馏（InsVSD）**：引入实例引导的变分分数蒸馏（Instance-guided VSD）。利用 **IP-Adapter** 提取参考图特征，并通过**实例交叉注意力机制（Instance Cross-attention）**，结合渲染时的实例掩码（Mask），将特定参考图的特征精准映射到 3D 场景的对应物体上。
    *   **像素级蒸馏**：集成了一个预训练的**超分辨率（SR）扩散模型**。在优化过程中，SR 模型提供高频细节的梯度，迫使生成的纹理超越普通扩散模型的模糊感，达到 4K 甚至更高分辨率的清晰度。
    *   **LoRA 动态学习**：在优化纹理的同时，训练一个轻量级的 LoRA 模块来捕捉参考图像的特定分布，从而在 VSD 框架下实现更稳定的分数估计。
*   **算法流程**：从随机视角渲染场景 -> 获取 RGB、深度图和实例掩码 -> 计算语义蒸馏梯度（对齐参考图）和像素蒸馏梯度（增强细节） -> 反向传播更新 UV 纹理场参数。

#### 6. 主要结论与发现
*   **实例一致性**：CustomTex 能够精准地将不同参考图的材质（如特定木纹、织物花纹）应用到指定的 3D 实例上。
*   **高质量纹理**：通过像素级蒸馏，生成的纹理在锐度和细节丰富度上显著优于基于文本的方法，且有效抑制了伪影。
*   **光影分离**：相比于全局扩散生成，实例级的控制减少了“烘焙阴影”，生成的纹理更接近于纯净的漫反射贴图（Albedo），更适合重光照应用。

#### 7. 优点
*   **高保真度**：引入超分辨率模型作为蒸馏先验，解决了扩散模型生成 3D 纹理普遍存在的模糊问题。
*   **强控制力**：支持多参考图像定制，解决了文本驱动难以描述复杂纹理（如特定壁纸图案）的痛点。
*   **灵活性**：基于隐式表示，可以生成任意分辨率（最高测试到 12K）的纹理贴图。

#### 8. 不足与局限
*   **训练成本高**：单个场景 48 小时的训练时间对于实时应用或大规模生产来说过于缓慢。
*   **材质属性单一**：目前仅专注于生成漫反射颜色贴图（Diffuse Albedo），尚未涉及法线贴图（Normal）、粗糙度（Roughness）等 PBR 材质属性的同步生成。
*   **几何依赖**：纹理质量高度依赖于输入 3D 模型的 UV 展开质量，若 UV 重叠或扭曲严重，生成效果会受限。

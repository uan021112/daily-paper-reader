## 摘要
高质量的关节式 3D 资产对于具身智能和物理仿真至关重要，然而目前的 3D 生成仍侧重于静态网格，在“仿真就绪”的可交互物体方面存在空白。最近的大多数关节式物体创建方法依赖于多阶段流水线，这会导致误差在解耦的模块间累积。相比之下，统一的多模态大语言模型 (MLLM) 提供了一条单阶段路径，可同时实现静态资产理解和仿真就绪资产生成。然而，基于密集体素的 3D 标记化会产生冗长的 3D 标记序列和高昂的内存开销，限制了其在复杂关节式物体上的可扩展性。为了解决这一问题，我们提出了 SIMART，这是一个统一的 MLLM 框架，可同时执行部件级分解和运动学预测。通过引入稀疏 3D VQ-VAE，SIMART 相比密集体素标记减少了 70% 的标记数量，从而实现了高保真的多部件装配。SIMART 在 PartNet-Mobility 和真实场景 AIGC 数据集上达到了最先进的性能，并支持基于物理的机器人仿真。

## Abstract
High-quality articulated 3D assets are indispensable for embodied AI and physical simulation, yet 3D generation still focuses on static meshes, leaving a gap in "sim-ready" interactive objects. Most recent articulated object creation methods rely on multi-stage pipelines that accumulate errors across decoupled modules. Alternatively, unified MLLMs offer a single-stage path to joint static asset understanding and sim-ready asset generation. However dense voxel-based 3D tokenization yields long 3D token sequences and high memory overhead, limiting scalability to complex articulated objects. To address this, we propose SIMART, a unified MLLM framework that jointly performs part-level decomposition and kinematic prediction. By introducing a Sparse 3D VQ-VAE, SIMART reduces token counts by 70% vs. dense voxel tokens, enabling high-fidelity multi-part assemblies. SIMART achieves state-of-the-art performance on PartNet-Mobility and in-the-wild AIGC datasets, and enables physics-based robotic simulation.

### 1. 核心问题与研究动机
*   **核心问题**：如何高效地将单一的静态 3D 网格（Monolithic Meshes）转化为可直接用于物理仿真和具身智能交互的“仿真就绪”（Sim-ready）关节式资产。
*   **研究背景**：
    *   现有的 3D 生成模型多产出静态网格，缺乏运动学（Kinematic）和物理元数据。
    *   传统方法多采用多阶段流水线（分解、推理、组装），容易导致模块间的误差累积。
    *   现有的 3D 多模态大模型（MLLM）受限于密集的体素标记化（Voxel Tokenization），在处理复杂物体时会产生极长的 Token 序列，导致内存溢出（OOM）且计算效率低下。

### 2. 方法论
*   **核心思想**：提出一个统一的 MLLM 框架，通过引入**稀疏 3D VQ-VAE** 降低计算开销，同步实现部件级几何分解与运动学参数预测。
*   **关键技术细节**：
    *   **Sparse 3D VQ-VAE**：将 $64^3$ 的体素网格编码为 $8^3$ 的潜在网格。创新点在于引入了专门的“零标记”（Zero Token），仅对被占据的表面体素进行量化，从而将 Token 数量减少了约 70%。
    *   **坐标感知标记化**：每个占据的体素被序列化为三元组：`<voxel> [xyz] [K]`，其中 `xyz` 是空间位置索引，`K` 是来自码本的几何特征索引。
    *   **统一 MLLM 主干**：采用 **Qwen3-VL-8B**。输入包含视觉图像（ViT 提取）、几何 Token（VQ-VAE 提取）和文本指令；输出为结构化的 JSON（包含 URDF 规格、物理属性）和各部件的几何 Token。
    *   **后处理分割**：利用 VQ-VAE 解码生成的点云作为种子点，通过基于图的表面分割算法（Graph-based segmentation）将标签映射回原始高保真网格，确保几何精度。

### 6. 主要结论与发现
*   **性能领先**：SIMART 在关节预测准确性和几何分解精度上均显著优于现有 SOTA 方法，尤其是在处理 AI 生成的非结构化网格时表现稳健。
*   **效率提升**：稀疏 VQ-VAE 成功解决了 MLLM 处理 3D 数据时的上下文长度瓶颈，使得模型能够处理具有更多部件的复杂物体。
*   **多模态协同**：视觉信息对于区分外观相似但运动结构不同的物体（如不同类型的柜子）至关重要。

### 7. 优点与亮点
*   **单阶段统一框架**：将感知、推理与生成整合，避免了多级模块间的误差传递。
*   **稀疏化创新**：通过“零标记”机制大幅降低了 3D Token 冗余，是 3D MLLM 走向实用化的重要改进。
*   **仿真就绪性**：输出不仅是分割后的网格，还包含完整的 URDF 信息（轴、限位、摩擦力、密度），可直接导入物理引擎。

### 8. 不足与局限
*   **数据依赖性**：尽管引入了 AIGC 数据，但高质量关节式 3D 数据的稀缺仍是限制模型泛化到极罕见类别的主要瓶颈。
*   **体素分辨率限制**：$64^3$ 的体素分辨率虽然通过后处理映射到了原始网格，但在处理极细微的机械结构时可能仍存在感知精度上限。
*   **应用限制**：目前主要针对刚性关节物体，对于布料、流体等非刚性或更复杂的物理交互支持尚不明确。

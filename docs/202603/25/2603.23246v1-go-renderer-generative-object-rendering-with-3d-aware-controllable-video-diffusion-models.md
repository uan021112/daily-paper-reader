## 摘要
从图像中重建可渲染的 3D 模型是一项具有实用价值但极具挑战性的任务。最近的前馈 3D 重建方法在高效恢复几何结构方面取得了显著成功，但仍无法准确建模这些 3D 重建模型的复杂外观。最近基于扩散的生成模型可以利用参考图像合成物体的逼真图像或视频，而无需显式建模其外观，这为物体渲染提供了一个极具前景的方向，但缺乏对视角的精确控制。在本文中，我们提出了 GO-Renderer，这是一个统一的框架，它集成了重建的 3D 代理（proxies）来引导视频生成模型，从而在任意光照条件下实现任意视角的高质量物体渲染。我们的方法不仅利用重建的 3D 代理实现了精确的视角控制，还能够利用扩散生成模型在不同的光照环境中实现高质量渲染，而无需显式建模复杂的材质和光照。大量实验表明，GO-Renderer 在各项物体渲染任务中均达到了最先进的性能，包括在新视角下合成图像、在全新光照环境中渲染物体以及将物体插入现有视频中。

## Abstract
Reconstructing a renderable 3D model from images is a useful but challenging task. Recent feedforward 3D reconstruction methods have demonstrated remarkable success in efficiently recovering geometry, but still cannot accurately model the complex appearances of these 3D reconstructed models. Recent diffusion-based generative models can synthesize realistic images or videos of an object using reference images without explicitly modeling its appearance, which provides a promising direction for object rendering, but lacks accurate control over the viewpoints. In this paper, we propose GO-Renderer, a unified framework integrating the reconstructed 3D proxies to guide the video generative models to achieve high-quality object rendering on arbitrary viewpoints under arbitrary lighting conditions. Our method not only enjoys the accurate viewpoint control using the reconstructed 3D proxy but also enables high-quality rendering in different lighting environments using diffusion generative models without explicitly modeling complex materials and lighting. Extensive experiments demonstrate that GO-Renderer achieves state-of-the-art performance across the object rendering tasks, including synthesizing images on new viewpoints, rendering the objects in a novel lighting environment, and inserting an object into an existing video.

#### 2. 论文提出的方法论
*   **核心思想**：利用粗糙的 3D 重建结果（如点云或网格）生成“物体中心坐标图（Coordinate Maps）”，将其作为空间约束输入到视频扩散模型中，引导模型学习从参考图像到目标视角的纹理映射。
*   **关键技术细节**：
    1.  **3D 代理与坐标图**：使用前馈重建模型（如 ReconViaGen）获取物体的粗糙几何，将其渲染为坐标图（像素颜色编码物体的局部 3D 坐标 $x, y, z$）。
    2.  **联合参考结构（Joint Reference Structure）**：将多张参考图像与其对应的参考坐标图在通道维度拼接；同时将目标视角的坐标图与潜在噪声拼接。模型通过共享的 3D 坐标实现像素级的“查表”和纹理转移。
    3.  **负向 RoPE 偏移（Negative RoPE Shift）**：修改旋转位置嵌入（RoPE），给参考帧分配负的时间索引（如 $-N \cdot g$），给生成帧分配正索引。这在时间上隔离了参考信息，防止生成视频首帧出现过渡伪影。
*   **算法流程**：输入参考图 $\rightarrow$ 快速 3D 重建 $\rightarrow$ 渲染参考/目标坐标图 $\rightarrow$ 视频扩散模型（DiT）去噪生成 $\rightarrow$ 高质量视频。

#### 6. 论文的主要结论与发现
*   **生成式渲染的优越性**：GO-Renderer 在 PSNR 和感知指标上显著优于传统的“重建-重光照”流程，证明了扩散模型可以有效替代复杂的物理材质建模。
*   **3D 代理的引导作用**：即使 3D 代理非常粗糙，坐标图也能提供足够的空间约束，确保精确的视角控制和多视角一致性。
*   **鲁棒性**：模型对 3D 代理的轻微空间偏移具有容错性，当几何完全错误时，模型会退化为依赖 2D 先验生成合理外观，而非直接崩溃。

#### 7. 优点（亮点）
*   **解耦物理建模**：无需显式估计 BRDF 或环境光，降低了渲染复杂场景的门槛。
*   **精确控制**：相比之前的生成式模型，通过坐标图实现了像素级的视角控制。
*   **多视图融合**：Joint Reference 结构允许模型同时利用多个视角的参考信息，减少了单图生成的幻觉问题。
*   **实用性强**：展示了在视频编辑和 3D 创作工作流中的直接应用潜力。

#### 8. 不足与局限
*   **对 3D 代理的依赖**：虽然有一定鲁棒性，但如果 3D 代理与参考图结构严重不匹配（如用瓶子的几何去渲染蛋糕），会导致严重的结构伪影。
*   **计算开销**：基于大型视频扩散模型（如 Wan2.2 5B），推理成本相对传统渲染器仍然较高。
*   **动态物体限制**：目前主要针对静态物体的视角变换和重光照，对于物体自身的形变（如动物跑动）支持有限。
